## 6. 連番対象外として維持する番号

- `SCR-AUTH`、`SCR-HOME`、`SCR-APPROVALS` 系は記号 ID 体系のため、数値画面の連番対象外。
- 連携 IF #3 / #11 は予約番号として既存の外部契約を維持し、今回の再採番対象外。
- FR / NFR / BR / AC のカテゴリ別番号帯と英字サフィックスは意味を持つ識別子のため、カテゴリをまたぐ全体連番には変更しない。
- FUT06 の 3 文書は同一将来機能の文書分割であり、重複 ID ではなく同一 ID の文書セットとして扱う。
