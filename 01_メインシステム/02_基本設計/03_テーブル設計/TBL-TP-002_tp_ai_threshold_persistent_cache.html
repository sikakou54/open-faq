概要

項目 内容
テーブル名 TP_ai_threshold_persistent_cache
論理名 AI しきい値キャッシュ
主キー id
外部キー
  • contract_owner_user_id → M_owners(id)
  • project_id → M_projects(id)

カラム定義

No 論理名 物理名 データ型 桁数 NULL PK FK UNIQUE DEFAULT 制約
1 ID id TEXT - NO ○
2 契約 owner ID contract_owner_user_id TEXT - YES M_owners(id)
3 プロジェクト ID project_id TEXT - YES M_projects(id)
4 スコープ scope TEXT - NO scope IN ('global','owner','project')
（複合）(scope = 'global' AND contract_owner_user_id IS NULL AND project_id IS NULL) OR (scope = 'owner' AND contract_owner_user_id IS NOT NULL AND project_id IS NULL) OR (scope = 'project' AND contract_owner_user_id IS NOT NULL AND project_id IS NOT NULL)
5 信頼度しきい値 confidence_threshold REAL - NO confidence_threshold BETWEEN 0.0 AND 1.0
6 関連度しきい値 relevance_threshold REAL - NO relevance_threshold BETWEEN 0.0 AND 1.0
7 バージョン version INTEGER - NO
8 受信日時 received_at TEXT - NO
9 更新日時 updated_at TEXT - NO
