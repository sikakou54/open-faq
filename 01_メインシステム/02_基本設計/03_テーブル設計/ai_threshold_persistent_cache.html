TBL-016 ai_threshold_persistent_cache

3 階層 AI しきい値を永続キャッシュするテーブルです。

概要

項目 内容
テーブル名 ai_threshold_persistent_cache
論理名 AI しきい値キャッシュ
説明 3 階層しきい値(global / owner / project)の永続キャッシュ
保持期間 / 保持起点 30 日 / 更新日時
contract_owner_user_id あり(scope=owner または project 時のみ)
retention_class general

カラム定義

No 論理名 物理名 データ型 桁数 NULL PK FK UNIQUE DEFAULT 説明
1 ID id TEXT - NO ○ ULID
2 契約 owner ID contract_owner_user_id TEXT - YES ○ scope=global 時 NULL
3 プロジェクト ID project_id TEXT - YES ○ scope=project 時のみ NOT NULL
4 スコープ scope TEXT - NO global / owner / project
5 信頼度しきい値 confidence_threshold REAL - NO 0.0〜1.0
6 関連度しきい値 relevance_threshold REAL - NO 0.0〜1.0
7 バージョン version INTEGER - NO 配信バージョン
8 受信日時 received_at TEXT - NO ISO 8601 UTC
9 更新日時 updated_at TEXT - NO ISO 8601 UTC

主キー

項目 内容
主キー名 (暗黙、PRIMARY KEY 句で指定)
対象カラム id

No 外部キー名 対象カラム 参照先テーブル 参照先カラム 更新時 削除時

2 (暗黙) project_id projects id NO ACTION CASCADE

インデックス

No インデックス名 対象カラム UNIQUE 用途

制約

No 制約名 種別 対象カラム 条件・内容
1 (暗黙) CHECK scope scope IN ('global','owner','project')
2 (暗黙) CHECK confidence_threshold confidence_threshold BETWEEN 0.0 AND 1.0
3 (暗黙) CHECK relevance_threshold relevance_threshold BETWEEN 0.0 AND 1.0
4 (暗黙) 複合CHECK scope × contract_owner_user_id × project_id (scope = 'global' AND contract_owner_user_id IS NULL AND project_id IS NULL) OR (scope = 'owner' AND contract_owner_user_id IS NOT NULL AND project_id IS NULL) OR (scope = 'project' AND contract_owner_user_id IS NOT NULL AND project_id IS NOT NULL)
