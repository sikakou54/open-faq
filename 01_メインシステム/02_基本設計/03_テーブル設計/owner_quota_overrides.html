TBL-023 owner_quota_overrides

運営者による契約単位のレート制限上書きを保持します。

概要

項目 内容
テーブル名 owner_quota_overrides
論理名 契約別レート制限上書き
主キー id
外部キー contract_owner_user_id → contract_owners(user_id)

カラム定義

No 論理名 物理名 データ型 桁数 NULL PK FK UNIQUE DEFAULT 制約
1 ID id TEXT - NO ○
2 契約 owner ID contract_owner_user_id TEXT - NO contract_owners(user_id)
3 リソース種別 resource_kind TEXT - NO resource_kind IN ('widget_ask_per_min','widget_inquiry_per_min','email_per_hour','admin_api_per_min')
4 しきい値 threshold INTEGER - NO
5 窓秒 window_sec INTEGER - YES
6 有効期限 valid_until TEXT - NO
7 理由 reason TEXT - YES
8 作成者 created_by TEXT - YES
9 作成日時 created_at TEXT - NO

インデックス

No インデックス名 対象カラム UNIQUE 用途
1 idx_tqo_owner_kind (contract_owner_user_id, resource_kind, valid_until) オーナー × リソース × 有効期限

コード値・区分値

カラム名 値 意味
resource_kind widget_ask_per_min ウィジェット質問レート
resource_kind widget_inquiry_per_min ウィジェット未解決質問送信レート
resource_kind email_per_hour メール送信レート
resource_kind admin_api_per_min 管理 API レート

*_monthly_limit 系コードは project_quota_limits(§3.24)へ移管したため、本テーブルには含めない。質問数だけが月次上限を持ち、FAQは無料枠行の識別子として使用する。

詳細は 10_課金・請求設計.html §3 / 04_権限設計.html §7 参照。
