TBL-025 service_announcements

運営者が発信するお知らせ本体(Control Plane)を保持します。主キーは id、5 年保持します(retention_class=operator_high_priv)。

概要

項目 内容
テーブル名 service_announcements
論理名 お知らせ(Control Plane)
説明 運営者発信のお知らせ本体
主管 FR -
保持期間 / 保持起点 5 年 / 配信日時
contract_owner_user_id なし(全契約横断 or announcement_audiences で限定)
retention_class operator_high_priv
設計判断 配信対象は announcement_audiences(§3.26)で M:N 管理。announcement_audiences 行が 0 件のお知らせは「全契約配信」を意味する。importance 4 値は共有概念正本(critical はメール強制送信)

カラム定義

No 論理名 物理名 データ型 桁数 NULL PK FK UNIQUE DEFAULT 説明
1 ID id TEXT - NO ○ ULID
2 タイトル title TEXT - NO 1〜200 文字
3 本文(HTML) body_html TEXT - NO サニタイズ済 HTML
4 重要度 importance TEXT - NO low / normal / high / critical
5 公開日時 published_at TEXT - NO ISO 8601 UTC
6 取り下げ日時 retracted_at TEXT - YES 取り下げ済みのみセット
7 作成者 created_by TEXT - YES ○ users(id) 参照(削除でも履歴維持)
8 作成日時 created_at TEXT - NO ISO 8601 UTC

主キー

項目 内容
主キー名 (暗黙、PRIMARY KEY 句で指定)
対象カラム id

外部キー

No 外部キー名 対象カラム 参照先テーブル 参照先カラム 更新時 削除時
1 (暗黙) created_by users id NO ACTION SET NULL

インデックス

No インデックス名 対象カラム UNIQUE 用途
1 idx_announcements_published published_at DESC WHERE retracted_at IS NULL 公開中の新着

制約

No 制約名 種別 対象カラム 条件・内容
1 (暗黙) CHECK title length(title) BETWEEN 1 AND 200
