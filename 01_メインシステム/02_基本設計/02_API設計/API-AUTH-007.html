API-AUTH-007 招待トークン検証・プレビュー

この API は、招待情報(プロジェクト名 / 付与ロール / 招待元オーナー名 / 招待者メール)をプレビュー取得する。

版数 v1.6更新 2026-06-17承認済

基本情報

項目内容
API IDAPI-AUTH-007
API 名招待トークン検証 + 招待情報プレビュー(機能 ID: F-016)
エンドポイント/auth/invitations/{token}/preview
HTTP メソッドPOST
概要招待情報(プロジェクト名 / 付与ロール / 招待元オーナー名 / 招待者メール)をプレビュー取得する
利用画面SCR-018 メンバーアカウント有効化(初期表示時)
認証未認証可(トークン認証のみ)
権限—

トークン形式 = activation purpose の access_tokens.token_hash 検証。検証成功時は招待情報を返し、SCR-018 の招待情報パネル表示に使用する。Turnstile 不要(プレビューは情報取得のみで状態変更なし)。

リクエストパラメータ

Path Parameters: {token}(招待トークン)。Request Body: なし。

レスポンス

(200):

  "userId": "01HZ...",
  "email": "newmember@...",
  "projectId": "proj_1",

  "invitedRole": "admin" | "member",
  "inviterOwnerName": "山田 一郎",

}

エラー

HTTP ステータスエラーコード内容
410TOKEN_EXPIRED(E-AUTH-TOKEN-EXPIRED)トークン期限切れ
410TOKEN_USED(E-AUTH-TOKEN-USED)トークン使用済み
404TOKEN_NOT_FOUNDトークンが存在しない
