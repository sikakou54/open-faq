API-AUTH-002 ログイン

この API は、メール / パスワードを認証しセッションを発行する。

版数 v1.6更新 2026-06-17承認済

基本情報

項目内容
API IDAPI-AUTH-002
API 名ログイン(機能 ID: F-002)
エンドポイント/auth/login
HTTP メソッドPOST
概要メール / パスワードを認証しセッションを発行する
利用画面SCR-001
認証不要
権限—(公開)

関連テーブル: accounts, sessions。

リクエストパラメータ

Query Parameters / Path Parameters: なし。Request Body:

{
  "email": "admin@example.com",
  "password": "...",
  "turnstileToken": "..."
}

レスポンス

(200):

{
  "userId": "01HZ...",
  "role": "admin",
  "requireTermsAgreement": false,
  "activeSessions": [
    { "id": "...", "ipAddress": "203.0.113.x", "createdAt": "..." }
  ]
}

エラー

HTTP ステータスエラーコード内容
401INVALID_CREDENTIALS(E-AUTH-CREDENTIAL)認証情報が不正
423LOCKED_OUT(E-AUTH-LOCKED)アカウントロック中
400TURNSTILE_REQUIREDTurnstile が必要
403CONTRACT_SUSPENDED(E-BILL-CONTRACT-SUSPENDED)契約停止中
