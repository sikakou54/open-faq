API-BIL-001 利用量サマリ(プロジェクト)

この API は、プロジェクト単位の利用量を返す。

版数 v1.6更新 2026-06-17承認済

基本情報

項目内容
API IDAPI-BIL-001
API 名利用量サマリ(プロジェクト、機能 ID: F-120)
エンドポイント/usage
HTTP メソッドGET
概要プロジェクト単位の利用量を返す
利用画面SCR-021 / SCR-008
認証Cookie
権限オーナー / 該当プロジェクトの admin または member

契約全体の利用状況は API-033 GET /owner/projects/usage を使用する。

リクエストパラメータ

Query Parameters は period=current_month、viewMode=project、projectId={id}。

レスポンス

(200、viewMode=project):

{
  "period": "2026-05-01T00:00:00Z",
  "projectId": "prj_...",
  "questions": { "used": 750, "limit": 1000, "freeQuota": 1000, "percentage": 75, "resetAt": "2026-06-01T..." },
  "faqs": { "used": 85, "freeQuota": 100 }
}
