API-WHK-001 外部 Webhook(Resend)

基本情報

項目内容
API IDAPI-WHK-001
API 名外部 Webhook(Resend)
エンドポイント/webhooks/resend
HTTP メソッドPOST
概要メール配信状態 Webhook を受信し配信状態 / 抑制リストを更新する
認証Resend 署名検証(HMAC-SHA256)
権限—(署名検証のみ)

リクエストパラメータ

Query Parameters

{}

Path Parameters

{}

Request Body

(Resend 仕様準拠):

{
  "type": "email.delivered|email.bounced|email.complained|email.delivery_delayed",
  "data": {
    "message_id": "...",
    "to": "...",
    "subject": "...",
    "timestamp": "..."
  }
}
項目型説明
typeenum配信イベント種別(email.delivered / email.bounced / email.complained / email.delivery_delayed)
dataobjectイベント詳細データ
　└ message_idstring対象メールのメッセージ ID
　└ tostring宛先メールアドレス
　└ subjectstringメール件名
　└ timestampstring (ISO 8601)イベント発生時刻

処理概要

メール配信状態 Webhook を受信し配信状態 / 抑制リストを更新する。

処理 ID処理内容
P-01Resend 署名(HMAC-SHA256)を検証する(失敗時は 401)
P-02イベント種別を判定する
P-03notification_logs.delivery_state を遷移する
P-04bounced / complained の場合は email_suppression_list に追加する(全契約横断)
P-05受領応答を返す

エラー

HTTP ステータスエラーコード内容
401SIGNATURE_INVALID署名検証失敗
200(冪等)既存処理結果を返却(冪等性違反)
