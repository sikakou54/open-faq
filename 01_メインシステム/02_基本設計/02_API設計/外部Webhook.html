API-WHK-001 外部 Webhook(Resend)

基本情報

項目内容
API IDAPI-WHK-001
API 名外部 Webhook(Resend)
エンドポイント/webhooks/resend
HTTP メソッドPOST
概要メール配信状態 Webhook を受信し配信状態 / 抑制リストを更新する
利用画面(Resend Webhook)
認証Resend 署名検証(HMAC-SHA256)
権限—(署名検証のみ)

リクエストパラメータ

(Resend 仕様準拠):

{
  "type": "email.delivered|email.bounced|email.complained|email.delivery_delayed",
  "data": {
    "message_id": "...",
    "to": "...",
    "subject": "...",
    "timestamp": "..."
  }
}

動作:

  1. 署名検証(失敗時は 401)
  2. notification_logs.delivery_state を遷移
  3. bounced / complained の場合は email_suppression_list に追加(全契約横断)

エラー

HTTP ステータスエラーコード内容
401SIGNATURE_INVALID署名検証失敗
200(冪等)既存処理結果を返却(冪等性違反)
