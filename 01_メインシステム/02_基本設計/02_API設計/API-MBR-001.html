API-MBR-001 メンバー一覧

この API は、指定プロジェクトのメンバー割当一覧を返す。

版数 v1.6更新 2026-06-17承認済

基本情報

項目内容
API IDAPI-MBR-001
API 名メンバー一覧(機能 ID: F-015)
エンドポイント/projects/{id}/members
HTTP メソッドGET
概要指定プロジェクトのメンバー割当一覧を返す
利用画面SCR-009
認証Cookie
権限オーナー / 当該プロジェクト管理者

リクエストパラメータ

Path Parameters: {id}(プロジェクト ID)。Query Parameters: cursor / limit(共通仕様)。Request Body: なし。

レスポンス

(200):

{
  "items": [
    {
      "id": "01HZ...",
      "email": "member@...",
      "status": "active|pending_activation",
      "role": "admin",
      "isOwner": false,
      "projectRole": "admin",
      "invitationExpiresAt": "2026-05-20T..."
    }
  ],
  "nextCursor": "01HZ..." | null
}
補足items[] は URL で指定したプロジェクトの割当だけを返す。最終ログイン日時と他プロジェクトの割当は返さない。
