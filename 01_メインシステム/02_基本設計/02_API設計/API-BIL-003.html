API-BIL-003 請求サマリ

この API は、当月請求見込み合計・次回請求日・請求状態・プロジェクト別内訳・支払方法登録状態を返す。

版数 v1.6更新 2026-06-17承認済

基本情報

項目内容
API IDAPI-BIL-003
API 名請求サマリ(機能 ID: F-BILLING-001)
エンドポイント/billing/summary
HTTP メソッドGET
概要当月請求見込み合計・次回請求日・請求状態・プロジェクト別内訳・支払方法登録状態を返す
利用画面SCR-022 請求
認証Cookie
権限オーナー専有

Query Parameters は period=current_month。
