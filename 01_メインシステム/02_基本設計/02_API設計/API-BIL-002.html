API-BIL-002 利用量サマリ(契約)

この API は、契約全体サマリーとプロジェクト別利用量を返す。

版数 v1.6更新 2026-06-17承認済

基本情報

項目内容
API IDAPI-BIL-002
API 名利用量サマリ(契約、機能 ID: F-120)
エンドポイント/owner/projects/usage
HTTP メソッドGET
概要契約全体サマリーとプロジェクト別利用量を返す
利用画面SCR-016 利用状況
認証Cookie
権限オーナー専有

契約全体サマリーとプロジェクト別の質問数、質問数上限に対する利用率、公開 FAQ 件数、最終更新日時を返す。

リクエストパラメータ

Query Parameters は period=current_month。
