API-ANN-001 お知らせ一覧

基本情報

項目内容
API IDAPI-ANN-001
API 名お知らせ一覧
エンドポイント/me/announcements
HTTP メソッドGET
概要お知らせ受信箱の一覧を返す
認証Cookie
権限admin 限定

リクエストパラメータ

Query Parameters

{ "cursor": "共通仕様", "limit": "共通仕様" }

Path Parameters

{}

Request Body

{}

処理概要

お知らせ受信箱の一覧を返す。

処理 ID処理内容
P-01認証・認可(admin 限定)を検証する
P-02当該ユーザー宛のお知らせを取得し既読状態を付与する
P-03カーソルページネーションで整形して返却する

レスポンス(200)

{
  "items": [
    {
      "id": "ann_...",
      "title": "...",
      "category": "billing|announcement|system",
      "priority": "low|normal|high|critical",
      "bodyHtml": "<p>本文</p>",
      "readAt": "..." | null,
      "createdAt": "..."
    }
  ],
  "nextCursor": null
}
項目型説明
itemsarray<object>お知らせの配列
　└ idstringお知らせの ID
　└ titlestringお知らせのタイトル
　└ categoryenumカテゴリ。billing / announcement / system
　└ priorityenum重要度。low / normal / high / critical
　└ bodyHtmlstringお知らせ本文(HTML)
　└ readAtstring (ISO 8601)既読日時(未読時は null)
　└ createdAtstring (ISO 8601)お知らせの配信日時
nextCursorstring次ページ取得用カーソル(末尾は null)

基本情報

項目内容

エンドポイント/me/announcements/{id}/read
HTTP メソッドPOST
概要お知らせ 1 件を既読にする
認証Cookie + CSRF
権限admin 限定

リクエストパラメータ

Query Parameters

{}

Path Parameters

{ "id": "お知らせ ID" }

Request Body

{}

処理概要

お知らせ 1 件を既読にする。

処理 ID処理内容
P-01認証・認可(admin 限定、CSRF)を検証する
P-02対象お知らせ 1 件を既読に更新する(既読済みは冪等)
P-03既読時刻を返却する

レスポンス(200)

{ "id": "...", "readAt": "..." }
項目型説明
idstring既読にしたお知らせの ID
readAtstring (ISO 8601)既読日時

API-ANN-003 お知らせ一括既読

基本情報

項目内容
API IDAPI-ANN-003
API 名お知らせ一括既読
エンドポイント/me/announcements/read
HTTP メソッドPOST
概要選択した複数のお知らせを一括で既読にする(1 リクエスト上限 100 件)
認証Cookie + CSRF
権限admin 限定

選択した複数のお知らせを一括で既読にする。1 リクエストの上限は 100 件。超過時は 400 VALIDATION_ERROR。冪等(既読済みは再記録しない)。一括既読操作は操作者・対象件数・実行日時を監査記録する(FR-152 相当、正本: 09_セキュリティ設計.html)。

リクエストパラメータ

Query Parameters

{}

Path Parameters

{}

Request Body

{ "ids": ["ann_...", "ann_..."] }

(全件既読を指定する場合は { "all": true })

処理概要

選択した複数のお知らせを一括で既読にする(1 リクエスト上限 100 件)。

処理 ID処理内容
P-01認証・認可(admin 限定、CSRF)を検証する
P-02件数上限(100 件)を検証する
P-03指定または全件のお知らせを一括既読に更新する(既読済みは冪等)
P-04操作者・対象件数・実行日時を監査記録する
P-05既読件数を返却する

レスポンス(200)

{ "readCount": 8, "readAt": "..." }
項目型説明
readCountinteger今回既読にした件数
readAtstring (ISO 8601)一括既読の実行日時

エラー

HTTP ステータスエラーコード内容
400VALIDATION_ERROR100 件超過

Path Parameters
