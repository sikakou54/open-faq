1. 画面概要

新規登録後にメール内の確認リンクから本人確認を完了する画面です。送信済み・確認成功・確認失敗の 3 状態を持ち、メール再送とログインへの導線を提供します。

画面 ID 画面名 機能概要
SCR-013メール確認新規登録後にメール内リンクで本人確認を完了し、送信済み / 成功 / 失敗の各状態を表示する
関連 内容
FR / BRFR-003 / —
関連画面SCR-002 アカウント登録 / SCR-001 ログイン
ステークホルダ 対象
対象ユーザー(認証前 / トークン)◯
補足本画面は認証前(メール確認トークンによる本人確認)に表示されるため権限は不要です。確認リンクの有効期限は 24 時間で、期限切れ・使用済みの場合は確認失敗状態を表示し新規登録への復旧導線を出します。

4. 画面項目定義

本画面の入出力項目(送信済み・確認成功・確認失敗の各状態の表示・操作)を定義します。項目の正本は本表です。

項目 ID 項目 入出力 種類 備考
IT-01状態タイムライン出力Timeline「① 新規登録 → ② 確認メールのリンクをクリック → ③ ログイン」
IT-02送信先メールアドレス出力テキスト(強調)確認メール送信先
IT-03案内文出力テキスト「確認リンクの有効期限は 24 時間です」+「メールが届かない場合は迷惑メールフォルダもご確認ください」
IT-04メールを再送する入力ボタン(Secondary)レート制限 5 分以内 1 回。カウントダウン併記
IT-05メールアドレスを変更する入力リンク(Secondary)SCR-002 へ戻る
IT-06確認成功出力完了画面「メールアドレスの確認が完了しました」+「ログインする」CTA
IT-07確認失敗出力エラー画面 + 復旧 CTA「確認リンクが期限切れ、または使用済みです」+「新規登録からやり直す」CTA
IT-08ログインする入力ボタン(Primary)確認成功後 SCR-001 へ
