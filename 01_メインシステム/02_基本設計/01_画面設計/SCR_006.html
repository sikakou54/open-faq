このページは、プロジェクトの FAQ を一覧表示し、検索・絞り込み・並び替え・一括操作・CSV エクスポートと編集・新規作成・CSV インポートへの導線を提供する画面 SCR-006 を定義します。 画面概要 / 画面遷移図 / 画面レイアウト / 画面項目定義 / 入出力一覧 / 画面イベント一覧 の 6 セクションで記述します。

1. 画面概要

プロジェクトの FAQ を一覧で確認し、検索・絞り込み・並び替え・一括操作・CSV エクスポートと、編集・新規作成・CSV インポートへの導線を提供する画面です。

4. 画面項目定義

本画面の入出力項目(検索・絞り込み・並び替え・一覧の列・件数表示・一括操作・空状態を含む)を定義します。項目の正本は本表です。一覧表に「操作」列は設けず、編集遷移は FAQ ID 列のリンクに集約します(遷移リンクは ID 列に付与する全画面共通方針)。

項目 ID 項目 入出力 種類 備考
IT-01キーワード検索入力テキストボックス質問・回答の全文検索
IT-02カテゴリフィルタ入力プルダウンプロジェクト内のカテゴリ
IT-03並び順入力プルダウン関連度 / 更新日時 / 作成日時
IT-04FAQ ID出力テキストリンク一覧先頭列。faqs.id(faq_… 形式)。クリックで編集画面(SCR-006-001)へ遷移
IT-05質問出力テキスト先頭 60 文字(クリック不可)
IT-06カテゴリ出力テキスト—
IT-07状態バッジ出力バッジdraft / published / hidden。色のみ依存禁止
IT-08更新日時出力テキスト相対表記 + tooltip 絶対日時
IT-09件数表示出力テキスト「1-50 / 全 124 件」形式
IT-10行選択チェックボックス入力チェックボックス一括操作の対象選択。最大 50 件
IT-11新規作成入力ボタン(Primary)編集画面(SCR-006-001)へ遷移
IT-12一括操作バー入力BulkActionBar1 件以上選択時に下部固定。公開 / 非公開化 / 削除(操作列ではなくチェックボックス選択への一括処理)
IT-13CSV をインポート入力ボタンSCR-006-M1 モーダルを開く
IT-14CSV をエクスポート入力ボタンフィルタ適用結果を CSV(UTF-8)でダウンロード
IT-15空状態出力EmptyState「FAQ がまだありません。最初の FAQ を作成しましょう。」

本画面が読み書きするテーブル・ファイルと、呼び出す API の一覧です。テーブルの正本は 03_テーブル設計、API の正本は 02_API設計 §5.4 です。

入出力名 種別 I/O アクセス種別(CRUD) 備考
C R U D
FAQテーブル入力 / 出力—◯◯◯一覧取得・一括状態変更・一括削除(論理削除)。faqs(テーブル設計 3.9)
FAQ 一覧取得API入力————GET /faqs(status / projectId / keyword / cursor)(API 設計 5.4.1)
FAQ 一括状態変更API出力————一括の公開 / 非公開化。POST /faqs/bulk-status(API 設計 5.4.2a)

FAQ CSVファイル出力————フィルタ適用結果のエクスポート(CSV UTF-8 / ダウンロード)
