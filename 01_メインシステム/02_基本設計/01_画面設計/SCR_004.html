1. 画面概要

オーナーが契約内のプロジェクトを一覧で確認し、新規作成・編集・削除を行う画面です(オーナー専有)。作成・編集・削除動線は SCR-004-M1 モーダルに集約します。

4. 画面項目定義

本画面の入出力項目(一覧の列・件数表示・空状態を含む)を定義します。項目の正本は本表です。一覧表に「操作」列・プロジェクト ID 列・更新日時列は設けず、編集遷移はプロジェクト名(主リンク)に集約します(クリックで SCR-004-M1 を編集モードで開く)。

項目 ID 項目 入出力 種類 表示条件 備考
IT-01+ 新規プロジェクトを作成入力ボタン(Primary)—ページヘッダー右上に 1 件のみ配置
IT-02プロジェクト名出力テキストリンク(主)—一覧先頭列(編集遷移リンクを兼ねる)
IT-03許可ドメイン出力タグバッジ群—最大 3 件表示、超過分は +N 件 で折り畳み。*.example.com 形式も表示。allowed_domains
IT-04ステータス出力状態バッジ—連絡先メール確認状態。確認済み(緑)/ 確認待ち(黄)/ 未設定(灰)。projects.contact_email / projects.contact_email_verified_at から UI 自動判定。色のみ依存禁止(テキストラベル併記)
IT-05連絡先メール出力テキスト—メールアドレス本文のみ。未設定行は —
IT-06件数表示出力テキスト1 件以上ある時「1-50 / 全 N 件」形式
IT-07空状態出力EmptyState0 件時(空状態)「プロジェクトがまだありません。最初のプロジェクトを作成しましょう」+「+ 新規プロジェクトを作成」CTA
IT-08ローディング状態出力LoadingSkeleton読み込み中のみテーブル / カード行スケルトン 3 行
