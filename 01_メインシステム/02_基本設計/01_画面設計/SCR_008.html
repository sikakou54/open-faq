SCR-008 概要(プロジェクト)

このページは、選択中プロジェクトの質問数・未解決数・公開 FAQ 件数を確認し、各ワークスペース画面への導線を提供するダッシュボード画面 SCR-008 を定義します。 画面概要 / 画面遷移図 / 画面レイアウト / 画面項目定義 / 入出力一覧 / 画面イベント一覧 の 6 セクションで記述します。

版数 v1.0更新 2026-06-17承認済

1. 画面概要

選択中プロジェクトの質問数・未解決数・公開 FAQ 件数を白背景モノクロの 4 カードで確認し、数値クリックで関連画面へ遷移するプロジェクト概要画面です。

4. 画面項目定義

本画面の入出力項目(ヘッダ・期間選択・状態アラート・KPI カード)を定義します。項目の正本は本表です。各カードは項目名と件数のみを表示し、前月比・コメント・ゲージは表示しません。0 件 / 集計中 / 取得失敗時のクリック非活性ルールは 画面設計 §1.5 に従います。

項目 ID 項目 入出力 種類 表示条件 備考
IT-01PageHeader出力PageHeader—タイトルは「概要」のみ。プロジェクト名は付加せず WorkspaceSwitcher で確認
IT-02期間選択入力プルダウン + 日付ピッカー—当月 / 前月 / 任意期間(最大 13 ヶ月)
IT-03最終更新タイムスタンプ出力テキスト—KPI カード群右上に「最終更新: YYYY-MM-DD HH:MM(JST)」。5 分以上前は「集計遅延」黄表記
IT-04サスペンション中の表示出力アラート契約状態が停止中(サスペンション)の場合のみ表示「現在ご利用いただけません」+ オーナー連絡導線(non-owner)/ 支払方法更新リンク(owner → SCR-022)
IT-05質問数出力SummaryCard—期間内の質問総数。数値クリック → SCR-005(?project_id={現在})
IT-06未解決数出力SummaryCard—期間内の未解決質問数。数値クリック → SCR-005(?status=open&project_id={現在})
IT-07公開 FAQ 件数出力SummaryCard—当該プロジェクトの公開 FAQ 件数。数値クリック → SCR-006 FAQ 一覧
IT-08無料利用枠超過バナー出力メッセージ / バナー無料利用枠を超過した場合のみ表示「無料利用枠を超過しました」+ 支払方法未登録時はウィジェット制限中の旨と「支払い方法を登録」導線(オーナー → SCR-022)。文言の正本は 06_メッセージ一覧、課金・上限ルールの正本は 10_課金・請求設計
IT-09質問数上限到達バナー出力メッセージ / バナー質問数が月次上限に到達した場合のみ表示「質問数が上限に達しました」+ ウィジェット制限中の旨と「利用量と上限へ」導線(SCR-021)。文言の正本は 06_メッセージ一覧、上限ルールの正本は 10_課金・請求設計

5. 入出力一覧

本画面が読み書きするテーブルと、呼び出す API の一覧です。テーブルの正本は 03_テーブル設計、API の正本は 02_API設計 §5.8a.1 です。本画面は集計取得のみで永続更新は行いません。
