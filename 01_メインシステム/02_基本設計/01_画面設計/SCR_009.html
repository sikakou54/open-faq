SCR-009 メンバー(プロジェクト)

このページは、当該プロジェクトに割り当てたメンバーを一覧表示し、招待・ロール変更・割当解除モーダルへの導線を提供する画面 SCR-009 を定義します。 画面概要 / 画面遷移図 / 画面レイアウト / 画面項目定義 / 入出力一覧 / 画面イベント一覧 の 6 セクションで記述します。

版数 v1.0更新 2026-06-17承認済

1. 画面概要

当該プロジェクトに割当のあるメンバーを一覧表示し、招待・ロール変更・割当解除モーダル(SCR-009-M1)への導線を提供する画面です。表示範囲は常に当該プロジェクト 1 件で、契約横断のメンバー管理は持ちません。

画面 ID 画面名 機能概要
SCR-009メンバー(プロジェクト)当該プロジェクトのメンバー一覧表示・絞り込みと、招待 / 編集モーダルへの導線を提供する

オーナー◯
プロジェクト管理者(admin)◯
メンバー(member)—
補足オーナーは暗黙の admin(project_users(role='admin', valid=1) を自動保持)として全プロジェクトを対象にできます。プロジェクト管理者の対象範囲は当該プロジェクトのみです。当該プロジェクトに admin ロールを持たないユーザー(member 含む)の URL 直アクセスは 403 → ダッシュボードへリダイレクトします。

4. 画面項目定義

本画面の入出力項目(絞り込み・一覧の列・件数表示・空状態を含む)を定義します。項目の正本は本表です。一覧表に「操作」列は設けず、編集遷移は利用者表示名のリンクに集約します(遷移リンクは名称列に付与する全画面共通方針)。一覧のバッジ表記のみ短縮形「管理者」を用い、編集モーダル・FR の正式呼称「プロジェクト管理者」とは表記レベルで分離します。

項目 ID 項目 入出力 種類 表示条件 備考
IT-01検索入力テキストボックス—表示名・メールの部分一致
IT-02招待状態フィルタ入力プルダウン—すべて / 招待中のみ / アクティベーション済み
IT-03件数表示出力テキスト—「1-50 / 全 N 件」形式
IT-04利用者表示名出力テキストリンクオーナー行・自分の行はテキスト表示のみ(リンク化しない)。招待中は名前隣に「招待中」バッジクリックで SCR-009-M1 編集モードを開く。users.name
IT-05メールアドレス出力テキスト—users.email
IT-06このプロジェクトでのロール出力バッジ—オーナー(青、project_users(role='admin') 自動付与・削除対象外)/ 管理者(緑、当該 PJ に admin)/ メンバー(灰、当該 PJ に member)。契約横断の「プロジェクト別ロール」列は持たない
IT-07ステータス出力バッジ—利用中 / 招待中 の 2 値。残日数は併記しない。users.status(active / pending_activation)
IT-08招待中行強調出力行背景色対象者が招待中(本人未有効化)の行のみ黄色背景一覧での視認性を確保
IT-09+ メンバーを招待入力ボタン(Primary)—SCR-009-M1 を「招待」モード(当該プロジェクトの 1 件の割当のみを扱う)で開く
IT-10空状態出力EmptyState割当メンバーが 0 件のとき「このプロジェクトにはまだメンバーが割当されていません。」
IT-11権限不足ガード出力PermissionTooltip当該プロジェクトのプロジェクト管理者権限を持たないユーザーが URL に直接アクセスした場合403 → ダッシュボードへリダイレクト
