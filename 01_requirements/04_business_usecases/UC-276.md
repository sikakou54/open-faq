# <span id="UC-276"></span>UC-276: 未解決時にウィジェット利用者へ案内する

> **AI が解決できなかった場合に未解決質問を登録しつつ、ウィジェット利用者へ案内することを定義します。**

*主アクター ウィジェット利用者 ・ ステータス ドラフト ・ 再構成 P7後続第2段*

| 項目 | 内容 |
|---|---|
| 業務ユースケースID | UC-276 |
| 業務ユースケース名 | 未解決時にウィジェット利用者へ案内する |
| 対応要件ID | [FR-136](../02_FunctionalRequirement/04_widget-fr.md#FR-136) ・ [BR-090](../01_BusinessRequirement/04_widget-br.md#BR-090) |
| 主アクター | ウィジェット利用者 |
| 目的 | AI が解決できなかった場合に未解決質問を登録しつつ、ウィジェット利用者へ案内することを定義します。 |

## 事前条件

ウィジェット利用者が質問し、AI が解決できなかった。

## 基本フロー

1. トリガー: AI が質問を解決できないと判定される。
2. システムが問い合わせ ID を付与して未解決質問を登録する(ID はウィジェットに表示しない)。
3. 会話履歴を保持したまま、回答できなかった旨と設定済みなら確認済み連絡先メールをシステム返信として表示する。
4. 別の FAQ 質問は引き続き入力・送信できる。
