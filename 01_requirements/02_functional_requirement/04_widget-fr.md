# ウィジェット・検索・入出力(機能要件)

> **このページは、ウィジェット・検索・入出力 に関する機能要件を定義します。**

*複数の個別要件を統合したカテゴリ別ページ。 ステータス ドラフト*

| ID | 名称 | 優先度 | 要件 |
|----|----|----|----|
| <span id="FR-125"></span>FR-125 | 埋め込みコードの取得と設置 | P0 | アカウント利用者は埋め込みコードを取得し、自社サイトに設置できること。 |
| <span id="FR-126"></span>FR-126 | 許可ドメイン上での動作制限 | P0 | ウィジェットは指定された許可ドメイン上でのみ動作すること。 |
| <span id="FR-127"></span>FR-127 | ウィジェット外観のプロジェクト別設定 | P1 | ウィジェットの基本的な見た目をプロジェクトごとに設定できること。設定可能項目は主色(プライマリカラー)とし、本サービスの提供元ロゴを必須表示とする。 |
| <span id="FR-128"></span>FR-128 | モバイル端末での利用対応 | P0 | ウィジェットはモバイル端末でも利用できること。 |
| <span id="FR-129"></span>FR-129 | ウィジェットのアクセシビリティ対応 | P1 | ウィジェットがアクセシビリティ要件(キーボード操作、スクリーンリーダー、コントラスト等)に配慮していること。 |
| <span id="FR-130"></span>FR-130 | サポート対象ブラウザでの動作 | P0 | ウィジェットはサポート対象ブラウザで動作すること。 |
| <span id="FR-131"></span>FR-131 | ウィジェット配信の高速化 | P1 | ウィジェット配信は高速化(CDN / キャッシュ)できること。 |
| <span id="FR-132"></span>FR-132 | ランチャーバッジとチャット UI の展開 | P0 | ウィジェットは初期状態で丸型のランチャーバッジとして表示し、クリックまたはキーボード操作でチャット UI を展開すること。チャット UI は質問と回答の履歴・質問入力欄・送信操作を備え、閉じると元のランチャーバッジへ戻る。 |
| <span id="FR-133"></span>FR-133 | チャット UI ヘッダーの表示要素 | P0 | チャット UI のヘッダーはウィジェットタイトル、現在状態、チャット UI を閉じるボタンのみを表示し、操作メニューや利用規約・プライバシーポリシーへの導線は表示しないこと。 |
| <span id="FR-134"></span>FR-134 | 受付制限中の案内表示 | P0 | 月次上限到達または支払方法ゲートにより新規質問を受け付けない場合、ウィジェットは受付制限中の案内をシステム返信として表示し、質問入力と送信を無効化すること。 |
| <span id="FR-135"></span>FR-135 | 未解決質問の登録と案内表示 | P0 | AI が質問を解決できなかった場合、未解決質問として登録しつつ、回答できなかった旨と連絡先メールをシステム返信として表示すること。会話履歴は保持したまま別の FAQ 質問は引き続き受け付ける。 |
| <span id="FR-164"></span>FR-164 | FAQ 検索と質問ログ検索 | P0 | アカウント利用者が FAQ のキーワード全文検索および質問ログのキーワード・期間検索を行えること。検索結果は並び替えとページングで参照でき、対象は現在開いているプロジェクトに限る。 |
| <span id="FR-165"></span>FR-165 | FAQ の CSV 一括取り込み | P1 | FAQ を CSV 形式(UTF-8、BOM 許容)で一括取り込みできること。行に含まれる FAQ 識別子の有無で新規登録か上書きかを判定し、部分失敗時は成功分を反映したうえで失敗理由をアカウント利用者が確認できる。 |
| <span id="FR-166"></span>FR-166 | FAQ の CSV 書き出し | P1 | FAQ を CSV 形式(UTF-8)で書き出し(エクスポート)できること。 |
