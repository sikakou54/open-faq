# セキュリティ・プライバシー(機能要件)

> **このページは、セキュリティ・プライバシー に関する機能要件を定義します。**

*複数の個別要件を統合したカテゴリ別ページ。 ステータス ドラフト*

| ID | 名称 | 優先度 | 要件 |
|----|----|----|----|
| <span id="FR-136"></span>FR-136 | 退会の即時実行と厳格な確認 | P0 | オーナーによる退会は即時に実行し、猶予期間は設けないこと。実行前に退会の影響を明示し、登録メールアドレスの入力一致と本人確認(再認証)を必須とする。 |
| <span id="FR-137"></span>FR-137 | 退会時のデータ削除と請求データの保持 | P0 | 退会時に運用データ(FAQ・プロジェクト・質問ログ等)を速やかに削除し、アカウント・請求関連データは法定保持期間(7年)にわたり保持すること。削除は不可逆とし、削除内容を監査記録として残す。 |
| <span id="FR-138"></span>FR-138 | 利用規約・プライバシーポリシー改定時に再同意を取得 | P0 | 利用規約・プライバシーポリシー改定時に再同意を取得できること。改定された文書のみを再同意の対象とし、文書ごとに個別に同意を取得する。 |
| <span id="FR-139"></span>FR-139 | 退会後の請求限定アクセス | P0 | 退会後も、アカウント・請求データの保持期間(7年)中はオーナーがログインでき、請求情報の閲覧のみ行えること。FAQ・プロジェクト等の操作および新規書込みはすべて不可とする。 |
| <span id="FR-140"></span>FR-140 | 通信の暗号化(HTTPS) | P0 | 通信は暗号化されること(HTTPS)。 |
| <span id="FR-141"></span>FR-141 | 機密項目の暗号化・ハッシュ保存 | P0 | 保存データのうち機密度の高い項目(パスワード、トークン、課金情報等)は暗号化または安全なハッシュで保存されること。 |
| <span id="FR-142"></span>FR-142 | API・ウィジェットのレート制限 | P0 | API・ウィジェットに対してレート制限ができること。 |
| <span id="FR-143"></span>FR-143 | 埋め込み元ドメインの検証・制限 | P0 | ウィジェットの埋め込み元ドメインを検証できること(許可ドメイン以外からの読み込み拒否)。 |
| <span id="FR-144"></span>FR-144 | 利用者の操作ログを監査用に保持 | P0 | 利用者の操作ログを監査用に保持できること。 |
| <span id="FR-145"></span>FR-145 | 最小権限によるアクセス制御・データ分離 | P0 | アクセス制御は最小権限の原則に従うこと(オーナー境界によるデータ分離、ユーザー種別別認可)。 |
| <span id="FR-146"></span>FR-146 | Bot 対策(レート制限・ロックアウト等)を必要に応じて実施 | P1 | Bot・不正自動アクセス対策を、ログイン連続失敗・未認証の再送・不審なリクエスト検知の各契機で実施できること。アカウントロックやレート制限により自動試行・短時間の連続要求を抑止する。 |
| <span id="FR-147"></span>FR-147 | 不審なリクエスト(大量アクセス、未許可ドメイン等)を検知・記録 | P0 | 不審なリクエスト(大量アクセス、未許可ドメイン等)を検知・記録できること。 |
| <span id="FR-148"></span>FR-148 | メンバー管理に関する操作を監査ログに記録 | P0 | メンバー管理に関する操作(招待・招待再送・プロジェクト割当変更・離脱・削除)を監査ログに記録できること。記録項目は操作者・対象メンバー・変更前後の差分・実行時刻を含む。 |
| <span id="FR-149"></span>FR-149 | ウィジェット公開キーのローテーション機能を利用者操作で提供 | P0 | ウィジェット公開キーのローテーション機能を利用者操作で提供すること。新キー発行と同時に旧キーを24時間の猶予期間付きの失効予告状態とし、猶予期間の経過後に旧キーを完全失効とする。 |
| <span id="FR-150"></span>FR-150 | 公開キーの無期限運用 | P0 | ウィジェット公開キーには有効期限を設けないこと(無期限)。キーの失効は再発行(ローテーション)によってのみ行う。 |
| <span id="FR-191"></span>FR-191 | 保持期間超過データの定期削除 | P0 | 保持期間を超過した運用データ(質問ログ・未解決質問・通知ログ・お知らせ受信箱)を定期的に検出し、対象に応じた方式で削除すること。削除を実行した結果を記録する。 |
