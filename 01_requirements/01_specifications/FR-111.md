# <span id="FR-111"></span>FR-111: KPI / 一覧 / アラートの各表示要素は、0 件・未集計・集計中・取

> **この機能要件は「KPI / 一覧 / アラートの各表示要素は、0 件・未集計・集計中・取」を定義します。**

*種別 機能要件 ・ 機能グループ 管理ダッシュボード ・ 優先度 P0 ・ ステータス ドラフト*

## <span id="要件"></span>要件

KPI / 一覧 / アラートの各表示要素は、0 件・未集計・集計中・取得失敗・期間外 の状態を区別して表示できること。状態ごとの文言と配色ルールは基本設計の画面設計に正本化し、各画面はそれを参照する
