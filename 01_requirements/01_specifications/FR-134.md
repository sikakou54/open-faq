# <span id="FR-134"></span>FR-134: チャット UI のヘッダーはウィジェットタイトル、現在状態、チャット U

> **この機能要件は「チャット UI のヘッダーはウィジェットタイトル、現在状態、チャット U」を定義します。**

*種別 機能要件 ・ 機能グループ ウィジェット ・ 優先度 P0 ・ ステータス ドラフト*

## <span id="要件"></span>要件

チャット UI のヘッダーはウィジェットタイトル、現在状態、チャット UI を閉じるボタンのみを表示すること。ハンバーガーメニューその他の操作メニュー、および「利用規約」「プライバシーポリシー」への導線は表示しない

## <span id="関連"></span>関連

| 関連業務要件 | 優先度 | 対応業務UC |
|----|----|----|
| [BR-089](BR-089.md#BR-089) | P0 | (表示・UX要件 — 業務UC対象外) |

P7 後続レビューで、本要件は表示・UX要件のため独立した業務UCを持たないと分類した。
