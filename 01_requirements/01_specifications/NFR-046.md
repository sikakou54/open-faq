# <span id="NFR-046"></span>NFR-046: データ保持

> **この非機能要件は「データ保持」区分の要件を定義します。**

*種別 非機能要件 ・ 分類 データ保持 ・ ステータス ドラフト*

## <span id="要件"></span>要件

データ保持期間はシステム固定値(NFR-055 / NFR-057 の値)とし、契約・管理者ユーザーによる UI 上の変更は不可とする。短縮が必要な場合は法令・契約義務範囲内でサポート窓口経由の対応とする
