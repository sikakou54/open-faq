# <span id="BR-140"></span>BR-140: 権限外 UI の扱い

> **この業務要件は「権限外 UI の扱い」を定義します。**

*種別 業務要件 ・ 機能グループ アクセス制御細部 ・ 優先度 P0 ・ ステータス ドラフト*

## <span id="要件"></span>要件

- **アカウント利用者：**現在の管理範囲をヘッダーおよびアカウントメニューで確認できる
- **システム：**権限外の主ナビゲーションと操作は原則表示せず、権限のない画面への直アクセスは説明と戻り先を添えて拒否する
