# <span id="FR-109"></span>FR-109: 主要指標(KPI)カード(質問数 / 未解決数 / 公開 FAQ 件数等

> **この機能要件は「主要指標(KPI)カード(質問数 / 未解決数 / 公開 FAQ 件数等」を定義します。**

*種別 機能要件 ・ 機能グループ 管理ダッシュボード ・ 優先度 P1 ・ ステータス ドラフト*

## <span id="要件"></span>要件

主要指標(KPI)カード(質問数 / 未解決数 / 公開 FAQ 件数等)はクリックで該当一覧画面へ遷移できること。
1. 遷移先とフィルタ条件は基本設計の画面設計に明記する
2. 0 件・集計中・取得失敗の状態ではクリック不可とする
