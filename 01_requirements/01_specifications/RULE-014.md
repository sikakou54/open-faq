# <span id="RULE-014"></span>RULE-014: 質問数アラート閾値

> **この業務ルールは「質問数アラート閾値」の定量しきい値・ポリシーを定義します。**

*種別 業務ルール ・ ステータス ドラフト*

## <span id="ルール"></span>ルール

質問数上限に対する 25% / 50% / 80% / 90% / 100% のアラート閾値を複数選択でき、当月初回到達時に通知する。
