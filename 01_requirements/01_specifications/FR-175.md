# <span id="FR-175"></span>FR-175: 利用者向け管理コンソールは、管理スコープ(契約範囲 / プロジェクト範囲

> **この機能要件は「利用者向け管理コンソールは、管理スコープ(契約範囲 / プロジェクト範囲」を定義します。**

*種別 機能要件 ・ 機能グループ UX 細部・データ運用 ・ 優先度 P1 ・ ステータス ドラフト*

## <span id="要件"></span>要件

利用者向け管理コンソールは、管理スコープ(契約範囲 / プロジェクト範囲 / 認証フロー / 共通領域)を画面から判別できるよう分離すること。スコープと各画面の対応の正本は基本設計の画面設計に置く
