# <span id="FR-158"></span>FR-158: アカウント利用者はお知らせを種別(請求確定 / 運営お知らせ / システ

> **この機能要件は「アカウント利用者はお知らせを種別(請求確定 / 運営お知らせ / システ」を定義します。**

*種別 機能要件 ・ 機能グループ お知らせ ・ 優先度 P1 ・ ステータス ドラフト*

## <span id="要件"></span>要件

アカウント利用者はお知らせを種別(請求確定 / 運営お知らせ / システム通知)、既読状態、期間で絞り込みできること
