# <span id="FR-012"></span>FR-012: オーナーおよび当該プロジェクトのメンバーは、自分が割り当てられたプロジェ

> **この機能要件は「オーナーおよび当該プロジェクトのメンバーは、自分が割り当てられたプロジェ」を定義します。**

*種別 機能要件 ・ 機能グループ ユーザー管理(オーナー + メンバー) ・ 優先度 P0 ・ ステータス ドラフト*

## <span id="要件"></span>要件

オーナーおよび当該プロジェクトのメンバーは、自分が割り当てられたプロジェクトでメンバーを招待できること。
1. 招待はメールアドレスを指定して行う(プロジェクト内の役割差はない)
2. 招待時に氏名(表示名)は入力しない(個人情報保護の原則により、他人の氏名を招待者が任意に登録することを禁止する)。氏名は招待された本人が有効化時に入力する(FR-013d)
3. メンバーの管理は常にプロジェクト単位で行う
4. 同一の人を複数プロジェクトに参加させる場合は各プロジェクトで個別に招待する
5. 同一オーナー配下でメールアドレスが一致する場合は同一の利用者として扱い、新たに参加するプロジェクト分の割当のみを追加する
