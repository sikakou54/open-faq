# <span id="BR-063"></span>BR-063: 上限アラート通知

> **この業務要件は「上限アラート通知」を定義します。**

*種別 業務要件 ・ 機能グループ 利用量・課金 ・ 優先度 P0 ・ ステータス ドラフト*

## <span id="要件"></span>要件

プロジェクトごとに選択された質問数上限アラート閾値(25% / 50% / 80% / 90% / 100%)へ当月初回到達した時点で、オーナー + 当該プロジェクトの有効なメンバーへメール送信する(同一メールアドレスは重複排除)。全閾値未選択時はアラート通知なしとする。
