# <span id="FR-178"></span>FR-178: 主ナビゲーションはユーザーの目的語を優先して構成

> **この機能要件は「主ナビゲーションはユーザーの目的語を優先して構成」を定義します。**

*種別 機能要件 ・ 機能グループ UX 細部・データ運用 ・ 優先度 P0 ・ ステータス ドラフト*

## <span id="要件"></span>要件

主ナビゲーションはユーザーの目的語を優先して構成すること。
1. 契約範囲では「利用状況 / プロジェクト / 請求 / 設定」の 4 項目だけを用いる
2. プロジェクト範囲では「概要 / 要対応の質問 / FAQ / ウィジェット / メンバー / 利用量と上限」を用いる
3. 配置の例外は次のとおりとする
    1. プロジェクトの作成・編集・削除は契約範囲の「プロジェクト」へ置く
    2. 通知はヘッダーへ置く
    3. 法務文書はヘルプまたはフッターへ置く
