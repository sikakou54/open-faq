# <span id="FR-094"></span>FR-094: プロジェクトごとに質問数上限に対する 25% / 50% / 80% /

> **この機能要件は「プロジェクトごとに質問数上限に対する 25% / 50% / 80% /」を定義します。**

*種別 機能要件 ・ 機能グループ 利用量・課金 ・ 優先度 P0 ・ ステータス ドラフト*

## <span id="要件"></span>要件

プロジェクトごとに質問数上限に対する **25% / 50% / 80% / 90% / 100%** のアラート閾値を複数選択でき、選択した各閾値へ当月初回到達した時点で、オーナー + 当該プロジェクトの有効なメンバーへメール送信できること(同一メールアドレスは重複排除)。送信先は固定とし、全閾値未選択はアラート通知なしとする。FAQ 件数は上限アラート対象外とする
