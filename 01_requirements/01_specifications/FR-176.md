# <span id="FR-176"></span>FR-176: 利用状況でプロジェクトが 0 件の場合は、説明文と「新規プロジェクトを作

> **この機能要件は「利用状況でプロジェクトが 0 件の場合は、説明文と「新規プロジェクトを作」を定義します。**

*種別 機能要件 ・ 機能グループ UX 細部・データ運用 ・ 優先度 P1 ・ ステータス ドラフト*

## <span id="要件"></span>要件

利用状況でプロジェクトが 0 件の場合は、説明文と「新規プロジェクトを作成」する導線だけを持つ空状態を表示すること。段階型のセットアップチェックリストは表示しない
