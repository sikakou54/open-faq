# <span id="FR-112"></span>FR-112: 利用状況でプロジェクトが 0 件の場合は、複雑なセットアップチェックリス

> **この機能要件は「利用状況でプロジェクトが 0 件の場合は、複雑なセットアップチェックリス」を定義します。**

*種別 機能要件 ・ 機能グループ 管理ダッシュボード ・ 優先度 P2 ・ ステータス ドラフト*

## <span id="要件"></span>要件

利用状況でプロジェクトが 0 件の場合は、複雑なセットアップチェックリストを表示せず、「プロジェクトがまだありません」と「新規プロジェクトを作成」する導線を持つ空状態を表示すること
