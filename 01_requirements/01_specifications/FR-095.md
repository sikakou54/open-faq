# <span id="FR-095"></span>FR-095: 各機能に対して契約・種別別のレート制限を DDoS / Bot / 暴走

> **この機能要件は「各機能に対して契約・種別別のレート制限を DDoS / Bot / 暴走」を定義します。**

*種別 機能要件 ・ 機能グループ 利用量・課金 ・ 優先度 P0 ・ ステータス ドラフト*

## <span id="要件"></span>要件

各機能に対して契約・種別別のレート制限を DDoS / Bot / 暴走対策(セキュリティ目的)として適用すること。本レート制限は契約単位で管理し、プロジェクト単位化(FR-065 / FR-066)の対象外とすること(月次上限件数・無料枠のみプロジェクト単位)

P7 後続(第2段)で本要件を実現する業務UC [UC-263](../02_business_usecases/UC-263.md#UC-263) を新設した。
