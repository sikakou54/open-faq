# <span id="UC-277"></span>UC-277: 利用状況と概要を迷わず行き来する

> **利用状況と各プロジェクト概要の間を迷わず行き来できる導線を定義します。**

*主アクター オーナー ・ ステータス ドラフト ・ 再構成 P7後続第2段*

| 項目 | 内容 |
|---|---|
| 業務ユースケースID | UC-277 |
| 業務ユースケース名 | 利用状況と概要を迷わず行き来する |
| 対応要件ID | [FR-106](../01_specifications/FR-106.md#FR-106) ・ [BR-074](../01_specifications/BR-074.md#BR-074) |
| 主アクター | オーナー |
| 目的 | 利用状況と各プロジェクト概要の間を迷わず行き来できる導線を定義します。 |

## 事前条件

オーナーがダッシュボードを表示している。

## 基本フロー

1. 契約全体の利用状況は当月固定・読み取り専用で表示する。
2. プロジェクトを開く・作成・編集は契約範囲の「プロジェクト」に集約する。
3. 各プロジェクトの概要へは管理範囲の切替から1操作で遷移する。
4. オーナーは1操作で利用状況へ戻れる。
