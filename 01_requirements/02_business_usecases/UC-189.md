# <span id="UC-189"></span>UC-189: Turnstile を実行

> **Turnstile の CAPTCHA トークンを取得し、失敗時は登録ボタンを無効化する、クライアント内処理のみのユースケース。**

*主アクター 招待メンバー(トークン) ・ ステータス ドラフト ・ 再構成 P2*

| 項目 | 内容 |
|---|---|
| 業務ユースケースID | UC-189 |
| 業務ユースケース名 | Turnstile を実行 |
| 対応要件ID | [FR-149](../01_specifications/06_security-fr.md#FR-149) |
| 主アクター | 招待メンバー(トークン) |
| 目的 | Turnstile の CAPTCHA トークンを取得し、失敗時は登録ボタンを無効化する、クライアント内処理のみのユースケース。 |

## 事前条件

Turnstile ウィジェット(IT-09)が表示されている

## 基本フロー

1. Turnstile ウィジェット(IT-09)が検証を行う。
2. 成功時、画面は CAPTCHA トークンを取得・保持する。

## 代替フロー

—(本イベントは単一の正常フロー。条件分岐は基本フローに含む)

## 例外フロー

- 検証失敗時: エラーを表示し「登録を完了する」(IT-10)を無効化する。

## 事後条件

CAPTCHA トークンを取得する。検証失敗時はエラーを表示し「登録を完了する」(IT-10)を無効化する
