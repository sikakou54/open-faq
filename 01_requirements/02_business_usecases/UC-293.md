# <span id="UC-293"></span>UC-293: ウィジェット配信を高速化する

> **ウィジェット配信を CDN / キャッシュにより高速化できることを定義します。**

*主アクター システム(バッチ・通知・非同期) ・ ステータス ドラフト ・ 再構成 P7後続第2段*

| 項目 | 内容 |
|---|---|
| 業務ユースケースID | UC-293 |
| 業務ユースケース名 | ウィジェット配信を高速化する |
| 対応要件ID | [FR-132](../01_specifications/FR-132.md#FR-132) ・ [BR-093](../01_specifications/BR-093.md#BR-093) |
| 主アクター | システム(バッチ・通知・非同期) |
| 目的 | ウィジェット配信を CDN / キャッシュにより高速化できることを定義します。 |

## 事前条件

ウィジェット配信基盤が稼働している。

## 基本フロー

1. トリガー: ウィジェットの配信リクエストが発生する。
2. システムが CDN / キャッシュを介して配信する。
3. 利用者へ高速にウィジェットが配信される。

## 事後条件

ウィジェットが高速に配信される。
