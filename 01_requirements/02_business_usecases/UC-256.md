# <span id="UC-256"></span>UC-256: 許可ドメインを複数設定する

> **ウィジェットの動作を許可する Web サイトのドメインを、プロジェクトに複数設定できることを定義します。**

*主アクター プロジェクトメンバー ・ ステータス ドラフト ・ 再構成 P7後続第2段*

| 項目 | 内容 |
|---|---|
| 業務ユースケースID | UC-256 |
| 業務ユースケース名 | 許可ドメインを複数設定する |
| 対応要件ID | [FR-042](../01_specifications/FR-042.md#FR-042) ・ [BR-023](../01_specifications/BR-023.md#BR-023) |
| 主アクター | プロジェクトメンバー |
| 目的 | ウィジェットの動作を許可する Web サイトのドメインを、プロジェクトに複数設定できることを定義します。 |

## 事前条件

対象プロジェクトを開いている。

## 基本フロー

1. 利用者が許可ドメイン設定を開く。
2. 利用者が許可するドメインを複数入力する。
3. システムが入力ドメインを検証して保存する。

## 代替フロー

—

## 例外フロー

不正な形式のドメインは保存せずエラーを表示する。

## 事後条件

複数の許可ドメインが保存され、ウィジェット動作判定に用いられる。
