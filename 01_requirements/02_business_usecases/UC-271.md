# <span id="UC-271"></span>UC-271: 送信レート・バウンス率・苦情率を監視して送信抑制する

> **契約 / プロジェクト単位で送信レート・バウンス率・苦情率を監視し、しきい値超過時に送信抑制することを定義します。**

*主アクター システム(バッチ・通知・非同期) ・ ステータス ドラフト ・ 再構成 P7後続第2段*

| 項目 | 内容 |

| 業務ユースケースID | UC-271 |

| 対応要件ID | [FR-117](../01_specifications/05_notification.md#FR-117) ・ [BR-083](../01_specifications/05_notification.md#BR-083) |
| 主アクター | システム(バッチ・通知・非同期) |
| 目的 | 契約 / プロジェクト単位で送信レート・バウンス率・苦情率を監視し、しきい値超過時に送信抑制することを定義します。 |

## 事前条件

通知送信実績が集計されている。しきい値が設定されている。

## 基本フロー

1. トリガー: 通知送信または定期監視を契機に指標を評価する。
2. システムが契約 / プロジェクト単位で送信レート・バウンス率・苦情率を監視する。
3. しきい値超過時はシステムが送信を抑制する。
