# <span id="UC-294"></span>UC-294: ランチャーバッジからチャット UI を展開する

> **ウィジェットを丸型ランチャーバッジとして表示し、操作でチャット UI を展開して会話できることを定義します。**

*主アクター ウィジェット利用者 ・ ステータス ドラフト ・ 再構成 P7後続第2段*

| 項目 | 内容 |
|---|---|
| 業務ユースケースID | UC-294 |
| 業務ユースケース名 | ランチャーバッジからチャット UI を展開する |
| 対応要件ID | [FR-133](../01_specifications/FR-133.md#FR-133) ・ [BR-089](../01_specifications/BR-089.md#BR-089) |
| 主アクター | ウィジェット利用者 |
| 目的 | ウィジェットを丸型ランチャーバッジとして表示し、操作でチャット UI を展開して会話できることを定義します。 |

## 事前条件

設置サイトにウィジェットが設置されている。

## 基本フロー

1. トリガー: 利用者が丸型ランチャーバッジをクリックまたはキーボード操作する。
2. システムがチャット UI を展開する(質問回答履歴・入力欄・送信操作を備える)。
3. 利用者が質問を送信し回答を得る。
4. 閉じる操作でランチャーバッジ表示へ戻る。

## 代替フロー

同一ページ内では会話履歴と受付状態を保持する。

## 例外フロー

—

## 事後条件

チャット UI が展開され会話でき、閉じるとバッジへ戻る。
