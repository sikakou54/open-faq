# <span id="UC-001"></span>UC-001: 初期表示

> **ログインフォームを表示し、既認証セッションが有効な場合は管理範囲に応じた初期画面へリダイレクトする軽量ユースケース。**

*主アクター 未認証ユーザー(既認証セッションを持つ場合あり) ・ ステータス ドラフト ・ 再構成 P2*

| 項目 | 内容 |
|---|---|
| 業務ユースケースID | UC-001 |
| 業務ユースケース名 | 初期表示 |
| 対応要件ID | [FR-004](../01_specifications/FR-004.md#FR-004) |
| 主アクター | 未認証ユーザー(既認証セッションを持つ場合あり) |
| 目的 | ログインフォームを表示し、既認証セッションが有効な場合は管理範囲に応じた初期画面へリダイレクトする軽量ユースケース。 |

## 事前条件

SCR-001 の URL にアクセスした

## 基本フロー

1. 画面がセッション Cookie の有無を確認する。
2. セッションが無い、または無効な場合はログインフォーム(IT-01〜IT-05)を表示して終了する。
3. セッションが有効な場合は利用者種別を判定し、オーナーは SCR-021 利用状況、メンバーは SCR-012 概要へリダイレクトする。
4. 未同意の規約改定がある場合は SCR-020 規約再同意へ割込み遷移する。

## 代替フロー

—(本イベントは単一の正常フロー。条件分岐は基本フローに含む)

## 例外フロー

- セッションが失効済み・検証失敗の場合はリダイレクトせず、ログインフォームを表示する。

## 事後条件

未認証時はログインフォームを表示する。既認証かつセッション有効時は管理範囲に応じた初期画面(オーナーは SCR-021 利用状況 / メンバーは SCR-012 概要)へ遷移する
