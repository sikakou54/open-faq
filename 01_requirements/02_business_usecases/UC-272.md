# <span id="UC-272"></span>UC-272: 要対応質問の状況を対応中・対応済みで確認する

> **要対応の質問の状況を、アカウント利用者向けに「対応中 / 対応済み」で確認できることを定義します。**

*主アクター プロジェクトメンバー ・ ステータス ドラフト ・ 再構成 P7後続第2段*

| 項目 | 内容 |
|---|---|
| 業務ユースケースID | UC-272 |
| 業務ユースケース名 | 要対応質問の状況を対応中・対応済みで確認する |
| 対応要件ID | [FR-101](../01_specifications/FR-101.md#FR-101) ・ [BR-070](../01_specifications/BR-070.md#BR-070) |
| 主アクター | プロジェクトメンバー |
| 目的 | 要対応の質問の状況を、アカウント利用者向けに「対応中 / 対応済み」で確認できることを定義します。 |

## 事前条件

要対応の質問が存在する。

## 基本フロー

1. 利用者が要対応の質問の状況を表示する。
2. システムが「対応中 / 対応済み」で状況を表示する。
