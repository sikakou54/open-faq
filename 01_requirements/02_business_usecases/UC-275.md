# <span id="UC-275"></span>UC-275: 受付制限中の案内を表示する

> **月次上限到達または支払方法ゲートで新規質問を受け付けない場合に、受付制限中の案内を表示することを定義します。**

*主アクター ウィジェット利用者 ・ ステータス ドラフト ・ 再構成 P7後続第2段*

| 項目 | 内容 |

| 業務ユースケース名 | 受付制限中の案内を表示する |
| 対応要件ID | [FR-135](../01_specifications/04_widget-fr.md#FR-135) ・ [BR-090](../01_specifications/04_widget-br.md#BR-090) |

| 目的 | 月次上限到達または支払方法ゲートで新規質問を受け付けない場合に、受付制限中の案内を表示することを定義します。 |

## 事前条件

月次上限到達または支払方法ゲートにより新規質問を受け付けない状態である。

## 基本フロー

1. トリガー: 新規質問を受け付けられない状態でウィジェットが利用される。
2. システムが受付制限中の旨をチャット内システム返信として表示する。
3. 確認済みプロジェクト連絡先メールがある場合はそのアドレスを案内する。
4. 制限中は質問入力と送信を無効化する。
