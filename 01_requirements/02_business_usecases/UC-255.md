# <span id="UC-255"></span>UC-255: プロジェクト削除時の関連データ取扱いを確認・選択する

> **プロジェクト削除時に FAQ・質問ログ・未解決質問の取扱いをアカウント利用者が確認・選択できることを定義します。**

*主アクター プロジェクトメンバー ・ ステータス ドラフト ・ 再構成 P7後続第2段*

| 項目 | 内容 |
|---|---|
| 業務ユースケースID | UC-255 |
| 業務ユースケース名 | プロジェクト削除時の関連データ取扱いを確認・選択する |
| 対応要件ID | [FR-045](../01_specifications/01_account-fr.md#FR-045) ・ [BR-026](../01_specifications/01_account-br.md#BR-026) |
| 主アクター | プロジェクトメンバー |
| 目的 | プロジェクト削除時に FAQ・質問ログ・未解決質問の取扱いをアカウント利用者が確認・選択できることを定義します。 |

## 事前条件

削除対象プロジェクトに関連データが存在する。

## 基本フロー

1. 利用者がプロジェクト削除を開始する。
2. システムが関連データ(FAQ・質問ログ・未解決質問)の取扱い選択肢を提示する。
3. 利用者が取扱いを確認・選択する。
4. システムが選択に従い削除処理を進める。

## 代替フロー

—

## 例外フロー

選択を中断した場合は削除を実行しない。

## 事後条件

利用者の選択に従って関連データが処理される。
