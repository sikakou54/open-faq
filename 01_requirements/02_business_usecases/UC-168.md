# <span id="UC-168"></span>UC-168: プライバシーポリシー同意をチェック

> **プライバシーポリシー同意チェックの状態を切り替え、両チェック充足に応じて続行ボタンの活性を更新する、クライアント内処理のみのユースケース。**

*主アクター オーナー / メンバー ・ ステータス ドラフト ・ 再構成 P2*

| 項目 | 内容 |
|---|---|
| 業務ユースケースID | UC-168 |
| 業務ユースケース名 | プライバシーポリシー同意をチェック |
| 対応要件ID | [FR-010](../01_specifications/FR-010.md#FR-010) |
| 主アクター | オーナー / メンバー |
| 目的 | プライバシーポリシー同意チェックの状態を切り替え、両チェック充足に応じて続行ボタンの活性を更新する、クライアント内処理のみのユースケース。 |

## 事前条件

プライバシーポリシーが改定対象で同意チェック(IT-03)が表示されている

## 基本フロー

1. 利用者がプライバシーポリシー同意(IT-03)をチェック / アンチェックする。
2. 画面は両チェックの充足状態を判定し、IT-04 の活性 / 非活性を更新する。

## 代替フロー

—(本イベントは単一の正常フロー。条件分岐は基本フローに含む)

## 例外フロー

- なし(画面内の状態更新のみ)。

## 事後条件

チェック状態を切り替え、両チェックの充足状態に応じて「同意して続行する」(IT-04)の活性 / 非活性を更新する
