# <span id="UC-295"></span>UC-295: チャット UI ヘッダーの表示要素を限定する

> **チャット UI ヘッダーにタイトル・現在状態・閉じるボタンのみを表示し、操作メニュー等を表示しないことを定義します。**

*主アクター ウィジェット利用者 ・ ステータス ドラフト ・ 再構成 P7後続第2段*

| 項目 | 内容 |
|---|---|
| 業務ユースケースID | UC-295 |
| 業務ユースケース名 | チャット UI ヘッダーの表示要素を限定する |
| 対応要件ID | [FR-134](../01_specifications/FR-134.md#FR-134) ・ [BR-089](../01_specifications/BR-089.md#BR-089) |
| 主アクター | ウィジェット利用者 |
| 目的 | チャット UI ヘッダーにタイトル・現在状態・閉じるボタンのみを表示し、操作メニュー等を表示しないことを定義します。 |

## 事前条件

チャット UI が展開されている。

## 基本フロー

1. トリガー: チャット UI が展開される。
2. システムがヘッダーにウィジェットタイトル・現在状態・閉じるボタンのみを表示する。
3. ハンバーガーメニューその他の操作メニュー、利用規約・プライバシーポリシー導線は表示しない。

## 代替フロー

—

## 例外フロー

—

## 事後条件

ヘッダーが限定された要素のみで表示される。
