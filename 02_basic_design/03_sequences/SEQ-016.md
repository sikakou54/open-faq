## 概要

利用者が要対応の質問一覧で状況フィルタをチェックすると、選択した状況条件に一致する未解決質問で一覧と件数を更新する。一致が 0 件のときは空状態を表示する。

## シーケンス図

```mermaid
sequenceDiagram
  autonumber
  actor U as 利用者
  participant Screen as 要対応の質問一覧
  participant Server as サーバー
  participant DB as DB
  U->>Screen: 状況フィルタをチェック
  Screen->>Server: 状況条件付きで未解決質問一覧を要求
  Server->>DB: 認証情報・データを取得
  DB-->>Server: 取得結果
  Server-->>Screen: 一致した未解決質問の一覧・件数
  alt 一致あり
    Screen-->>U: 一覧・件数を更新
  else 一致なし
    Screen-->>U: 空状態を表示
  end
```
