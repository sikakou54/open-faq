# <span id="SEQ-043"></span>SEQ-043: 初期表示

## 概要

メンバー画面を開いたとき、当該プロジェクトのメンバー一覧と件数を表示する。割当が 0 件のときは空状態を表示する。

## シーケンス図

```mermaid
sequenceDiagram
  autonumber
  actor U as 利用者
  participant Screen as メンバー(プロジェクト)
  participant Server as サーバー
  participant DB as DB
  U->>Screen: 画面を開く(初期表示)
  Screen->>Server: メンバー一覧取得要求
  Server->>Server: 認証・認可
  Server->>DB: 認証情報・データを取得
  DB-->>Server: 取得結果
  Server-->>Screen: メンバー一覧(0 件もありうる)
  alt 1 件以上
    Screen-->>U: 一覧と件数を表示
  else 0 件
    Screen-->>U: 空状態を表示
  end
```

## 備考

- 本図は基本設計レベルの抽象度(ユーザー / 画面 / サーバー、システム起点は外部システム・スケジューラ・バッチを加える)で記述する。DB 操作は DB アクターへのメッセージで表し、テーブル別 CRUD は本図に書かず 関連テーブル 欄で示す。
- 図の出典は業務ユースケース [UC-018](../../01_requirements/04_business_usecases/UC-018.md#UC-018)。画面イベントとの対応は UC-018 を参照。
