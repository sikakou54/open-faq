# <span id="SEQ-022"></span>SEQ-022: 「対応済みにする」を押下

> **このページは、業務ユースケース UC-031（メンバーが未解決質問の対応状況を更新する）のシーケンス図を定義します。**

| ID | 業務ユースケースID | イベント(画面ID EVT-NN) | テーブルID |
|----|----|----|----|
| SEQ-022 | [UC-031](../../01_requirements/04_business_usecases/UC-031.md#UC-031) | SCR-007 EVT-03 | [TBL-017](../02_backend/04_database/TBL-017.md#TBL-017) |

## 概要

要対応の質問詳細画面で「対応済みにする」を押下すると確認ダイアログを表示し、確定で対応状況を「対応済み」に更新して状況バッジを更新する。取り消した場合は変更しない。

## シーケンス図

```mermaid
sequenceDiagram
  autonumber
  actor U as 利用者
  participant Screen as 要対応の質問詳細
  participant Server as サーバー
  participant DB as DB
  U->>Screen: 「対応済みにする」を押下
  Screen->>U: 確認ダイアログを表示
  alt 確定(OK)
    Screen->>Server: 対応状況の更新要求(対応済み)
    Server->>Server: 入力検証
    Server->>DB: 業務データを更新
    DB-->>Server: 更新完了
    alt 更新成功
      Server-->>Screen: 更新後の対応状況
      Screen-->>U: 状況バッジを更新
    else 更新失敗
      Server-->>Screen: エラー
      Screen-->>U: エラー通知・変更前に戻す
    end
  else 取り消し(キャンセル)
    Screen-->>U: 変更せずダイアログを閉じる
  end
```
