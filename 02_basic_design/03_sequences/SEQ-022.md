## 概要

要対応の質問詳細画面で対応状況に「対応中」を選択すると、対応状況を対応中に保存し、状況バッジを更新する。

## シーケンス図

```mermaid
sequenceDiagram
  autonumber
  actor U as 利用者
  participant Screen as 要対応の質問詳細
  participant Server as サーバー
  participant DB as DB
  U->>Screen: 対応状況で「対応中」を選択
  Screen->>Server: 対応状況の更新要求(対応中)
  Server->>Server: 入力検証
  Server->>DB: 業務データを更新
  DB-->>Server: 更新完了
  alt 更新成功
    Server-->>Screen: 更新後の対応状況
    Screen-->>U: 状況バッジを更新
  else 更新失敗
    Server-->>Screen: エラー
    Screen-->>U: エラー通知・選択値を戻す
  end
```
