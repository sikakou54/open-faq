# <span id="SEQ-020"></span>SEQ-020: ページを選択

## 概要

未解決質問が 2 ページ以上ある状態で、利用者がページネーションから別ページを選択すると、選択したページの未解決質問で一覧を更新する。

## シーケンス図

```mermaid
sequenceDiagram
  autonumber
  actor U as 利用者
  participant Screen as 要対応の質問一覧
  participant Server as サーバー
  participant DB as DB
  U->>Screen: ページを選択
  Screen->>Server: 該当ページの未解決質問一覧を要求
  Server->>DB: 認証情報・データを取得
  DB-->>Server: 取得結果
  Server-->>Screen: 該当ページの未解決質問一覧
  Screen-->>U: 一覧・件数を更新
```
