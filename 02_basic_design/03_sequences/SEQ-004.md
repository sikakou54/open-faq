## 概要

メールアドレスの形式を検証してパスワード再設定要求をサーバーへ送信し、対象アカウントが存在する場合のみ再設定リンクメールを送る。サーバーは存在有無を区別しない一律応答を返し、画面は送信済み案内と再送ボタンを表示する。

## シーケンス図

```mermaid
sequenceDiagram
  autonumber
  actor U as 利用者
  participant Screen as パスワード再設定
  participant Server as サーバー
  participant MAIL as メール配信IF
  U->>Screen: 送信を押下
  Screen->>Screen: 入力検証
  Screen->>Server: パスワード再設定要求
  Server->>Server: 認証・データ取得
  alt 対象アカウントあり
    Server->>MAIL: 再設定リンクメール送信依頼
    MAIL-->>Server: 受理
  else 対象アカウントなし
    Note over Server: メール送信せず処理継続
  end
  Server-->>Screen: 一律応答(存在有無を区別しない)
  Screen-->>U: 送信済み案内・再送ボタンを表示
```

## 例外フロー

- 形式不正: 入力欄直下にエラーメッセージを表示し、要求を中断する。
- 要求失敗: 送信済み案内へ遷移せず、エラーを表示する(列挙攻撃対策のため対象有無は明かさない)。
