## 概要

設置サイトでウィジェットが読み込まれると、サーバーが読み込み元のドメインを取得し、対象プロジェクトに登録された許可ドメインと照合する。読み込み元が許可ドメインに一致する場合のみウィジェットを動作させ、対象プロジェクトの範囲内でFAQ・回答を取り扱う。許可ドメインに一致しない場合、および許可ドメインが設定されていない場合はウィジェットを動作させず、不正設置や他契約・他プロジェクトのデータ参照を防ぐ。

## シーケンス図

```mermaid
sequenceDiagram
  autonumber
  participant Site as 設置サイト(外部)
  participant Server as サーバー
  participant DB as DB
  Site->>Server: ウィジェットの読み込み(読み込み元ドメインを含む)
  Server->>Server: 読み込み元のドメインを取得する
  Server->>DB: 対象プロジェクトを特定する
  DB-->>Server: 取得結果
  alt 対象プロジェクトに許可ドメインが設定されている
    Server->>DB: 読み込み元ドメインを許可ドメインと照合する
    DB-->>Server: 照合結果
    alt 読み込み元が許可ドメインに一致する
      Server->>Server: ウィジェットを動作させ対象プロジェクト範囲に限定する
      Server-->>Site: ウィジェットを動作させFAQ対応を提供する
    else 許可ドメインに一致しない
      Server-->>Site: ウィジェットを動作させない
    end
  else 許可ドメインが設定されていない
    Server-->>Site: ウィジェットを動作させない
  end
```
