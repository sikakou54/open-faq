## 概要

質問数上限設定モーダルを開いた際に、当該プロジェクトの上限 ON/OFF・件数・アラート閾値を取得し、各項目の初期値・活性状態へ反映する。

## シーケンス図

```mermaid
sequenceDiagram
  autonumber
  actor U as オーナー / メンバー
  participant Screen as 質問数上限設定モーダル
  participant Server as サーバー
  participant DB as DB
  U->>Screen: モーダルを開く(初期表示)
  Screen->>Server: 上限・アラート設定の取得要求
  Server->>DB: 認証情報・データを取得
  DB-->>Server: 取得結果
  Server-->>Screen: 上限 ON/OFF・件数・アラート閾値
  alt 上限あり(ON)
    Screen-->>U: 件数・閾値を初期値に設定し各項目を活性化
  else 上限なし(OFF)
    Screen-->>U: 上限項目を非活性で表示
  end
```
