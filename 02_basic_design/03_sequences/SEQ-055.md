## 概要

お知らせ一覧で「すべてクリア」を押下し、適用中のフィルタ条件をすべて解除して一覧を更新する。解除後は適用済フィルタチップを非表示にする。

## シーケンス図

```mermaid
sequenceDiagram
  autonumber
  actor U as 利用者
  participant Screen as お知らせ一覧
  participant Server as サーバー
  participant DB as DB
  U->>Screen: 「すべてクリア」を押下
  Screen->>Screen: 適用中のフィルタ条件を解除
  Screen->>Server: お知らせ一覧取得要求(条件なし)
  Server->>DB: 認証情報・データを取得
  DB-->>Server: 取得結果
  Server-->>Screen: お知らせ一覧
  Screen-->>U: 一覧を更新し適用済フィルタチップを非表示
```
