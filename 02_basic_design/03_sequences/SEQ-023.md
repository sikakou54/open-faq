# <span id="SEQ-023"></span>SEQ-023: 確認ダイアログの「OK」を押下

> **このページは、業務ユースケース UC-031（確認ダイアログの「OK」を押下）のシーケンス図を定義します。**

## 概要

要対応の質問詳細で「対応済み」を選んで表示した確認ダイアログの「OK」を押下すると、対応状況を `closed`（対応済み）に保存し、状況バッジを更新する。

## シーケンス図

```mermaid
sequenceDiagram
  autonumber
  actor U as 利用者
  participant Screen as 要対応の質問詳細
  participant Server as サーバー
  participant DB as DB
  U->>Screen: 確認ダイアログの「OK」を押下
  Screen->>Server: 状況切替要求（対応済み）
  Server->>Server: 認証・認可・入力検証
  Server->>DB: 業務データを更新
  DB-->>Server: 更新完了
  alt 更新成功
    Server-->>Screen: 更新済み（対応済み）
    Screen-->>U: 状況バッジを更新・ダイアログを閉じる
  else 更新失敗
    Server-->>Screen: エラー
    Screen-->>U: エラートースト・選択値を戻す
  end
```
