## 概要

メンバー画面で招待状態フィルタ（すべて / 招待中のみ / アクティベーション済み）を選択すると、選択条件に一致するメンバーで一覧と件数表示を更新する。0 件のときは空状態を表示する。

## シーケンス図

```mermaid
sequenceDiagram
  autonumber
  actor U as 利用者
  participant Screen as メンバー(プロジェクト)
  participant Server as サーバー
  U->>Screen: 招待状態フィルタを選択
  Screen->>Server: メンバー一覧要求(状態条件付き)
  Server->>Server: 認証・認可
  Server->>Server: 認証・データ取得
  Server-->>Screen: 一致したメンバー一覧
  alt 該当あり
    Screen-->>U: 一覧・件数を更新
  else 該当なし
    Screen-->>U: 空状態を表示
  end
```
