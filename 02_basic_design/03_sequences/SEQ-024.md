## 概要

FAQ 一覧画面を開いたとき、状態・プロジェクトの条件で FAQ を取得して一覧表示する。取得結果が 0 件のときは空状態を表示する。

## シーケンス図

```mermaid
sequenceDiagram
  autonumber
  actor U as 利用者
  participant Screen as FAQ 一覧
  participant Server as サーバー
  participant DB as DB
  U->>Screen: 画面を開く(初期表示)
  Screen->>Server: FAQ 一覧取得要求
  Server->>Server: 認証・認可
  Server->>DB: 認証情報・データを取得
  DB-->>Server: 取得結果
  Server-->>Screen: FAQ 一覧(0 件もありうる)
  alt 1 件以上
    Screen-->>U: 一覧と件数を表示
  else 0 件
    Screen-->>U: 空状態を表示
  end
```
