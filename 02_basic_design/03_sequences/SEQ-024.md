## 概要

FAQ 一覧画面を開いたとき、状態・プロジェクト・キーワードの条件で FAQ を取得して一覧表示する。取得結果が 0 件のときは空状態を表示する。

## シーケンス図

```mermaid
sequenceDiagram
  autonumber
  actor U as 利用者
  participant Screen as FAQ 一覧
  participant Server as サーバー
  U->>Screen: 画面を開く(初期表示)
  Screen->>Server: FAQ 一覧取得要求
  Server->>Server: 認証・認可
  Server->>Server: 認証・データ取得
  Server-->>Screen: FAQ 一覧(0 件もありうる)
  alt 1 件以上
    Screen-->>U: 一覧と件数を表示
  else 0 件
    Screen-->>U: 空状態を表示
  end
```

## 備考

- 本図は基本設計レベルの抽象度(ユーザー / 画面 / サーバー、システム起点は外部システム・スケジューラ・バッチを加える)で記述する。DB 操作はサーバー自己メッセージで表し、テーブル別 CRUD は本図に書かず 関連テーブル 欄で示す。
