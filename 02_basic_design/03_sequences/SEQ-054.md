## 概要

お知らせ一覧画面を開いたとき、未読のみを既定条件にお知らせ一覧と未読件数を取得して表示する。1 件以上あれば一覧と件数を表示し、0 件のときは空状態を表示する。

## シーケンス図

```mermaid
sequenceDiagram
  autonumber
  actor U as オーナー / 当該スコープのメンバー
  participant Screen as お知らせ一覧
  participant Server as サーバー
  U->>Screen: 画面を開く(初期表示)
  Screen->>Server: お知らせ一覧(未読のみ)取得要求
  Server->>Server: 認証・データ取得
  Server-->>Screen: お知らせ一覧(0 件もありうる)
  Screen->>Server: 未読件数取得要求
  Server->>Server: 認証・データ取得
  Server-->>Screen: 未読件数
  alt 1 件以上
    Screen-->>U: 一覧と未読件数を表示
  else 0 件
    Screen-->>U: 空状態を表示
  end
```
