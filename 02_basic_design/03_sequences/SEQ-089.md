## 概要

質問送信の応答が未解決のとき、回答できなかった旨をシステム返信として表示し、質問ログと未解決質問を登録する。連絡先メール設定済みのときは連絡先メールを案内表示し、別の質問の入力・送信は引き続き可能とする。

## シーケンス図

```mermaid
sequenceDiagram
  autonumber
  actor U as ウィジェット利用者
  participant Screen as エンドユーザー向け FAQ ウィジェット
  participant Server as サーバー
  participant DB as DB
  Note over Screen: 直前の質問送信の応答が未解決
  Screen-->>U: 回答できなかった旨をシステム返信で表示
  Screen->>Server: 未解決質問の登録要求
  Server->>Server: 入力検証
  Server->>DB: 業務データを更新
  DB-->>Server: 更新完了
  alt 登録成功
    Server-->>Screen: 登録完了
    Screen-->>U: 連絡先メールを案内(設定済みのとき)
  else 登録失敗
    Server-->>Screen: エラー
    Screen-->>U: エラー表示・再試行案内(連打防止)
  end
```

## 例外フロー

- 未解決質問の登録に失敗した場合は、ウィジェットにエラーを表示し再試行を案内する(連打防止)。
