## 概要

当該プロジェクトへの割当がないログイン済みユーザーが利用量と上限画面の URL に直接アクセスした際、権限不足ガードにより 403 を返し、ダッシュボードへリダイレクトする。

## シーケンス図

```mermaid
sequenceDiagram
  autonumber
  actor U as 当該プロジェクトに割当のないログイン済みユーザー
  participant Screen as 利用量と上限(プロジェクト単位)
  participant Server as サーバー
  participant DB as DB
  U->>Screen: 画面の URL へ直接アクセス
  Screen->>Server: 利用量サマリ取得要求
  Server->>Server: 認証・認可判定(対象プロジェクトへのアクセス権)
  alt アクセス権あり
    Server->>DB: 認証情報・データを取得
    DB-->>Server: 取得結果
    Server-->>Screen: 利用量サマリを返却
    Screen-->>U: 利用量と上限を表示
  else 権限不足(割当なし)
    Server-->>Screen: 権限不足(アクセス不可)
    Screen-->>U: ダッシュボードへリダイレクト
  end
```

## 例外フロー

- 当該プロジェクトへの割当がない場合、サーバーは認可判定でアクセス不可と判断し、画面はダッシュボードへリダイレクトする。
