## 概要

オーナーが利用状況画面を開くと、契約全体サマリーとプロジェクト別の利用状況を取得して表示する。プロジェクトが 0 件のときは空状態を表示する。

## シーケンス図

```mermaid
sequenceDiagram
  autonumber
  actor U as オーナー
  participant Screen as 利用状況
  participant Server as サーバー
  participant DB as DB
  U->>Screen: 画面を開く(初期表示)
  Screen->>Server: 契約全体サマリーとプロジェクト別利用量を要求
  Server->>DB: 認証情報・データを取得
  DB-->>Server: 取得結果
  Server-->>Screen: 契約全体サマリー・プロジェクト別利用量
  alt プロジェクト 1 件以上
    Screen-->>U: 全体サマリー・プロジェクト別を表示(必要時 支払い警告)
  else 0 件
    Screen-->>U: 空状態を表示
  end
```
