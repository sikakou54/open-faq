## 概要

招待リンクから開いた際に招待トークンを検証し、招待情報(プロジェクト名 / 招待元)を取得する。成功時は招待情報パネルと入力フォームを表示し、無効 / 期限切れ・使用済みの場合は各エラー画面を表示する。

## シーケンス図

```mermaid
sequenceDiagram
  autonumber
  actor U as 利用者
  participant Screen as メンバーアカウント有効化
  participant Server as サーバー
  participant DB as DB
  U->>Screen: 招待リンクから開く(初期表示)
  Screen->>Server: 招待トークン検証・プレビュー要求
  Server->>DB: 認証情報・データを取得
  DB-->>Server: 取得結果
  Server-->>Screen: 検証結果(成功 / 期限切れ / 使用済み)
  alt 成功
    Screen-->>U: 招待情報・入力フォームを表示
  else 無効 / 期限切れ
    Screen-->>U: トークン無効 / 期限切れエラー画面を表示
  else 使用済み
    Screen-->>U: 既使用エラー画面を表示
  end
```

## 例外フロー

- トークン無効 / 期限切れ: トークン無効 / 期限切れエラー画面を表示する(有効期限 7 日)。
- トークン使用済み: 既使用エラー画面を表示する。
- トークン不存在: トークンが存在しない旨のエラーを表示する。
