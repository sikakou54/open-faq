## 概要

オーナーがプロジェクト画面を開くと、契約配下のプロジェクト一覧を取得して件数とともに表示する。取得中はローディング状態、0 件のときは空状態を表示する。

## シーケンス図

```mermaid
sequenceDiagram
  autonumber
  actor U as オーナー
  participant Screen as プロジェクト
  participant Server as サーバー
  participant DB as DB
  U->>Screen: 画面を開く(初期表示)
  Screen->>U: ローディング状態を表示
  Screen->>Server: プロジェクト一覧取得要求
  Server->>Server: 認証・認可(オーナー専有)
  Server->>DB: 認証情報・データを取得
  DB-->>Server: 取得結果
  Server-->>Screen: 一覧(0 件もありうる)
  alt 1 件以上
    Screen-->>U: 一覧と件数を表示
  else 0 件
    Screen-->>U: 空状態を表示
  end
```
