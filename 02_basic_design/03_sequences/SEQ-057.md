## 概要

利用者がお知らせ一覧で対象お知らせの ID リンクを押下すると、サーバーが当該お知らせを既読化し、お知らせ詳細画面へ遷移する。

## シーケンス図

```mermaid
sequenceDiagram
  autonumber
  actor U as 利用者
  participant Screen as お知らせ一覧
  participant Server as サーバー
  participant DB as DB
  U->>Screen: お知らせ ID リンクを押下
  Screen->>Server: 既読化要求
  Server->>Server: 認証・認可を検証
  Server->>DB: 業務データを更新
  DB-->>Server: 更新完了
  alt 既読化成功
    Server-->>Screen: 既読化完了
    Screen-->>U: お知らせ詳細へ遷移
  else 既読化失敗
    Server-->>Screen: エラー応答
    Screen-->>U: 既読化エラーを表示し遷移は続行
  end
```
