## 概要

FAQ 一覧画面でキーワードを入力すると、その条件に一致する FAQ を取得して一覧と件数を更新する。

## シーケンス図

```mermaid
sequenceDiagram
  autonumber
  actor U as オーナー / メンバー
  participant Screen as FAQ 一覧
  participant Server as サーバー
  participant DB as DB
  U->>Screen: キーワードを入力
  Screen->>Screen: 入力検証
  Screen->>Server: キーワード条件で FAQ 一覧を要求
  Server->>DB: 認証情報・データを取得
  DB-->>Server: 取得結果
  Server-->>Screen: 一致した FAQ 一覧
  Screen-->>U: 一覧・件数を更新
```

## 備考

- 本図は基本設計レベルの抽象度(ユーザー / 画面 / サーバー、システム起点は外部システム・スケジューラ・バッチを加える)で記述する。DB 操作は DB アクターへのメッセージで表し、テーブル別 CRUD は本図に書かず 関連テーブル 欄で示す。
