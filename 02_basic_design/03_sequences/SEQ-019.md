## 概要

要対応の質問一覧で検索ボックスにフリーワードを入力すると、問い合わせ ID または質問テキストに一致する未解決質問で一覧を更新する。一致が 0 件のときは空状態を表示する。

## シーケンス図

```mermaid
sequenceDiagram
  autonumber
  actor U as 利用者
  participant Screen as 要対応の質問一覧
  participant Server as サーバー
  participant DB as DB
  U->>Screen: 検索ボックスにフリーワードを入力
  Screen->>Screen: 入力検証
  Screen->>Server: フリーワード条件付きで一覧取得を要求
  Server->>DB: 認証情報・データを取得
  DB-->>Server: 取得結果
  Server-->>Screen: 一致した未解決質問の一覧
  alt 一致あり
    Screen-->>U: 一覧・件数を更新
  else 0 件
    Screen-->>U: 空状態を表示
  end
```
