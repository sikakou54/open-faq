## 概要

アカウント利用者が削除確認ダイアログで OK を押下すると、対象 FAQ を論理削除する。成功時は FAQ 一覧へ遷移し、失敗時はダイアログを閉じてエラーを表示する。

## シーケンス図

```mermaid
sequenceDiagram
  autonumber
  actor U as アカウント利用者
  participant Screen as FAQ 一覧
  participant Server as サーバー
  participant DB as DB
  U->>Screen: 削除確認 OK を押下
  Screen->>Server: FAQ 論理削除を要求
  Server->>Server: 認証・認可を検証
  Server->>Server: 入力検証
  alt 版衝突(楽観ロック不一致)
    Server-->>Screen: 競合エラー
    Screen-->>U: ダイアログを閉じエラー表示
  else 削除成功
    Server->>DB: 業務データを更新
    DB-->>Server: 更新完了
    Server-->>Screen: 成功
    Screen-->>U: FAQ 一覧へ遷移
  end
```

## 例外フロー

- 楽観ロック競合(`version` 不一致)時は削除を中断し、ダイアログを閉じてエラーを表示する（[ERR-023](../05_errors/ERR-023.md#ERR-023)）。
- 入力検証エラー時は削除を中断し、エラーを表示する（[ERR-001](../05_errors/ERR-001.md#ERR-001)）。
