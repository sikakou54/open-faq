## <span id="body"></span>3. テンプレート定義

発火条件・テンプレート変数・件名・本文(テキスト / HTML 版)です。本文の件名・テキストは送信される文言そのものの正本です。

月次請求の確定をオーナーへ通知する `high` メールです。請求サマリと明細 PDF(添付 + DL URL)・請求画面 deeplink を提示します。

**発火条件**: FR-120 月次請求確定バッチ完了時、当該課金アカウントのオーナー(自分が作成したプロジェクト全体の請求名義)へ送信。請求は各プロジェクトの内訳明細を含めてオーナー宛に 1 通にまとめる。

**テンプレート変数**:

| 変数                       | 説明                            |
|----------------------------|---------------------------------|
| `{recipient_display_name}` | オーナー表示名                  |
| `{billing_period_label}`   | 対象期間(`2026 年 4 月`)        |
| `{invoice_amount_jpy}`     | 請求金額(¥1,234)                |
| `{question_count}`         | 質問数                          |
| `{overage_amount_jpy}`     | 上限超過分(¥0 の場合は記載省略) |
| `{invoice_pdf_url}`        | 明細 PDF DL URL(有効期限はシステム仕様書 §4) |
| `{billing_screen_url}`     | SCR-028 請求 deeplink           |

**件名**: `{billing_period_label} の請求書が発行されました(¥{invoice_amount_jpy})`

**本文(テキスト版)**:

```json
{recipient_display_name} 様

open-faq の {billing_period_label} ご利用分の請求書を発行しました。

  - 請求金額: ¥{invoice_amount_jpy}
  - 対象期間: {billing_period_label}
  - 質問数: {question_count}
  - 上限超過分: ¥{overage_amount_jpy}

明細 PDF を添付しています。また以下からも有効期限内に DL いただけます。

▶ 請求明細 PDF を開く
{invoice_pdf_url}

請求書一覧および過去の請求は以下からご確認いただけます。

▶ 請求画面を開く
{billing_screen_url}

──────────────────────────────────────────────
このメールに直接返信しないでください。
open-faq <noreply@<service-domain>>
```

**本文(HTML 版)**: 請求サマリ表 + プライマリボタン「請求画面を開く」+ PDF DL セカンダリボタン。

**備考**: PDF は §2.7 のとおり必ず添付。ダウンロード有効期限は [システム仕様書 §4](../07_system-spec.md#4-データ保持期間削除猶予) を参照する。`{overage_amount_jpy}` が `0` の場合は当該行を出力しない。
