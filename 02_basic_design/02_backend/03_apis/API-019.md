# <span id="API-019"></span>API-019: ウィジェット鍵ローテーション

> **このページは ウィジェット鍵ローテーション API の仕様を定義します。** 公開キーを再発行する(旧キーは [システム仕様書 §4](../../07_system-spec.md#4-データ保持期間削除猶予) の猶予で失効)。

| ID | 業務ユースケースID | イベント(画面ID EVT-NN) | テーブルID |
|----|----|----|----|
| API-019 | [UC-039](../../../01_requirements/04_business_usecases/UC-039.md#UC-039) | SCR-011 EVT-09 | [TBL-004](../04_database/TBL-004.md#TBL-004) ・ [TBL-015](../04_database/TBL-015.md#TBL-015) |

| API名 | エンドポイント | HTTPメソッド |
|----|----|----|
| ウィジェット鍵ローテーション | `/projects/{id}/widget-key/rotate` | POST |

## 処理概要

公開キーを再発行する(旧キーは [システム仕様書 §4](../../07_system-spec.md#4-データ保持期間削除猶予) の猶予で失効)。

| 処理 ID | 処理内容                                       |
|---------|------------------------------------------------|
| P-00    | 再認証トークン(リクエストボディ `reauthToken`)を検証する |
| P-01    | 認証・認可(オーナー / 当該プロジェクトのメンバー)と再認証を検証する     |
| P-02    | 新しいウィジェット公開キーを発行して有効化する |
| P-03    | 旧キーを [システム仕様書 §4](../../07_system-spec.md#4-データ保持期間削除猶予) の猶予で失効予約する(猶予期限 `grace_until` を記録。失効の実体は失効バッチではなく、ウィジェット起動 [API-037](API-037.md#API-037) の起動時キー判定で猶予期限超過を失効として扱う遅延失効方式で行う) |
| P-04    | 新キーと失効予告を返す                         |

## リクエスト

#### 認証ヘッダ

本 API は利用者セッション(Cookie)と CSRF トークンを必要とする状態変更系である。

| ヘッダ | 要否 | 説明 |
|---|---|---|
| `Cookie: session=<token>` | 必須 | 利用者セッション |
| `X-CSRF-Token: <token>` | 必須 | CSRF 対策(状態変更系) |
| `Idempotency-Key: <ULID>` | 任意 | 冪等性(書き込み再送の重複防止) |

> 本操作は再認証必須。直近の再認証([API-005](API-005.md#API-005))で得た再認証状態を伴う必要がある。

#### Query Parameters

```json
{}
```

#### Path Parameters

```json
{ "id": "プロジェクト ID" }
```

| 項目 | 型 | 説明 |
|----|----|----|
| `id` | string | 鍵をローテーションする対象プロジェクトの ID(必須・パス内で指定) |

#### Request Body

```json
{ "reauthToken": "reauth_..." }
```

| 項目 | 型 | 説明 |
|----|----|----|
| `reauthToken` | string | 再認証トークン(必須・API-005 で取得) |

公開キーは無期限のため有効期限指定パラメータは持たない。

## レスポンス

**レスポンス(201)**

```json
{ "newKey": "pk_live_xyz789...", "deprecationWarning": "..." }
```

| 項目                 | 型     | 説明                                   |
|----------------------|--------|----------------------------------------|
| `newKey`             | string | 再発行したウィジェット公開キー         |
| `deprecationWarning` | string | 旧キーの [システム仕様書 §4](../../07_system-spec.md#4-データ保持期間削除猶予) の猶予失効予告メッセージ |

(`deprecationWarning` は旧キーの [システム仕様書 §4](../../07_system-spec.md#4-データ保持期間削除猶予) の猶予失効予告)
