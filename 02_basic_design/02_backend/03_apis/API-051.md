# <span id="API-051"></span>API-051: お知らせ未読件数

> **このページは お知らせ未読件数 API の契約を定義します。** 未読件数と直近お知らせを返す。

## 項目

| 項目 | 内容 |
|---|---|
| API ID | `API-051` |
| API名 | お知らせ未読件数 |
| 対応業務UC | [UC-045](../../../01_requirements/04_business_usecases/UC-045.md#UC-045) ・ [UC-088](../../../01_requirements/04_business_usecases/UC-088.md#UC-088) |
| 対応画面ID | [SCR-016](../../01_frontend/01_screens/SCR-016.md#SCR-016) |
| 対応画面イベントID | EVT-136 |
| エンドポイント | `/me/announcements/unread-summary` |
| HTTPメソッド | GET |
| 認証 | Cookie |
| 認可 | オーナー / メンバー |

## 処理概要

未読件数と直近お知らせを返す。

| 処理 ID | 処理内容                               |
|---------|----------------------------------------|
| P-01    | 認証・認可(オーナー / メンバー)を検証する       |
| P-02    | 未読件数を集計し直近お知らせを取得する |
| P-03    | 整形して返却する                       |

## レスポンス

**レスポンス(200)**

```json
{ "unreadsCount": 5, "recent": [...] }
```

| 項目 | 型 | 説明 |
|----|----|----|
| `unreadsCount` | integer | 未読お知らせの件数 |
| `recent` | array\<object\> | 直近お知らせの配列(各要素は API-048 の `items` と同形式) |
