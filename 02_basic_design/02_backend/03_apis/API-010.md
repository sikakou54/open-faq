# <span id="API-010"></span>API-010: パスワード再設定確定

> **このページは パスワード再設定確定 API の仕様を定義します。** 再設定トークンと新しいパスワードを受け取り、パスワードハッシュを更新して当該ユーザーの全セッションを失効する。

| ID | 業務ユースケースID | イベント(画面ID EVT-NN) | テーブルID |
|----|----|----|----|
| API-010 | [UC-005](../../../01_requirements/04_business_usecases/UC-005.md#UC-005) | SCR-003 EVT-04 ・ SCR-003 EVT-06 | [TBL-002](../04_database/TBL-002.md#TBL-002) ・ [TBL-003](../04_database/TBL-003.md#TBL-003) ・ [TBL-013](../04_database/TBL-013.md#TBL-013) ・ [TBL-014](../04_database/TBL-014.md#TBL-014) |

| API名 | エンドポイント | HTTPメソッド |
|----|----|----|
| パスワード再設定確定 | `/auth/password-reset` | POST |

## 処理概要

再設定トークンと新しいパスワードを受け取り、パスワードハッシュを更新して当該ユーザーの全セッションを失効する。

| 処理 ID | 処理内容 |
|----|----|
| P-01 | 再設定トークンの有効性(期限・使用済み)を検証する |
| P-02 | トークンから対象アカウント(ユーザー)を特定する |
| P-03 | 新パスワードの強度要件(FR-006: 12 文字以上・英大小文字 / 数字 / 記号 3 種類以上)を検証する |
| P-04 | 対象アカウントのパスワードハッシュを更新する |
| P-05 | 再設定トークンを使用済みにする |
| P-06 | 当該ユーザーの全セッションを失効する |
| P-07 | 確定結果を返す |

## リクエスト

#### 認証ヘッダ

本 API はセッション認証を必要とせず、検証用トークンをリクエストボディで受け取って検証する。

| ヘッダ | 要否 | 説明 |
|---|---|---|
| `Idempotency-Key: <ULID>` | 任意 | 冪等性(再送の重複防止) |

#### Query Parameters

```json
{}
```

#### Path Parameters

```json
{}
```

#### Request Body

```json
{
  "token": "reset_...",
  "password": "P@ssw0rd123!",
  "passwordConfirm": "P@ssw0rd123!"
}
```

| 項目              | 型     | 説明                                                                    |
|-------------------|--------|-------------------------------------------------------------------------|
| `token`           | string | 再設定トークン(必須・メールリンクから取得)                              |
| `password`        | string | 新しいパスワード(必須・12 文字以上、英大小文字 / 数字 / 記号 3 種類以上) |
| `passwordConfirm` | string | 新しいパスワード確認用(必須・`password` と一致)                        |

## レスポンス

**レスポンス(200)**

```json
{ "ok": true }
```

| 項目 | 型      | 説明                     |
|------|---------|--------------------------|
| `ok` | boolean | パスワード再設定確定の成否 |

## バリデーション

入力値の検証規則は `## リクエスト` の各パラメータ説明を参照する。

## エラー

| HTTP ステータス | エラーコード                          | 内容                   |
|-----------------|---------------------------------------|------------------------|
| 410 | `TOKEN_EXPIRED` → [ERR-006](../../05_errors/ERR-006.md#ERR-006) | トークン期限切れ |
| 410 | `TOKEN_USED` → [ERR-007](../../05_errors/ERR-007.md#ERR-007) | トークン使用済み |
| 404 | `TOKEN_NOT_FOUND` → [ERR-008](../../05_errors/ERR-008.md#ERR-008) | トークンが存在しない |
| 400 | `VALIDATION_ERROR` → [ERR-001](../../05_errors/ERR-001.md#ERR-001) | パスワード強度不足等 |
