# <span id="API-064"></span>API-064: 自己プロフィール取得

> **このページは 自己プロフィール取得 API の仕様を定義します。** 認証中の利用者自身の表示名・連絡先メールと確認状態・参加プロジェクト一覧(プロジェクトごとの立場を含む)を返す参照系 API。

## 項目

| 項目 | 内容 |
|---|---|
| API ID | `API-064` |
| API名 | 自己プロフィール取得 |
| トレーサビリティID | [TR-008](../../00_traceability/index.md#TR-008) |
| エンドポイント | `/me/profile` |
| HTTPメソッド | GET |
| 認証 | セッション(Cookie) |
| 認可 | 認証済み利用者(自分の情報) |

## 処理概要

認証中の利用者の表示名・連絡先メール(確認状態を含む)・参加プロジェクト一覧(プロジェクトごとの立場を含む)を返す。個人設定画面([SCR-022](../../01_frontend/01_screens/SCR-022.md#SCR-022))の初期表示で使用する。立場(オーナー / メンバー)はアカウント全体で固定されず、プロジェクトごとに定まる。

| 処理 ID | 処理内容 |
|---------|----------|
| P-01 | セッションから認証済み利用者を特定する(無効・失効は 401) |
| P-02 | 利用者の表示名・連絡先メールと確認状態を取得する |
| P-03 | 利用者が参加するプロジェクトの一覧(ID・名称・当該プロジェクトでの立場)を取得する。立場はプロジェクトの作成者(オーナー)か割当による参加者(メンバー)かで判定する |
| P-04 | 取得した情報をまとめて返す |

## リクエスト

#### 認証ヘッダ

本 API は利用者セッション(Cookie)による認証を必要とする参照系であり、CSRF トークンは不要である。

| ヘッダ | 要否 | 説明 |
|---|---|---|
| `Cookie: session=<token>` | 必須 | 利用者セッション |

## レスポンス

認証中の利用者のプロフィールと参加プロジェクト一覧を返す。

**レスポンス(200)**

```json
{
  "displayName": "山田 太郎",
  "email": "taro@example.com",
  "emailVerified": true,
  "projects": [
    { "id": "prj_01", "name": "サポートサイト", "role": "owner" },
    { "id": "prj_02", "name": "ヘルプセンター", "role": "member" }
  ]
}
```

| 項目 | 型 | 説明 |
|---|---|---|
| `displayName` | string | 表示名 |
| `email` | string | 連絡先メールアドレス |
| `emailVerified` | boolean | 連絡先メールの確認状態 |
| `projects` | array | 参加プロジェクト一覧 |
| `projects[].id` | string | プロジェクト ID |
| `projects[].name` | string | プロジェクト名 |
| `projects[].role` | enum | 当該プロジェクトでの立場(`owner` / `member`) |

## バリデーション

入力パラメータを持たない参照系のため、入力値の検証は行わない。

## エラー

| HTTP ステータス | エラーコード | 内容 |
|-----------------|---------------------|--------------------------------|
| 401 | `SESSION_EXPIRED` → [ERR-033](../../05_errors/ERR-033.md#ERR-033) | セッションが無効または失効している |

## 利用テーブル

本 API が参照・更新するテーブルと CRUD 区分。

| テーブル | CRUD |
|---|---|
| [`M_USER`](../04_database/TBL-001.md#TBL-001) | `-R--` |
| [`M_PRJ_USERS`](../04_database/TBL-003.md#TBL-003) | `-R--` |
| [`M_PROJECTS`](../04_database/TBL-004.md#TBL-004) | `-R--` |

## 備考

表示名・連絡先メールは全ユーザー共通の `M_USER` に保持する。参加プロジェクト一覧はプロジェクト割当(`M_PRJ_USERS`)とプロジェクト(`M_PROJECTS`)から構成し、各プロジェクトでの立場(オーナー / メンバー)はプロジェクトの作成者(`M_PROJECTS.owner_user_id` が当該利用者と一致すればオーナー、それ以外でプロジェクトに割り当てられていればメンバー)で判定する。立場はアカウント全体で固定されず、プロジェクトごとに定まる。更新は別系統の API([API-012](API-012.md#API-012) 自己プロフィール更新)が担い、本 API は参照のみを行う。
