# <span id="API-052"></span>API-052: 利用規約 最新版取得

> **このページは 利用規約 最新版取得 API の契約を定義します。** 利用規約の最新版を返す。

## 項目

| 項目 | 内容 |
|---|---|
| API ID | `API-052` |
| API名 | 利用規約 最新版取得 |
| トレーサビリティID | [TR-011](../../00_traceability/index.md#TR-011) ・ [TR-013](../../00_traceability/index.md#TR-013) |
| エンドポイント | `/terms/current` |
| HTTPメソッド | GET |
| 認証 | 不要 |
| 認可 | —(公開) |

## 処理概要

利用規約の最新版を返す。

| 処理 ID | 処理内容                                   |
|---------|--------------------------------------------|
| P-01    | 最新版の利用規約を取得する                 |
| P-02    | 本文・施行日・主な変更点を整形して返却する |

## リクエスト

#### 認証ヘッダ

本 API は公開エンドポイントのため、認証ヘッダを必要としない。

|---|---|---|
| —(なし) | — | 公開取得。認証・セッション不要 |

## レスポンス

**レスポンス(200)**

```json
{ "docType": "terms_of_service", "version": "...", "effectiveDate": "...", "bodyHtml": "...", "diffSummary": "..." }
```

| 項目 | 型 | 説明 |
|----|----|----|
| `docType` | string | 文書種別(利用規約は `terms_of_service` 固定) |
| `version` | string | 返却した利用規約のバージョン |
| `effectiveDate` | string (ISO 8601) | 当該バージョンの施行日 |
| `bodyHtml` | string | 利用規約の本文 HTML |
| `diffSummary` | string | 主な変更点(`M_TERMS_VER.diff_summary`。NULL の場合は省略) |

(最新版のみ。`diffSummary` は `M_TERMS_VER.diff_summary`。SCR-020 の「主な変更点」表示に用いる。NULL の場合は省略)
