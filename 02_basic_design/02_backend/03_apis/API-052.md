# <span id="API-052"></span>API-052: 利用規約 最新版取得

> **このページは 利用規約 最新版取得 API の仕様を定義します。** 利用規約の最新版を返す。

| ID | 業務ユースケースID | イベント(画面ID EVT-NN) | テーブルID |
|----|----|----|----|
| API-052 | [UC-011](../../../01_requirements/04_business_usecases/UC-011.md#UC-011) ・ [UC-013](../../../01_requirements/04_business_usecases/UC-013.md#UC-013) | SCR-015 EVT-01 ・ SCR-020 EVT-01 | [TBL-012](../04_database/TBL-012.md#TBL-012) ・ [TBL-024](../04_database/TBL-024.md#TBL-024) |

| API名 | エンドポイント | HTTPメソッド |
|----|----|----|
| 利用規約 最新版取得 | `/terms/current` | GET |

## 処理概要

利用規約の最新版を返す。

| 処理 ID | 処理内容                                   |
|---------|--------------------------------------------|
| P-01    | 最新版の利用規約を取得する                 |
| P-02    | 本文・施行日・主な変更点を整形して返却する |
| P-03    | 認証済みリクエスト時のみ、利用者の最新版同意状態(同意有無・同意日・再同意要否)を `T_TERMS_AGREE` × `M_TERMS_VER`(最新版・`doc_type = 'terms_of_service'`)から照合して返却する(未認証時は省略) |

## リクエスト

#### 認証ヘッダ

| ヘッダ | 要否 | 説明 |

## レスポンス

**レスポンス(200)**

```json
{ "docType": "terms_of_service", "version": "...", "effectiveDate": "...", "bodyHtml": "...", "diffSummary": "...", "consentStatus": { "agreed": true, "agreedAt": "...", "requiresReconsent": false } }
```

| 項目 | 型 | 説明 |
|----|----|----|
| `docType` | string | 文書種別(利用規約は `terms_of_service` 固定) |
| `version` | string | 返却した利用規約のバージョン |
| `effectiveDate` | string (ISO 8601) | 当該バージョンの施行日 |
| `bodyHtml` | string | 利用規約の本文 HTML |
| `diffSummary` | string | 主な変更点(`M_TERMS_VER.diff_summary`。NULL の場合は省略) |
| `consentStatus` | object | 認証済みリクエスト時のみ設定する利用者の同意状態。未認証(認証ヘッダ無し)時は本オブジェクトを省略する |
| `└ agreed` | boolean | 利用者が最新版に同意済みか |
| `└ agreedAt` | string (ISO 8601) \| null | 同意日(`T_TERMS_AGREE.agreed_at`)。未同意のときは null |
| `└ requiresReconsent` | boolean | 再同意要否(現行最新版に対する同意が無い場合 true) |

(最新版のみ。`diffSummary` は `M_TERMS_VER.diff_summary`。SCR-020 の「主な変更点」表示に用いる。NULL の場合は省略)

`consentStatus` は認証済みリクエスト時のみ返し、`T_TERMS_AGREE`(`user_id` × `doc_type = 'terms_of_service'` × `terms_version`)と `M_TERMS_VER`(最新 `version`)を照合して算出する(DB カラム追加は不要)。SCR-015 の同意状態バッジ表示に用いる。

## 利用テーブル

本 API が参照・更新するテーブルと CRUD 区分。

| テーブル | CRUD |
|---|---|
| [`T_TERMS_AGREE`](../04_database/TBL-024.md#TBL-024) | `-R--` |

`T_TERMS_AGREE` は認証済みリクエスト時の同意状態(`consentStatus`)照合に用いる(未認証時は参照しない)。DB カラム追加は不要。
