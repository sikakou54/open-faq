# <span id="API-033"></span>API-033: FAQ 個別取得

> **このページは FAQ 個別取得 API の契約を定義します。** 指定した FAQ ID の詳細情報を 1 件取得する。

## 項目

| 項目 | 内容 |
|---|---|
| API ID | `API-033` |
| API名 | FAQ 個別取得 |
| 対応業務UC | [UC-025](../../../01_requirements/04_business_usecases/UC-025.md#UC-025) |
| 対応画面ID | [SCR-009](../../01_frontend/01_screens/SCR-009.md#SCR-009) |
| 対応画面イベントID | EVT-076 |
| エンドポイント | `/faqs/{id}` |
| HTTPメソッド | GET |
| 認証 | Cookie |
| 認可 | オーナー / 当該プロジェクトのメンバー |

## 処理概要

指定した FAQ ID の詳細情報を 1 件取得する。

| 処理 ID | 処理内容                                                    |
|---------|-------------------------------------------------------------|
| P-01    | 認証・認可(オーナー / 該当プロジェクトの編集権限)を検証する |
| P-02    | 指定 FAQ が当該契約・プロジェクトに属するか検証する         |
| P-03    | FAQ の詳細情報を返す                                        |

## リクエスト

#### Query Parameters

```json
{}
```

#### Path Parameters

```json
{ "id": "FAQ ID(必須)" }
```

| 項目 | 型     | 説明                              |
|------|--------|-----------------------------------|
| `id` | string | 取得対象 FAQ の ID(必須・パス内) |

#### Request Body

```json
{}
```

## レスポンス

**レスポンス(200)**

```json
{ "id": "faq_...", "question": "...", "answer": "...", "category": "...", "status": "draft", "version": 1, "publishedAt": null }
```

| 項目          | 型                | 説明                                                         |
|---------------|-------------------|--------------------------------------------------------------|
| `id`          | string            | FAQ ID                                                       |
| `question`    | string            | 質問文                                                       |
| `answer`      | string            | 回答文                                                       |
| `category`    | string            | カテゴリ                                                     |
| `status`      | enum              | FAQ 状態(`draft` / `published` / `hidden`)                   |
| `version`     | integer           | 楽観ロック用バージョン番号                                   |
| `publishedAt` | string (ISO 8601) | 公開日時(`published` 化時に記録、未公開は `null`)            |

## バリデーション

入力値の検証規則は `## リクエスト` の各パラメータ説明を参照する。

## エラー

| HTTP ステータス | エラーコード            | 内容                                     |
|-----------------|-------------------------|------------------------------------------|
| 403 | `PROJECT_ACCESS_DENIED` → [ERR-021](../../05_errors/ERR-021.md#ERR-021) | 当該プロジェクトへの権限なし |
| 404 | `NOT_FOUND` → [ERR-019](../../05_errors/ERR-019.md#ERR-019) | 指定した FAQ が存在しない / 論理削除済み |
