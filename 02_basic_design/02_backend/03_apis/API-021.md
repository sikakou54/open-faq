# <span id="API-021"></span>API-021: メンバー招待

> **このページは メンバー招待 API の契約を定義します。** 該当プロジェクト 1 件に対する単発招待を行う。

## 項目

| 項目 | 内容 |
|---|---|
| API ID | `API-021` |
| API名 | メンバー招待 |
| 対応業務UC | [UC-019](../../../01_requirements/04_business_usecases/UC-019.md#UC-019) |
| 対応画面ID | [SCR-014](../../01_frontend/01_screens/SCR-014.md#SCR-014) |
| 対応画面イベントID | [EVT-126](../../01_frontend/02_screen_events/EVT-126.md#EVT-126) |
| エンドポイント | `/projects/{id}/members` |
| HTTPメソッド | POST |
| 認証 | Cookie + CSRF + 再認証必須 |
| 認可 | オーナー / 当該プロジェクトのメンバー |

## 処理概要

該当プロジェクト 1 件に対する単発招待を行う。

| 処理 ID | 処理内容 |
|----|----|
| P-01 | 認証・認可(オーナー / 当該プロジェクトのメンバー)と再認証を検証する |
| P-02 | 入力値(メール形式)を検証し、当該プロジェクトへの既存割当(重複)を判定する |
| P-03 | 既存アカウントは当該プロジェクトの割当を追加し、新規メールは予約 `M_USER`(`name`/`password_hash`=NULL, `status='pending_activation'`)と無効状態の割当(`M_PRJ_USERS.valid=0`)を作成する |
| P-04 | 新規メールには有効化トークンを発行し招待メールを送信する |
| P-05 | 招待結果を返す |

## リクエスト

#### Query Parameters

```json
{}
```

#### Path Parameters

```json
{ "id": "プロジェクト ID" }
```

| 項目 | 型     | 説明                                        |
|------|--------|---------------------------------------------|
| `id` | string | 招待先プロジェクトの ID(必須・パス内で指定) |

#### Request Body

```json
{
  "email": "newmember@..."
}
```

| 項目    | 型     | 説明                       |
|---------|--------|----------------------------|
| `email` | string | 招待先メールアドレス(必須) |

## レスポンス

**レスポンス(201)**

```json
{ "userId": "01HZ...", "status": "active|pending_activation", "expiresAt": "..." | null }
```

| 項目 | 型 | 説明 |
|----|----|----|
| `userId` | string | 招待した(または既存の)メンバーの ID |
| `status` | enum | アカウント状態(`active` / `pending_activation`) |
| `expiresAt` | string (ISO 8601) | 新規招待時のトークン有効期限(既存アカウントは `null`) |

## バリデーション

入力値の検証規則は `## リクエスト` の各パラメータ説明を参照する。

## エラー

| HTTP ステータス | エラーコード                | 内容                           |
|-----------------|-----------------------------|--------------------------------|
| 409 | `ALREADY_EXISTS_IN_PROJECT` → [ERR-020](../../05_errors/ERR-020.md#ERR-020) | 既に該当プロジェクトに割当あり |
| 400 | `VALIDATION_ERROR` → [ERR-001](../../05_errors/ERR-001.md#ERR-001) | 入力値の検証エラー |
| 403 | `PROJECT_ACCESS_DENIED` → [ERR-021](../../05_errors/ERR-021.md#ERR-021) | 当該プロジェクトへの権限なし |
