# <span id="API-042"></span>API-042: 利用量サマリ(オーナー)

> **このページは 利用量サマリ(オーナー) API の仕様を定義します。** 自分が作成したプロジェクト全体のサマリーとプロジェクト別利用量を返す。

| ID | 業務ユースケースID | イベント(画面ID EVT-NN) | テーブルID |
|----|----|----|----|
| API-042 | [UC-035](../../../01_requirements/04_business_usecases/UC-035.md#UC-035) | SCR-021 EVT-01 | [TBL-004](../04_database/TBL-004.md#TBL-004) ・ [TBL-005](../04_database/TBL-005.md#TBL-005) ・ [TBL-006](../04_database/TBL-006.md#TBL-006) ・ [TBL-009](../04_database/TBL-009.md#TBL-009) ・ [TBL-017](../04_database/TBL-017.md#TBL-017) ・ [TBL-018](../04_database/TBL-018.md#TBL-018) ・ [TBL-020](../04_database/TBL-020.md#TBL-020) ・ [TBL-025](../04_database/TBL-025.md#TBL-025) |

| API名 | エンドポイント | HTTPメソッド |
|----|----|----|
| 利用量サマリ(オーナー) | `/owner/projects/usage` | GET |

## 処理概要

自分が作成したプロジェクト全体のサマリーとプロジェクト別利用量を返す。

| 処理 ID | 処理内容                                                  |
|---------|-----------------------------------------------------------|
| P-01    | 認証・認可(オーナー専有)を検証する                        |
| P-02    | 自分が作成した(オーナーである)全プロジェクトの質問数・公開 FAQ 件数・解決率・当月 AI 推論コスト・前月実績・制限中プロジェクト数を集計する |
| P-03    | オーナー単位のサマリーとプロジェクト別内訳を整形して返却する    |

## リクエスト

#### 認証ヘッダ

本 API は利用者セッション(Cookie)による認証を必要とする。

| ヘッダ | 要否 | 説明 |
|---|---|---|
| `Cookie: session=<token>` | 必須 | 利用者セッション |

#### Query Parameters

```json
{ "period": "current_month" }
```

| 項目     | 型   | 説明                               |
|----------|------|------------------------------------|
| `period` | enum | 集計期間。任意。既定は `current_month`(当月固定スナップショット)。本サマリーは当月固定で集計し、画面に集計期間セレクトは設けない |

#### Path Parameters

```json
{}
```

#### Request Body

```json
{}
```

## レスポンス

**レスポンス(200)**

```json
{
  "period": { "from": "2026-06-01T00:00:00Z", "to": "2026-06-30T23:59:59Z" },
  "summary": {
    "projectCount": 3,
    "questionCount": 2450,
    "publishedFaqCount": 184,
    "resolutionRate": 82.3,
    "aiCostYen": 38420,
    "limitedProjectCount": 1,
    "previousMonth": { "questionCount": 2264, "resolutionRate": 81.1, "aiCostYen": 35110 }
  },
  "projects": [
    {
      "projectId": "prj_...",
      "projectName": "サポートサイト",
      "questions": { "used": 750, "limitEnabled": true, "limit": 1000, "percentage": 75 },
      "publishedFaqCount": 85
    }
  ],
  "updatedAt": "2026-06-14T05:00:00Z"
}
```

| 項目                    | 型                | 説明                         |
|-------------------------|-------------------|------------------------------|
| `period`                | object            | 集計対象期間                 |
| `　└ from`              | string (ISO 8601) | 期間の起点                   |
| `　└ to`                | string (ISO 8601) | 期間の終点                   |
| `summary`               | object            | 自分が作成したプロジェクト全体のサマリー |
| `　└ projectCount`      | integer           | 自分が作成したプロジェクト数 |
| `　└ questionCount`     | integer           | 自分が作成したプロジェクト全体の質問数 |
| `　└ publishedFaqCount` | integer           | 自分が作成したプロジェクト全体の公開 FAQ 件数 |
| `　└ resolutionRate`    | number            | 自分が作成したプロジェクト全体の解決率(%)。当月の解決済み質問数 ÷ 当月質問数 × 100。**解決済み質問数 = AI が回答を生成し、回答フィードバックでも未解決化されなかった質問数**(質問ログの結果が回答ありのもの。[`H_QUESTION_LOGS.result_reason_code`](../04_database/TBL-025.md#TBL-025) = `answered`)。未解決質問が生成された質問(`no_faq_match` / `low_confidence` / `pii_detected` / `contradiction` / `user_unresolved`)および処理エラー・受付不可ログ(`ai_unavailable` / `rate_limited` / `usage_limit_reached` / `payment_required`)は解決済みに含めない。分母は当月の全質問数(回答可否を問わない)。質問が 0 件のときは 0 とする |
| `　└ aiCostYen`         | integer           | 当月の自分が作成したプロジェクト全体 AI 推論コスト(円) |
| `　└ limitedProjectCount` | integer         | 制限中プロジェクト数。当月質問数が月次上限に到達(利用率 100% 以上)または課金停止中のプロジェクト件数 |
| `　└ previousMonth`     | object            | 前月実績(前月比は本値と当月値から画面側で算出) |
| `　　└ questionCount`   | integer           | 前月の自分が作成したプロジェクト全体の質問数 |
| `　　└ resolutionRate`  | number            | 前月の解決率(%)             |
| `　　└ aiCostYen`       | integer           | 前月の自分が作成したプロジェクト全体 AI 推論コスト(円) |
| `projects`              | array\<object\>   | プロジェクト別の利用量内訳   |
| `　└ projectId`         | string            | プロジェクトの ID            |
| `　└ projectName`       | string            | プロジェクト名               |
| `　└ questions`         | object            | プロジェクトの質問数利用状況 |
| `　　└ used`            | integer           | 当月の質問数                 |
| `　　└ limitEnabled`    | boolean           | 月次上限が有効か             |
| `　　└ limit`           | integer           | 月次上限件数                 |
| `　　└ percentage`      | integer           | 上限に対する利用率(%)        |
| `　└ publishedFaqCount` | integer           | プロジェクトの公開 FAQ 件数  |
| `updatedAt`             | string (ISO 8601) | 集計値の最終更新日時         |
