# <span id="API-054"></span>API-054: 利用規約 同意

> **このページは 利用規約 同意 API の仕様を定義します。** 利用規約への同意を記録する(冪等)。

| ID | 業務ユースケースID | イベント(画面ID EVT-NN) | テーブルID |
|----|----|----|----|
| API-054 | [UC-013](../../../01_requirements/04_business_usecases/UC-013.md#UC-013) | SCR-015 EVT-03 ・ SCR-020 EVT-06 | [TBL-012](../04_database/TBL-012.md#TBL-012) ・ [TBL-024](../04_database/TBL-024.md#TBL-024) |

| API名 | エンドポイント | HTTPメソッド |
|----|----|----|
| 利用規約 同意 | `/terms/agree` | POST |

## 処理概要

利用規約への同意を記録する(冪等)。

| 処理 ID | 処理内容                                                       |
|---------|----------------------------------------------------------------|
| P-01    | 認証・認可(オーナー / メンバー、CSRF)を検証する                |
| P-02    | 指定バージョンへの利用規約同意を記録する(既存同意があれば冪等) |
| P-03    | 同意記録結果を返却する                                         |

## リクエスト

#### 認証ヘッダ

本 API は利用者セッション(Cookie)と CSRF トークンを必要とする状態変更系である。

| ヘッダ | 要否 | 説明 |
|---|---|---|
| `Cookie: session=<token>` | 必須 | 利用者セッション |
| `X-CSRF-Token: <token>` | 必須 | CSRF 対策(状態変更系) |
| `Idempotency-Key: <ULID>` | 任意 | 冪等性(書き込み再送の重複防止) |

#### Query Parameters

```json
{}
```

#### Path Parameters

```json
{}
```

#### Request Body

```json
{ "version": "..." }
```

| 項目      | 型     | 説明                               |
|-----------|--------|------------------------------------|
| `version` | string | 同意対象の利用規約バージョン(必須) |

`T_TERMS_AGREE` に `(user_id, doc_type='terms_of_service', terms_version)` で 1 行記録(冪等: 既存同意があれば 200)。

## レスポンス

**レスポンス(200)**

同意記録結果。
