# <span id="API-054"></span>API-054: 利用規約 同意

> **このページは 利用規約 同意 API の契約を定義します。** 利用規約への同意を記録する(冪等)。

## 項目

| 項目 | 内容 |
|---|---|
| API ID | `API-054` |
| API名 | 利用規約 同意 |
| 対応業務UC | [UC-013](../../../01_requirements/04_business_usecases/UC-013.md#UC-013) |
| 対応画面ID | [SCR-015](../../01_frontend/01_screens/SCR-015.md#SCR-015) ・ [SCR-020](../../01_frontend/01_screens/SCR-020.md#SCR-020) |
| 対応画面イベントID | EVT-135 ・ EVT-169 |
| エンドポイント | `/terms/agree` |
| HTTPメソッド | POST |
| 認証 | Cookie + CSRF |
| 認可 | オーナー / メンバー |

## 処理概要

利用規約への同意を記録する(冪等)。

| 処理 ID | 処理内容                                                       |
|---------|----------------------------------------------------------------|
| P-01    | 認証・認可(オーナー / メンバー、CSRF)を検証する                |
| P-02    | 指定バージョンへの利用規約同意を記録する(既存同意があれば冪等) |
| P-03    | 同意記録結果を返却する                                         |

## リクエスト

#### 認証ヘッダ

本 API は利用者セッション(Cookie)と CSRF トークンを必要とする状態変更系である。

| ヘッダ | 要否 | 説明 |
|---|---|---|
| `Cookie: session=<token>` | 必須 | 利用者セッション |
| `X-CSRF-Token: <token>` | 必須 | CSRF 対策(状態変更系) |
| `Idempotency-Key: <ULID>` | 任意 | 冪等性(書き込み再送の重複防止) |

#### Query Parameters

```json
{}
```

#### Path Parameters

```json
{}
```

#### Request Body

```json
{ "version": "..." }
```

| 項目      | 型     | 説明                               |
|-----------|--------|------------------------------------|
| `version` | string | 同意対象の利用規約バージョン(必須) |

`T_TERMS_AGREE` に `(user_id, doc_type='terms_of_service', terms_version)` で 1 行記録(冪等: 既存同意があれば 200)。

## レスポンス

**レスポンス(200)**

同意記録結果。
