# <span id="API-024"></span>API-024: 招待メール再送

> **このページは 招待メール再送 API の契約を定義します。** `status='pending_activation'` のメンバーへ招待メールを再送する。

## 項目

| 項目 | 内容 |
|---|---|
| API ID | `API-024` |
| API名 | 招待メール再送 |
| トレーサビリティID | [TR-019](../../00_traceability/index.md#TR-019) |
| エンドポイント | `/members/{id}/resend-invitation` |
| HTTPメソッド | POST |
| 認証 | Cookie + CSRF + 再認証必須 |
| 認可 | オーナー / 当該プロジェクトのメンバー |

## 処理概要

`status='pending_activation'` のメンバーへ招待メールを再送する。

| 処理 ID | 処理内容 |
|----|----|
| P-00 | 再認証トークン(リクエストボディ `reauthToken`)を検証する |
| P-01 | 認証・認可(オーナー / 当該プロジェクトのメンバー)と再認証を検証する |
| P-02 | 対象が未有効化(`pending_activation`)のメンバーであることを確認する(オーナー / 自身は対象外) |
| P-03 | 旧有効化トークンを失効させ、同一の招待先情報で新規トークンを再発行する |
| P-04 | 招待メールを再送し再発行結果を返す |

## リクエスト

#### 認証ヘッダ

本 API は利用者セッション(Cookie)と CSRF トークンを必要とする状態変更系である。

| ヘッダ | 要否 | 説明 |
|---|---|---|
| `Cookie: session=<token>` | 必須 | 利用者セッション |
| `X-CSRF-Token: <token>` | 必須 | CSRF 対策(状態変更系) |
| `Idempotency-Key: <ULID>` | 任意 | 冪等性(書き込み再送の重複防止) |

> 本操作は再認証必須。直近の再認証([API-005](API-005.md#API-005))で得た再認証状態を伴う必要がある。

#### Query Parameters

```json
{}
```

#### Path Parameters

```json
{ "id": "メンバーユーザー ID" }
```

| 項目 | 型 | 説明 |
|----|----|----|
| `id` | string | 招待メールを再送する対象メンバーの利用者 ID(必須・パス内で指定) |

#### Request Body

```json
{ "reauthToken": "reauth_..." }
```

| 項目 | 型 | 説明 |
|----|----|----|
| `reauthToken` | string | 再認証トークン(必須・API-005 で取得) |

氏名 `displayName` を受け付けない(FR-020 / FR-022)。

## レスポンス

**レスポンス(200)**

再発行結果(`{ "ok": true, "expiresAt": "..." }`)。

| 項目        | 型                | 説明                             |
|-------------|-------------------|----------------------------------|
| `ok`        | boolean           | 招待メール再送の成否             |
| `expiresAt` | string (ISO 8601) | 再発行した招待トークンの有効期限 |

## バリデーション

入力値の検証規則は `## リクエスト` の各パラメータ説明を参照する。

## エラー

| HTTP ステータス | エラーコード              | 内容             |
|-----------------|---------------------------|------------------|
| 401 | `REAUTH_REQUIRED` → [ERR-013](../../05_errors/ERR-013.md#ERR-013) | 再認証トークンが無効または未提示 |
| 403 | `OWNER_PROTECTED` → [ERR-021](../../05_errors/ERR-021.md#ERR-021) | オーナーは対象外 |
| 403 | `SELF_MUTATION_FORBIDDEN` → [ERR-022](../../05_errors/ERR-022.md#ERR-022) | 自分自身は対象外 |
| 404 | `NOT_FOUND` → [ERR-017](../../05_errors/ERR-017.md#ERR-017) | 対象が存在しない |
