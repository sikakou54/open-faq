# <span id="API-036"></span>API-036: 未解決質問 CSV エクスポート

> **このページは 未解決質問 CSV エクスポート API の契約を定義します。** 現在のフィルタ条件(状況・期間・フリーワード)に一致する未解決質問を全件 CSV 形式で生成して返却する。

*版数 v2.0 ・ 更新 2026-06-21*

## 項目

| 項目 | 内容 |
|---|---|
| API ID | `API-036` |
| API名 | 未解決質問 CSV エクスポート |
| 対応業務UC | [UC-030](../../../01_requirements/04_business_usecases/UC-030.md#UC-030) |
| 対応画面ID | [SCR-006](../../01_frontend/01_screens/SCR-006.md#SCR-006) |
| 対応画面イベントID | [EVT-049](../../01_frontend/02_screen_events/EVT-049.md#EVT-049) |

| HTTPメソッド | GET |
| 認証 | Cookie |
| 認可 | オーナー / 当該プロジェクトのメンバー |

## 処理概要

現在のフィルタ条件(状況・期間・フリーワード)に一致する未解決質問を全件 CSV 形式で生成して返却する。

| 処理 ID | 処理内容                                                        |
|---------|-----------------------------------------------------------------|
| P-01    | 認証・認可(対象プロジェクトへのアクセス権)を検証する           |
| P-02    | 状況・期間・フリーワード・プロジェクトの条件で未解決質問を全件取得する |
| P-03    | CSV 形式に整形して `Content-Disposition: attachment` で返却する |

## リクエスト

#### Query Parameters

```json
{
  "status": "(任意・open / closed のいずれか、未指定で両方)",
  "projectId": "(任意)",
  "dateFrom": "(任意・ISO 8601 日付文字列)",
  "dateTo": "(任意・ISO 8601 日付文字列)",
  "q": "(任意・フリーワード検索文字列)"
}
```

| 項目       | 型     | 説明                                                    |
|------------|--------|---------------------------------------------------------|
| `status`   | enum   | 状況での絞り込み。任意。`open` / `closed`(未指定で両方) |
| `projectId`| string | プロジェクトでの絞り込み。任意                          |
| `dateFrom` | string | 更新日時の開始日。任意(ISO 8601)                        |
| `dateTo`   | string | 更新日時の終了日。任意(ISO 8601)                        |
| `q`        | string | フリーワード検索文字列。任意                            |

#### Path Parameters

```json
{}
```

```json
{}
```

## レスポンス

**レスポンス(200)**

`Content-Type: text/csv; charset=utf-8` / `Content-Disposition: attachment; filename="inquiries.csv"`。列構成は `問い合わせID,状況,質問,未解決理由,最終更新日時`。
