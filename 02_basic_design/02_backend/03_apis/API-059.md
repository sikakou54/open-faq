# <span id="API-059"></span>API-059: 外部 Webhook(Resend)

> **このページは 外部 Webhook(Resend) API の契約を定義します。** メール配信状態 Webhook を受信し配信状態 / 抑制リストを更新する。

## 項目

| 項目 | 内容 |
|---|---|
| API ID | `API-059` |
| API名 | 外部 Webhook(Resend) |
| トレーサビリティID | [TR-062](../../00_traceability/index.md#TR-062) ・ [TR-069](../../00_traceability/index.md#TR-069) |
| エンドポイント | `/webhooks/resend` |
| HTTPメソッド | POST |
| 認証 | Resend 署名検証(HMAC-SHA256) |
| 認可 | —(署名検証のみ) |

## 処理概要

メール配信状態 Webhook を受信し配信状態 / 抑制リストを更新する。

| 処理 ID | 処理内容 |
|----|----|
| P-01 | Resend 署名(HMAC-SHA256)を検証する(失敗時は 401) |
| P-02 | イベント種別を判定する |
| P-03 | `H_NOTIF_LOGS.delivery_state` を遷移する。通知のイベント発生時刻が記録済みの状態より古い場合は上書きしない(順序逆転・遅延到達への整合性保証) |
| P-04 | `bounced` / `complained` の場合は `M_EMAIL_SUPPRESS` に追加する(全契約横断) |
| P-05 | 受領応答を返す |

## リクエスト

#### 認証ヘッダ

本 API は Resend が付与する署名(HMAC-SHA256)を検証して受信元を認証する(認可はなし)。

| ヘッダ | 要否 | 説明 |
|---|---|---|
| `Svix-Signature: <署名>` | 必須 | Resend(Svix)署名。HMAC-SHA256 で検証(失敗時 401 [ERR-031](../../05_errors/ERR-031.md#ERR-031)) |

#### Query Parameters

```json
{}
```

#### Path Parameters

```json
{}
```

#### Request Body

(Resend 仕様準拠):

```json
{
  "type": "email.delivered|email.bounced|email.complained|email.delivery_delayed",
  "data": {
    "message_id": "...",
    "to": "...",
    "subject": "...",
    "timestamp": "..."
  }
}
```

| 項目 | 型 | 説明 |
|----|----|----|
| `type` | enum | 配信イベント種別(`email.delivered` / `email.bounced` / `email.complained` / `email.delivery_delayed`) |
| `data` | object | イベント詳細データ |
| 　└ `message_id` | string | 対象メールのメッセージ ID |
| 　└ `to` | string | 宛先メールアドレス |
| 　└ `subject` | string | メール件名 |
| 　└ `timestamp` | string (ISO 8601) | イベント発生時刻 |
