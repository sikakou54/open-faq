# <span id="API-015"></span>API-015: 契約設定更新

> **このページは 契約設定更新 API の契約を定義します。** 契約レベルの設定情報(請求連絡先メール・タイムゾーン)を更新する。契約名は読み取り専用であり本 API では変更できない。

*版数 v2.0 ・ 更新 2026-06-21*

## 項目

| 項目 | 内容 |
|---|---|
| API ID | `API-015` |
| API名 | 契約設定更新 |
| 対応業務UC | [UC-022](../../../01_requirements/04_business_usecases/UC-022.md#UC-022) |
| 対応画面ID | [SCR-029](../../01_frontend/01_screens/SCR-029.md#SCR-029) |
| 対応画面イベントID | [EVT-217](../../01_frontend/02_screen_events/EVT-217.md#EVT-217) |
| エンドポイント | `/owner/settings` |
| HTTPメソッド | PATCH |
| 認証 | 要(Cookie + CSRF) |
| 認可 | オーナー専有 |

## 処理概要

契約レベルの設定情報(請求連絡先メール・タイムゾーン)を更新する。契約名は読み取り専用であり本 API では変更できない。

| 処理 ID | 処理内容                                                       |
|---------|----------------------------------------------------------------|
| P-01    | 認証・認可(オーナー専有)を検証する                             |
| P-02    | 入力値(メールアドレス形式・タイムゾーン識別子)を検証する       |
| P-03    | `M_CONTRACT` の連絡先メール・タイムゾーンを更新する            |
| P-04    | 更新後の値を返却する                                           |

## リクエスト

#### Query Parameters

```json
{}
```

#### Path Parameters

```json
{}
```

#### Request Body

```json
{
  "contactEmail": "billing@example.com",
  "timezone": "Asia/Tokyo"
}
```

| 項目           | 型     | 説明                                                           |
|----------------|--------|----------------------------------------------------------------|
| `contactEmail` | string | 新しい連絡先メールアドレス(任意・メール形式)                   |
| `timezone`     | string | 新しいタイムゾーン(任意・IANA タイムゾーン識別子)              |

## レスポンス

**レスポンス(200)**

```json
{
  "contractName": "株式会社サンプル",
  "contactEmail": "billing@example.com",
  "timezone": "Asia/Tokyo"
}
```

| 項目           | 型     | 説明                                                 |
|----------------|--------|------------------------------------------------------|
| `contractName` | string | 契約名(更新不可・現在値を返す)                       |
| `contactEmail` | string | 更新後の連絡先メールアドレス                         |
| `timezone`     | string | 更新後のタイムゾーン(IANA タイムゾーン識別子)        |

## バリデーション

入力値の検証規則は `## リクエスト` の各パラメータ説明を参照する。

| 403 | `E-AUTHZ-OWNER-ONLY` → [ERR-017](../../05_errors/ERR-017.md#ERR-017) | オーナー以外は不可 |
