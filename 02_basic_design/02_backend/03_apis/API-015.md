# <span id="API-015"></span>API-015: 契約設定更新

> **このページは 契約設定更新 API の契約を定義します。** 契約レベルの組織名を更新する。連絡先メールはオーナーの `M_USER.email` を参照する読み取り専用であり、本 API では変更しない(連絡先メールの変更はプロフィール更新 API-012 で行う)。

## 項目

| 項目 | 内容 |
|---|---|
| API ID | `API-015` |
| API名 | 契約設定更新 |
| 対応業務UC | [UC-022](../../../01_requirements/04_business_usecases/UC-022.md#UC-022) |
| 対応画面ID | [SCR-029](../../01_frontend/01_screens/SCR-029.md#SCR-029) |
| 対応画面イベントID | EVT-216 |
| エンドポイント | `/owner/settings` |
| HTTPメソッド | PATCH |
| 認証 | 要(Cookie + CSRF) |
| 認可 | オーナー専有 |

## 処理概要

契約レベルの組織名を更新する。連絡先メールはオーナーの `M_USER.email` を参照する読み取り専用であり、本 API では変更しない(連絡先メールの変更はプロフィール更新 API-012 で行う)。

| 処理 ID | 処理内容                                                       |
|---------|----------------------------------------------------------------|
| P-01    | 認証・認可(オーナー専有)を検証する                             |
| P-02    | 入力値(組織名の文字数等)を検証する                            |
| P-03    | `M_CONTRACT.organization`(組織名)を更新する                  |
| P-04    | 更新後の値を返却する                                           |

## リクエスト

#### 認証ヘッダ

本 API は利用者セッション(Cookie)と CSRF トークンを必要とする状態変更系である。

| ヘッダ | 要否 | 説明 |
|---|---|---|
| `Cookie: session=<token>` | 必須 | 利用者セッション |
| `X-CSRF-Token: <token>` | 必須 | CSRF 対策(状態変更系) |
| `Idempotency-Key: <ULID>` | 任意 | 冪等性(書き込み再送の重複防止) |

#### Query Parameters

```json
{}
```

#### Path Parameters

```json
{}
```

#### Request Body

```json
{
  "organizationName": "株式会社サンプル"
}
```

| 項目               | 型     | 説明                                                       |
|--------------------|--------|------------------------------------------------------------|
| `organizationName` | string | 新しい組織名(必須・1〜100 文字)。`M_CONTRACT.organization` に保存する |

## レスポンス

**レスポンス(200)**

```json
{
  "organizationName": "株式会社サンプル",
  "contactEmail": "owner@example.com"
}
```

| 項目               | 型     | 説明                                                 |
|--------------------|--------|------------------------------------------------------|
| `organizationName` | string | 更新後の組織名(`M_CONTRACT.organization`)           |
| `contactEmail`     | string | 連絡先メール(オーナーの `M_USER.email`・読み取り専用) |
