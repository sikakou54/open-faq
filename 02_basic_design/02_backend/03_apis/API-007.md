# <span id="API-007"></span>API-007: 招待トークン検証・プレビュー

> **このページは 招待トークン検証・プレビュー API の仕様を定義します。** 登録済みユーザーが受け取った招待トークンを検証し、参加先プロジェクトの情報(プロジェクト名 / 招待元オーナー名 / 招待者メール)をプレビュー取得する。アカウント新規作成用のフィールドは扱わない(未登録メールの招待は不可。新規アカウント作成は独立サインアップ [API-001](API-001.md#API-001) で行う)。

## 処理概要

登録済みユーザー宛の招待トークンを検証し、参加先プロジェクトの情報(プロジェクト名 / 招待元オーナー名 / 招待者メール)をプレビュー取得する。アカウント作成は行わない。

| 処理 ID | 処理内容                                                         |
|---------|------------------------------------------------------------------|
| P-01    | 招待トークンの存在・有効性(期限・使用済み)を検証する             |
| P-02    | トークンから招待先プロジェクト・招待元オーナー・招待先の登録済みユーザーを取得する |
| P-03    | 状態変更を行わず参加先プロジェクトのプレビューを返す             |

## リクエスト

#### 認証ヘッダ

本 API はセッション認証を必要とせず、検証用トークンを URL パスで受け取って検証する。

| ヘッダ | 要否 | 説明 |
|---|---|---|
| `Idempotency-Key: <ULID>` | 任意 | 冪等性(再送の重複防止) |

#### Query Parameters

```json
{}
```

#### Path Parameters

```json
{ "token": "招待トークン" }
```

| 項目    | 型     | 説明                             |
|---------|--------|----------------------------------|
| `token` | string | 招待トークン(必須・パス内で指定) |

#### Request Body

```json
{}
```

## レスポンス

**レスポンス(200)**

```json
{
  "userId": "01HZ...",
  "email": "newmember@...",
  "projectId": "proj_1",
  "projectName": "顧客サポート",
  "inviterOwnerName": "山田 一郎",
  "expiresAt": "2026-05-28T..."
}
```

| 項目               | 型                | 説明                                 |
|--------------------|-------------------|--------------------------------------|
| `userId`           | string            | 招待された利用者の ID                |
| `email`            | string            | 招待先メールアドレス                 |
| `projectId`        | string            | 招待先プロジェクトの ID              |
| `projectName`      | string            | 招待先プロジェクトの名称             |
| `inviterOwnerName` | string            | 招待元オーナーの表示名                 |
| `expiresAt`        | string (ISO 8601) | 招待トークンの有効期限               |

## バリデーション

入力値の検証規則は `## リクエスト` の各パラメータ説明を参照する。
