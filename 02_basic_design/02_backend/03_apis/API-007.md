# <span id="API-007"></span>API-007: 招待トークン検証・プレビュー

> **このページは 招待トークン検証・プレビュー API の契約を定義します。** 招待情報(プロジェクト名 / 招待元オーナー名 / 招待者メール)をプレビュー取得する。

*版数 v2.0 ・ 更新 2026-06-21*

## 項目

| 項目 | 内容 |
|---|---|
| API ID | `API-007` |
| API名 | 招待トークン検証・プレビュー |
| 対応業務UC | [UC-006](../../../01_requirements/04_business_usecases/UC-006.md#UC-006) |
| 対応画面ID | [SCR-023](../../01_frontend/01_screens/SCR-023.md#SCR-023) |
| 対応画面イベントID | [EVT-181](../../01_frontend/02_screen_events/EVT-181.md#EVT-181) |
| エンドポイント | `/auth/invitations/{token}/preview` |
| HTTPメソッド | POST |
| 認証 | 未認証可(トークン認証のみ) |
| 認可 | — |

## 処理概要

招待情報(プロジェクト名 / 招待元オーナー名 / 招待者メール)をプレビュー取得する。

| 処理 ID | 処理内容                                                         |
|---------|------------------------------------------------------------------|
| P-01    | 招待トークンの存在・有効性(期限・使用済み)を検証する             |
| P-02    | トークンから招待先プロジェクト・招待元情報を取得する |
| P-03    | 状態変更を行わず招待情報のプレビューを返す                       |

## リクエスト

#### Query Parameters

```json
{}
```

#### Path Parameters

```json
{ "token": "招待トークン" }
```

| 項目    | 型     | 説明                             |
|---------|--------|----------------------------------|
| `token` | string | 招待トークン(必須・パス内で指定) |

#### Request Body

```json
{}
```

## レスポンス

**レスポンス(200)**

```json
{
  "userId": "01HZ...",
  "email": "newmember@...",
  "projectId": "proj_1",
  "projectName": "顧客サポート",
  "inviterOwnerName": "山田 一郎",
  "expiresAt": "2026-05-28T..."
}
```

| 項目               | 型                | 説明                                 |
|--------------------|-------------------|--------------------------------------|
| `userId`           | string            | 招待された利用者の ID                |
| `email`            | string            | 招待先メールアドレス                 |
| `projectId`        | string            | 招待先プロジェクトの ID              |
| `projectName`      | string            | 招待先プロジェクトの名称             |
| `inviterOwnerName` | string            | 招待元オーナーの氏名                 |
| `expiresAt`        | string (ISO 8601) | 招待トークンの有効期限               |

## バリデーション

入力値の検証規則は `## リクエスト` の各パラメータ説明を参照する。
