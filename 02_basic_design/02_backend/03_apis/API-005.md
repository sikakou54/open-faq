## 処理概要

機微操作の前にパスワードを再確認する。

| 処理 ID | 処理内容                                   |
|---------|--------------------------------------------|
| P-01    | セッションから認証済み利用者を特定する     |
| P-02    | パスワードを再照合する(不一致は 401)       |
| P-03    | 機微操作に使う再認証トークンを発行して返す |

## リクエスト

#### Query Parameters

```json
{}
```

#### Path Parameters

```json
{}
```

#### Request Body

```json
{ "password": "..." }
```

| 項目       | 型     | 説明                       |
|------------|--------|----------------------------|
| `password` | string | 再確認するパスワード(必須) |

## レスポンス

**レスポンス(200)**

```json
{ "ok": true, "token": "reauth_..." }
```

| 項目    | 型      | 説明                         |
|---------|---------|------------------------------|
| `ok`    | boolean | 再認証の成否                 |
| `token` | string  | 機微操作に使う再認証トークン |

## バリデーション

入力値の検証規則は `## リクエスト` の各パラメータ説明を参照する。
