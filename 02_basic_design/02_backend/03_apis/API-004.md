## 処理概要

再設定メールの送信を要求する。

| 処理 ID | 処理内容 |
|----|----|
| P-01 | 該当アカウントが存在する場合のみ再設定トークンを発行し再設定メールを送信する |
| P-02 | アカウントの存在有無を漏らさず一律の受付結果を返す |

## リクエスト

#### 認証ヘッダ

本 API は公開エンドポイントで、認証セッションを必要としない(資格情報・入力はリクエストボディで送出)。

| ヘッダ | 要否 | 説明 |
|---|---|---|
| `Idempotency-Key: <ULID>` | 任意 | 冪等性(再送の重複防止) |

#### Query Parameters

```json
{}
```

#### Path Parameters

```json
{}
```

#### Request Body

```json
{ "email": "admin@example.com" }
```

| 項目             | 型     | 説明                                     |
|------------------|--------|------------------------------------------|
| `email`          | string | 再設定メールの送信先メールアドレス(必須) |

## レスポンス

**レスポンス(202)**

```json
{ "ok": true }
```

| 項目 | 型 | 説明 |
|----|----|----|
| `ok` | boolean | 再設定要求の受付成否(アカウントの存在有無に関わらず一律 `true`) |

> [!NOTE]
> **補足** 列挙攻撃対策として、アカウントの存在有無は応答に含めない。
