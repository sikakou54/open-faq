# <span id="API-056"></span>API-056: 退会申請

> **このページは 退会申請 API の契約を定義します。** オーナーの退会を申請する。

*版数 v2.0 ・ 更新 2026-06-21*

## 項目

| 項目 | 内容 |
|---|---|
| API ID | `API-056` |
| API名 | 退会申請 |
| 対応業務UC | [UC-023](../../../01_requirements/04_business_usecases/UC-023.md#UC-023) ・ [UC-087](../../../01_requirements/04_business_usecases/UC-087.md#UC-087) |
| 対応画面ID | [SCR-019](../../01_frontend/01_screens/SCR-019.md#SCR-019) |
| 対応画面イベントID | [EVT-159](../../01_frontend/02_screen_events/EVT-159.md#EVT-159) |
| エンドポイント | `/withdrawal-requests` |
| HTTPメソッド | POST |
| 認証 | オーナー専有、再認証必須 |
| 認可 | オーナー専有 |

## 処理概要

オーナーの退会を申請する。

| 処理 ID | 処理内容                                       |
|---------|------------------------------------------------|
| P-01    | 認証・認可(オーナー専有、再認証必須)を検証する |
| P-02    | 退会申請を登録し受付状態とする                 |
| P-03    | 受付結果を返却する                             |
