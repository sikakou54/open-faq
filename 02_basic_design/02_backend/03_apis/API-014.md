# <span id="API-014"></span>API-014: 契約設定取得

> **このページは 契約設定取得 API の契約を定義します。** 契約レベルの設定情報(組織名・連絡先メール)を取得する。連絡先メールはオーナーの `M_USER.email` を参照する。

## 項目

| 項目 | 内容 |
|---|---|
| API ID | `API-014` |
| API名 | 契約設定取得 |
| トレーサビリティID | [TR-022](../../00_traceability/index.md#TR-022) |
| エンドポイント | `/owner/settings` |
| HTTPメソッド | GET |
| 認証 | 要(Cookie) |
| 認可 | オーナー専有 |

## 処理概要

契約レベルの設定情報(組織名・連絡先メール)を取得する。連絡先メールはオーナーの `M_USER.email` を参照する。

| 処理 ID | 処理内容                                         |
|---------|--------------------------------------------------|
| P-01    | 認証・認可(オーナー専有)を検証する               |
| P-02    | `M_CONTRACT` から組織名(`organization`)を、オーナーの `M_USER` から連絡先メール(`email`)を取得する |
| P-03    | 整形して返却する                                 |

## リクエスト

#### 認証ヘッダ

## レスポンス

**レスポンス(200)**

```json
{
  "organizationName": "株式会社サンプル",
  "contactEmail": "owner@example.com"
}
```

| 項目               | 型     | 説明                                               |
|--------------------|--------|----------------------------------------------------|
| `organizationName` | string | 組織名(`M_CONTRACT.organization`)                  |
| `contactEmail`     | string | 請求・重要通知の連絡先メール(オーナーの `M_USER.email`・読み取り専用) |
