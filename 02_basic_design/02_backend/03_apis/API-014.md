# <span id="API-014"></span>API-014: アカウント設定取得

> **このページは アカウント設定取得 API の仕様を定義します。** ログイン中ユーザーのアカウント/プロフィール設定情報(連絡先メール)を取得する。連絡先メールはユーザー本人の `M_USER.email` を参照する。

## 項目

| 項目 | 内容 |
|---|---|
| API ID | `API-014` |
| API名 | アカウント設定取得 |
| トレーサビリティID | [TR-022](../../00_traceability/index.md#TR-022) |
| エンドポイント | `/owner/settings` |
| HTTPメソッド | GET |
| 認証 | 要(Cookie) |
| 認可 | 本人(ログインユーザー) |

## 処理概要

ログイン中ユーザーのアカウント/プロフィール設定情報(連絡先メール)を取得する。連絡先メールはユーザー本人の `M_USER.email` を参照する。

| 処理 ID | 処理内容                                         |
|---------|--------------------------------------------------|
| P-01    | 認証・認可(ログインユーザー本人)を検証する         |
| P-02    | ログインユーザーの `M_USER` から連絡先メール(`email`)を取得する |
| P-03    | 整形して返却する                                 |

## リクエスト

#### 認証ヘッダ

本 API は利用者セッション(Cookie)による認証を必要とする。

| ヘッダ | 要否 | 説明 |
|---|---|---|
| `Cookie: session=<token>` | 必須 | 利用者セッション |

## バリデーション

入力値の検証規則は `## リクエスト` の各パラメータ説明を参照する。

## エラー

本 API は認証済みユーザーが自身のアカウント設定を取得する。認可上の固有エラーはなく、未認証時は共通の認証エラーで応答する。
