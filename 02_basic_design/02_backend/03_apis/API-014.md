# <span id="API-014"></span>API-014: 契約設定取得

> **このページは 契約設定取得 API の契約を定義します。** 契約レベルの設定情報(契約名・請求連絡先メール・タイムゾーン)を取得する。

## 項目

| 項目 | 内容 |
|---|---|
| API ID | `API-014` |
| API名 | 契約設定取得 |
| 対応業務UC | [UC-022](../../../01_requirements/04_business_usecases/UC-022.md#UC-022) |
| 対応画面ID | [SCR-029](../../01_frontend/01_screens/SCR-029.md#SCR-029) |
| 対応画面イベントID | EVT-215 |
| エンドポイント | `/owner/settings` |
| HTTPメソッド | GET |
| 認証 | 要(Cookie) |
| 認可 | オーナー専有 |

## 処理概要

契約レベルの設定情報(契約名・請求連絡先メール・タイムゾーン)を取得する。

| 処理 ID | 処理内容                                         |
|---------|--------------------------------------------------|
| P-01    | 認証・認可(オーナー専有)を検証する               |
| P-02    | `M_CONTRACT` から契約名・連絡先メール・タイムゾーンを取得する |
| P-03    | 整形して返却する                                 |

## レスポンス

**レスポンス(200)**

```json
{
  "contractName": "株式会社サンプル",
  "contactEmail": "billing@example.com",
  "timezone": "Asia/Tokyo"
}
```

| 項目            | 型     | 説明                                               |
|-----------------|--------|----------------------------------------------------|
| `contractName`  | string | 契約名(読み取り専用・変更不可)                     |
| `contactEmail`  | string | 請求・重要通知の連絡先メールアドレス               |
| `timezone`      | string | 契約のタイムゾーン(IANA タイムゾーン識別子)        |
