# <span id="API-006"></span>API-006: メール確認

> **このページは メール確認 API の仕様を定義します。** 登録メールアドレスを確認する。

| ID | 業務ユースケースID | イベント(画面ID EVT-NN) | テーブルID |
|----|----|----|----|
| API-006 | [UC-003](../../../01_requirements/04_business_usecases/UC-003.md#UC-003) | SCR-018 EVT-01 | [TBL-001](../04_database/TBL-001.md#TBL-001) ・ [TBL-014](../04_database/TBL-014.md#TBL-014) ・ [TBL-027](../04_database/TBL-027.md#TBL-027) |

| API名 | エンドポイント | HTTPメソッド |
|----|----|----|
| メール確認 | `/auth/email-verifications/{token}` | POST |

## 処理概要

登録メールアドレスを確認する。

| 処理 ID | 処理内容                                       |
|---------|------------------------------------------------|
| P-01    | メール確認トークンの有効性(期限)を検証する     |
| P-02    | 対象アカウントのメールアドレスを確認済みにする |
| P-03    | 確認トークンを使用済みにする                   |
| P-04    | 確認結果を返す                                 |

## リクエスト

#### 認証ヘッダ

本 API はセッション認証を必要とせず、検証用トークンを URL パスで受け取って検証する。

| ヘッダ | 要否 | 説明 |
|---|---|---|
| `Idempotency-Key: <ULID>` | 任意 | 冪等性(再送の重複防止) |

#### Query Parameters

```json
{}
```

#### Path Parameters

```json
{ "token": "メール確認トークン" }
```

| 項目    | 型     | 説明                                     |
|---------|--------|------------------------------------------|
| `token` | string | メール確認トークン(必須・パス末尾で指定) |

#### Request Body

```json
{}
```

## レスポンス

**レスポンス(200)**

```json
{ "userId": "...", "verifiedAt": "..." }
```

| 項目         | 型                | 説明                               |
|--------------|-------------------|------------------------------------|
| `userId`     | string            | メール確認の対象アカウント ID      |
| `verifiedAt` | string (ISO 8601) | メールアドレスを確認済みとした日時 |

## バリデーション

入力値の検証規則は `## リクエスト` の各パラメータ説明を参照する。

## エラー

| HTTP ステータス | エラーコード                          | 内容             |
|-----------------|---------------------------------------|------------------|
| 400 | `VALIDATION_ERROR` → [ERR-001](../../05_errors/ERR-001.md#ERR-001) | トークン形式不正 |
| 410 | `TOKEN_EXPIRED` → [ERR-006](../../05_errors/ERR-006.md#ERR-006) | トークン期限切れ |
| 410 | `TOKEN_USED` → [ERR-007](../../05_errors/ERR-007.md#ERR-007) | トークン使用済み |
| 404 | `TOKEN_NOT_FOUND` → [ERR-008](../../05_errors/ERR-008.md#ERR-008) | トークンが存在しない |
