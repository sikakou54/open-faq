## 処理概要

指定プロジェクトのメンバー割当一覧を返す。

| 処理 ID | 処理内容                                                |
|---------|---------------------------------------------------------|
| P-01    | 認証・認可(オーナー / 当該プロジェクトのメンバー)を検証する |
| P-02    | 指定プロジェクトの有効な割当に M_USER を結合し、email / name / status と参加日(granted_at)を取得する。各利用者が当該プロジェクトのオーナー(`M_PROJECTS.owner_user_id` と一致)かどうかを判定して付与する |
| P-03    | ページング(カーソル)で整形して返す                      |

## リクエスト

#### 認証ヘッダ

本 API は利用者セッション(Cookie)による認証を必要とする。

| ヘッダ | 要否 | 説明 |
|---|---|---|
| `Cookie: session=<token>` | 必須 | 利用者セッション |

#### Query Parameters

```json
{
  "q": "(任意)",
  "status": "all | active | pending_activation",
  "cursor": "(共通仕様)",
  "limit": "(共通仕様)"
}
```

| 項目     | 型      | 説明                                             |
|----------|---------|--------------------------------------------------|
| `q`      | string  | 検索文字列(任意・メンバーの表示名 / メールアドレスに部分一致で絞り込み) |
| `status` | enum    | 招待状態フィルタ(任意・`all` 既定 / `active`=有効 / `pending_activation`=招待中・有効化待ち) |
| `cursor` | string  | 次ページ取得用カーソル(任意・共通ページング仕様) |
| `limit`  | integer | 1 ページの取得件数(任意・共通ページング仕様)     |

#### Path Parameters

```json
{ "id": "プロジェクト ID" }
```

| 項目 | 型     | 説明                                      |
|------|--------|-------------------------------------------|
| `id` | string | 対象プロジェクトの ID(必須・パス内で指定) |

#### Request Body

```json
{}
```

## レスポンス

**レスポンス(200)**

```json
{
  "items": [
    {
      "id": "01HZ...",
      "email": "member@...",
      "name": "田中 太郎" | null,
      "status": "active|pending_activation",
      "isOwner": false,
      "joinedAt": "2025-11-02T..." | null,
      "invitationExpiresAt": "2026-05-20T..."
    }
  ],
  "nextCursor": "01HZ..." | null
}
```

| 項目 | 型 | 説明 |
|----|----|----|
| `items` | array\<object\> | 指定プロジェクトのメンバー割当一覧 |
| `　└ id` | string | メンバー(利用者)の ID |
| `　└ email` | string | メンバーのメールアドレス |
| `　└ name` | string \| null | メンバーの表示名(招待中=未有効化は `null`) |
| `　└ status` | enum | アカウント状態(`active` / `pending_activation`) |
| `　└ isOwner` | boolean | 当該プロジェクトのオーナーかどうか(`M_PROJECTS.owner_user_id` と一致する利用者は `true`) |
| `　└ joinedAt` | string (ISO 8601) \| null | 当該プロジェクトへの参加日(招待中=未有効化は `null`) |
| `　└ invitationExpiresAt` | string (ISO 8601) | 未有効化時の招待トークン有効期限 |
| `nextCursor` | string | 次ページ取得用カーソル(末尾は `null`) |

> [!NOTE]
> **補足** `items[]` は URL で指定したプロジェクトの割当だけを返す。最終ログイン日時と他プロジェクトの割当は返さない。
