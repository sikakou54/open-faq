# <span id="API-066"></span>API-066: プロジェクト削除影響プレビュー取得

> **このページは プロジェクト削除影響プレビュー取得 API の契約を定義します。** プロジェクト削除で影響する関連データの件数と取扱い選択肢を、削除確定前に返す参照系 API。

## 項目

| 項目 | 内容 |
|---|---|
| API ID | `API-066` |
| API名 | プロジェクト削除影響プレビュー取得 |
| トレーサビリティID | [TR-079](../../00_traceability/index.md#TR-079) |
| エンドポイント | `/projects/{id}/deletion-impact` |
| HTTPメソッド | GET |
| 認証 | セッション(Cookie) |
| 認可 | オーナー専有 |

## 処理概要

プロジェクト削除で影響する関連データ(FAQ・質問ログ・未解決質問・メンバー割当)の件数と取扱い選択肢を返す。削除確定前のプレビューであり、本 API は参照のみを行う(実際の削除は [API-018](API-018.md#API-018) が担う)。

| 処理 ID | 処理内容 |
|---------|----------|
| P-01 | リクエスト元が当該プロジェクトを保有するオーナーかを判定する(オーナー以外は 403) |
| P-02 | 対象プロジェクトの存在を確認する(存在しなければ 404) |
| P-03 | 削除で影響する FAQ・質問ログ・未解決質問・メンバー割当の件数を集計する |
| P-04 | 各関連データの取扱い選択肢(削除 / 保持など)を添えて返す |

## リクエスト

#### 認証ヘッダ

本 API は利用者セッション(Cookie)による認証を必要とする参照系であり、CSRF トークンは不要である。

| ヘッダ | 要否 | 説明 |
|---|---|---|
| `Cookie: session=<token>` | 必須 | 利用者セッション |

#### Query Parameters

```json
{}
```

#### Path Parameters

```json
{
  "id": "対象プロジェクト ID"
}
```

| 項目 | 型 | 説明 |
|---|---|---|
| `id` | string | 対象プロジェクト ID(必須) |

## レスポンス

削除で影響する関連データの件数と取扱い選択肢を返す。

**レスポンス(200)**

```json
{
  "projectId": "prj_01",
  "impact": {
    "faqCount": 124,
    "questionLogCount": 3820,
    "openInquiryCount": 17,
    "memberAssignmentCount": 5
  },
  "options": ["delete_all"]
}
```

| 項目 | 型 | 説明 |
|---|---|---|
| `projectId` | string | 対象プロジェクト ID |
| `impact.faqCount` | number | 削除で影響する FAQ 件数 |
| `impact.questionLogCount` | number | 削除で影響する質問ログ件数 |
| `impact.openInquiryCount` | number | 削除で影響する未解決質問件数 |
| `impact.memberAssignmentCount` | number | 削除で解除されるメンバー割当件数 |
| `options` | array | 関連データの取扱い選択肢 |

## バリデーション

入力値の検証規則は `## リクエスト` の各パラメータ説明を参照する。

## エラー

| HTTP ステータス | エラーコード | 内容 |
|-----------------|---------------------|--------------------------------|
| 403 | `OWNER_ONLY` → [ERR-017](../../05_errors/ERR-017.md#ERR-017) | オーナー以外のユーザーのアクセス |
| 404 | `PROJECT_NOT_FOUND` → [ERR-013](../../05_errors/ERR-013.md#ERR-013) | 対象プロジェクトが存在しない |
