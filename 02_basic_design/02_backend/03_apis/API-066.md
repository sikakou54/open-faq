| API-066 | プロジェクト削除影響プレビュー取得 | `/projects/{id}/deletion-impact` | GET |

## 処理概要

プロジェクト削除により自動的に論理削除される関連データ(FAQ・質問ログ・未解決質問・メンバー割当)の件数を返す。削除確定前のプレビューであり、本 API は参照のみを行う(実際の削除は [API-018](API-018.md#API-018) が担う)。

| 処理 ID | 処理内容 |
|---------|----------|
| P-01 | リクエスト元が当該プロジェクトを保有するオーナーかを判定する(オーナー以外は 403) |
| P-02 | 対象プロジェクトの存在を確認する(存在しなければ 404) |
| P-03 | 削除により自動的に論理削除される FAQ・質問ログ・未解決質問・メンバー割当の件数を集計して返す |

## リクエスト

#### 認証ヘッダ

本 API は利用者セッション(Cookie)による認証を必要とする参照系であり、CSRF トークンは不要である。

| ヘッダ | 要否 | 説明 |
|---|---|---|
| `Cookie: session=<token>` | 必須 | 利用者セッション |

#### Query Parameters

```json
{}
```

#### Path Parameters

```json
{
  "id": "対象プロジェクト ID"
}
```

| 項目 | 型 | 説明 |
|---|---|---|
| `id` | string | 対象プロジェクト ID(必須) |

## レスポンス

削除により自動的に論理削除される関連データの件数を返す。

**レスポンス(200)**

```json
{
  "projectId": "prj_01",
  "impact": {
    "faqCount": 124,
    "questionLogCount": 3820,
    "openInquiryCount": 17,
    "memberAssignmentCount": 5
  }
}
```

| 項目 | 型 | 説明 |
|---|---|---|
| `projectId` | string | 対象プロジェクト ID |
| `impact.faqCount` | number | 削除により自動的に論理削除される FAQ 件数 |
| `impact.questionLogCount` | number | 削除により自動的に論理削除される質問ログ件数 |
| `impact.openInquiryCount` | number | 削除により自動的に論理削除される未解決質問件数 |
| `impact.memberAssignmentCount` | number | 削除で解除されるメンバー割当件数 |

## バリデーション

入力値の検証規則は `## リクエスト` の各パラメータ説明を参照する。

## エラー

| HTTP ステータス | エラーコード | 内容 |
|-----------------|---------------------|--------------------------------|
| 403 | `OWNER_ONLY` → [ERR-015](../../05_errors/ERR-015.md#ERR-015) | オーナー以外のユーザーのアクセス |
| 404 | `PROJECT_NOT_FOUND` → [ERR-011](../../05_errors/ERR-011.md#ERR-011) | 対象プロジェクトが存在しない |
