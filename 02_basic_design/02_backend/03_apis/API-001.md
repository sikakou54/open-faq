# <span id="API-001"></span>API-001: 新規登録

> **このページは 新規登録 API の契約を定義します。** オーナーアカウントを新規登録し、メール確認を要求する。

## 項目

| 項目 | 内容 |
|---|---|
| API ID | `API-001` |
| API名 | 新規登録 |
| トレーサビリティID | [TR-002](../../00_traceability/index.md#TR-002) ・ [TR-003](../../00_traceability/index.md#TR-003) |
| エンドポイント | `/auth/signup` |
| HTTPメソッド | POST |
| 認証 | 不要 |
| 認可 | —(公開) |

## 処理概要

オーナーアカウントを新規登録し、メール確認を要求する。

| 処理 ID | 処理内容 |
|----|----|
| P-01 | 入力値(メール形式・パスワード強度・規約 / プライバシー同意)を検証する |
| P-02 | `M_USER`(オーナー本体)と `M_CONTRACT` を作成する |
| P-03 | 利用規約・プライバシーポリシーへの同意を記録する |
| P-04 | メール確認トークンを発行し確認メールを送信する |
| P-05 | 受付結果を返す |

## リクエスト

#### 認証ヘッダ

本 API は公開エンドポイントで、認証セッションを必要としない(資格情報・入力はリクエストボディで送出)。

| ヘッダ | 要否 | 説明 |
|---|---|---|
| `Idempotency-Key: <ULID>` | 任意 | 冪等性(再送の重複防止) |

#### Query Parameters

```json
{}
```

#### Path Parameters

```json
{}
```

#### Request Body

```json
{
  "email": "admin@example.com",
  "password": "P@ssw0rd123!",
  "agreedTerms": true,
  "agreedPrivacy": true
}
```

| 項目              | 型      | 説明                                       |
|-------------------|---------|--------------------------------------------|
| `email`           | string  | 登録するメールアドレス(必須・RFC 5322 準拠のメール形式・最大 254 文字) |
| `password`        | string  | 初回パスワード(必須・12 文字以上、英大文字 / 小文字 / 数字 / 記号のうち 3 種類以上 = FR-006) |
| `agreedTerms`     | boolean | 利用規約への同意(必須・`true`)             |
| `agreedPrivacy`   | boolean | プライバシーポリシーへの同意(必須・`true`) |

## レスポンス

**レスポンス(202)**

```json
{ "ok": true }
```

| 項目 | 型      | 説明                                                  |
|------|---------|-------------------------------------------------------|
| `ok` | boolean | 登録受付の成否(確認メール送信を受け付けた場合 `true`) |

## バリデーション

入力値の検証規則は `## リクエスト` の各パラメータ説明を参照する。主な規則は次のとおり。

- `email`: 必須・RFC 5322 準拠のメール形式・最大 254 文字。
- `password`: 必須・12 文字以上、英大文字 / 小文字 / 数字 / 記号のうち 3 種類以上(FR-006)。
- `agreedTerms` / `agreedPrivacy`: 必須・`true`。
- 既存メール重複(処理概要 P-02 のチェック失敗)は 409 `EMAIL_ALREADY_USED` を返す。

## エラー

| HTTP ステータス | エラーコード                   | 内容               |
|-----------------|--------------------------------|--------------------|
| 400 | `VALIDATION_ERROR` → [ERR-001](../../05_errors/ERR-001.md#ERR-001) | 入力値の検証エラー(メール形式 / パスワード強度) |
| 409 | `EMAIL_ALREADY_USED` → [ERR-016](../../05_errors/ERR-016.md#ERR-016) | メールアドレス重複(既存登録あり。P-02 のチェック失敗時) |

## 備考

- 本 API は **オーナーアカウントの新規登録**(`M_USER`(オーナー本体)・`M_CONTRACT` を新規作成し確認メールを送信)を担う。確認メール未達時の再送(確認待ち画面のメール再送)にも当面は本 API を用いる。将来、新規作成を伴わない再送専用 API への分離を検討する(別途課題で追跡)。
