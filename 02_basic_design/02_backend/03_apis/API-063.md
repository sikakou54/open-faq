# <span id="API-063"></span>API-063: セットアップ進捗取得

> **このページは セットアップ進捗取得 API の契約を定義します。** ウィジェット利用開始までの設定ステップ(プロジェクト作成・FAQ 登録・ウィジェット埋め込み)の完了状態を返す。

## 項目

| 項目 | 内容 |
|---|---|
| API ID | `API-063` |
| API名 | セットアップ進捗取得 |
| トレーサビリティID | [TR-036](../../00_traceability/index.md#TR-036) |
| エンドポイント | `/onboarding/progress` |
| HTTPメソッド | GET |
| 認証 | Cookie |
| 認可 | オーナー / メンバー |

## 処理概要

ウィジェットを利用可能にするための 3 ステップ(プロジェクト作成 / FAQ 登録 / ウィジェット埋め込みコード配置)の完了状態と、全体完了状態を返す。

| 処理 ID | 処理内容 |
|----|----|
| P-01 | 認証・認可(契約スコープのアクセス権)を検証する |
| P-02 | プロジェクトが 1 件以上あるか(`project_created`)を判定する |
| P-03 | FAQ が 1 件以上あるか(`faq_registered`)を判定する |
| P-04 | 許可ドメインが 1 件以上あるか(`widget_embedded`)を判定する |
| P-05 | 各ステップの完了状態と全体完了状態を整形して返却する |

## レスポンス

**レスポンス(200)**

```json
{
  "steps": [
    { "key": "project_created", "label": "プロジェクトを作成", "done": true },
    { "key": "faq_registered", "label": "FAQを登録", "done": true },
    { "key": "widget_embedded", "label": "ウィジェット埋め込みコードを配置", "done": false }
  ],
  "completed": false
}
```

| 項目 | 型 | 説明 |
|----|----|----|
| `steps` | array\<object\> | セットアップステップの配列(定義順) |
| `　└ key` | string | ステップ識別子。`project_created` / `faq_registered` / `widget_embedded` |
| `　└ label` | string | ステップの表示名 |
| `　└ done` | boolean | 当該ステップの完了状態(`true` で完了) |
| `completed` | boolean | 全ステップが完了しているか(`true` でウィジェット利用可能) |

## バリデーション

入力パラメータを持たないため、本 API 固有のバリデーションはない。

## エラー

本 API 固有のエラーコードはない。想定エラー(未認証 401 / 権限不足 403)は認可ミドルウェア共通とする。
