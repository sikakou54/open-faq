# <span id="API-023"></span>API-023: プロジェクト割当解除

> **このページは プロジェクト割当解除 API の契約を定義します。** 対象プロジェクトの `M_PRJ_USERS.valid=0` を更新する。

## 項目

| 項目 | 内容 |
|---|---|
| API ID | `API-023` |
| API名 | プロジェクト割当解除 |
| トレーサビリティID | [TR-021](../../00_traceability/index.md#TR-021) ・ [TR-077](../../00_traceability/index.md#TR-077) |
| エンドポイント | `/projects/{id}/members/{userId}` |
| HTTPメソッド | DELETE |
| 認証 | Cookie + CSRF + 再認証必須 |
| 認可 | オーナー / 当該プロジェクトのメンバー |

## 処理概要

対象プロジェクトの `M_PRJ_USERS.valid=0` を更新する。

| 処理 ID | 処理内容 |
|----|----|
| P-00 | 再認証トークン(リクエストボディ `reauthToken`)を検証する |
| P-01 | 認証・認可(オーナー / 当該プロジェクトのメンバー)と再認証を検証する |
| P-02 | 対象が自分自身またはオーナーの場合は拒否する |
| P-03 | 当該プロジェクトの割当を論理削除する |
| P-04 | 他プロジェクトの有効割当が 0 件になる場合はアカウントを論理削除し、全セッション失効と未使用招待トークン失効を行う |
| P-05 | 解除結果(アカウント無効化の有無)を返す |

## リクエスト

#### 認証ヘッダ

本 API は利用者セッション(Cookie)と CSRF トークンを必要とする状態変更系である。

| ヘッダ | 要否 | 説明 |
|---|---|---|
| `Cookie: session=<token>` | 必須 | 利用者セッション |
| `X-CSRF-Token: <token>` | 必須 | CSRF 対策(状態変更系) |
| `Idempotency-Key: <ULID>` | 任意 | 冪等性(書き込み再送の重複防止) |

> 本操作は再認証必須。直近の再認証([API-005](API-005.md#API-005))で得た再認証状態を伴う必要がある。

#### Query Parameters

```json
{}
```

#### Path Parameters

```json
{
  "id": "プロジェクト ID",
  "userId": "対象ユーザー ID"
}
```

| 項目     | 型     | 説明                                                |
|----------|--------|-----------------------------------------------------|
| `id`     | string | 割当を解除するプロジェクトの ID(必須・パス内で指定) |
| `userId` | string | 割当を解除する対象利用者の ID(必須・パス内で指定)   |

#### Request Body

```json
{ "reauthToken": "reauth_..." }
```

| 項目 | 型 | 説明 |
|----|----|----|
| `reauthToken` | string | 再認証トークン(必須・API-005 で取得) |

## レスポンス

**レスポンス(200)**

```json
{ "userId": "...", "projectId": "...", "accountDeactivated": true|false }
```

| 項目 | 型 | 説明 |
|----|----|----|
| `userId` | string | 割当を解除した対象利用者の ID |
| `projectId` | string | 割当を解除したプロジェクトの ID |
| `accountDeactivated` | boolean | 他プロジェクトの有効割当が 0 件となりアカウントを論理削除したかどうか |

## バリデーション

入力値の検証規則は `## リクエスト` の各パラメータ説明を参照する。

## エラー

| HTTP ステータス | エラーコード              | 内容                         |
|-----------------|---------------------------|------------------------------|
| 401 | `REAUTH_REQUIRED` → [ERR-013](../../05_errors/ERR-013.md#ERR-013) | 再認証トークンが無効または未提示 |
| 403 | `OWNER_PROTECTED` → [ERR-021](../../05_errors/ERR-021.md#ERR-021) | オーナーは解除不可 |
| 403 | `SELF_MUTATION_FORBIDDEN` → [ERR-022](../../05_errors/ERR-022.md#ERR-022) | 自分自身は解除不可 |
| 403 | `PROJECT_ACCESS_DENIED` → [ERR-019](../../05_errors/ERR-019.md#ERR-019) | 当該プロジェクトへの権限なし |
| 404 | `NOT_FOUND` → [ERR-017](../../05_errors/ERR-017.md#ERR-017) | 対象が存在しない |
