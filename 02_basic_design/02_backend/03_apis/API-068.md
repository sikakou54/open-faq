# <span id="API-068"></span>API-068: FAQ取込ジョブ状態取得

> **このページは FAQ取込ジョブ状態取得 API の契約を定義します。** 指定した取込ジョブの状態・進捗・結果サマリを返す([SCR-010](../../01_frontend/01_screens/SCR-010.md#SCR-010) のポーリング先)。

## 項目

| 項目 | 内容 |
|---|---|
| API ID | `API-068` |
| API名 | FAQ取込ジョブ状態取得 |
| トレーサビリティID | [TR-050](../../00_traceability/index.md#TR-050) |
| エンドポイント | `/faqs/import/{jobId}` |
| HTTPメソッド | GET |
| 認証 | セッション(Cookie) |
| 認可 | オーナー / 当該プロジェクトのメンバー |

## 処理概要

指定した取込ジョブ ID の FAQ 一括取込ジョブの状態・進捗・結果サマリを返す。[SCR-010](../../01_frontend/01_screens/SCR-010.md#SCR-010) が [API-028](API-028.md#API-028) で受領した `jobId` をキーに、本 API を定期的にポーリングして進捗バー・結果サマリ・失敗行一覧を更新する。

| 処理 ID | 処理内容 |
|----|----|
| P-01 | 認証・認可(オーナー / 当該プロジェクトのメンバー)を検証する |
| P-02 | 指定ジョブが当該契約・プロジェクトに属するか検証する(不一致は 403) |
| P-03 | 指定ジョブが存在するか検証する(不在は 404) |
| P-04 | ジョブの状態・進捗(総数 / 処理済み / 成功 / 失敗)・失敗行明細を返す |

## リクエスト

#### 認証ヘッダ

本 API は利用者セッション(Cookie)による認証を必要とする。

| ヘッダ | 要否 | 説明 |
|---|---|---|
| `Cookie: session=<token>` | 必須 | 利用者セッション |

#### Query Parameters

```json
{}
```

#### Path Parameters

```json
{ "jobId": "取込ジョブ ID(必須)" }
```

| 項目 | 型 | 説明 |
|----|----|----|
| `jobId` | string | 状態取得対象の取込ジョブ ID(必須・パス内)。[API-028](API-028.md#API-028) のレスポンスで受領した値 |

#### Request Body

```json
{}
```

## レスポンス

**レスポンス(200)**

```json
{ "jobId": "...", "status": "processing", "total": 1000, "processed": 320, "succeeded": 300, "failed": 20, "errors": [ { "row": 12, "code": "CSV_INVALID", "message": "..." } ] }
```

| 項目 | 型 | 説明 |
|----|----|----|
| `jobId` | string | 取込ジョブ ID |
| `status` | enum | ジョブ状態(`queued` / `processing` / `completed` / `failed`) |
| `total` | integer | 取込対象の総行数(受付確定前は `null`) |
| `processed` | integer | 処理済み行数(成功 + 失敗) |
| `succeeded` | integer | 成功行数 |
| `failed` | integer | 失敗行数 |
| `errors` | array | 失敗行明細の配列(失敗行がない場合は空配列) |
| `errors[].row` | integer | 失敗した CSV の行番号(ヘッダ行を除く 1 始まり) |
| `errors[].code` | string | 失敗理由のエラーコード(例 `CSV_INVALID`) |
| `errors[].message` | string | 失敗理由の説明 |

## バリデーション

入力値の検証規則は `## リクエスト` の各パラメータ説明を参照する。

- `jobId`: 必須・パス内。

## エラー

| HTTP ステータス | エラーコード | 内容 |
|-----------------|---------------------|--------------------------------|
| 403 | `PROJECT_ACCESS_DENIED` → [ERR-019](../../05_errors/ERR-019.md#ERR-019) | 当該プロジェクトへの権限なし |
| 404 | `NOT_FOUND` → [ERR-017](../../05_errors/ERR-017.md#ERR-017) | 指定した取込ジョブが存在しない |

共通の認証・認可エラー(401 等)は [§0 共通仕様](index.md#common) に従う。

## 備考

取込ジョブの作成・進捗更新は FAQ 一括取り込みジョブ([SYS-016](../01_system/SYS-016.md#SYS-016))が行い、本 API は状態・進捗・結果サマリの参照のみを担う。
