# <span id="API-016"></span>API-016: プロジェクト一覧

> **このページは プロジェクト一覧 API の契約を定義します。** 契約配下のプロジェクト一覧を返す。

*版数 v2.0 ・ 更新 2026-06-21*

## 項目

| 項目 | 内容 |
|---|---|
| API ID | `API-016` |
| API名 | プロジェクト一覧 |
| 対応業務UC | [UC-014](../../../01_requirements/04_business_usecases/UC-014.md#UC-014) |
| 対応画面ID | [SCR-004](../../01_frontend/01_screens/SCR-004.md#SCR-004) |

| HTTPメソッド | GET |
| 認証 | Cookie |
| 認可 | **オーナー専有** |

## 処理概要

契約配下のプロジェクト一覧を返す。

| 処理 ID | 処理内容 |
|----|----|
| P-01 | 認証・認可(オーナー専有)を検証する |
| P-02 | 契約配下の有効なプロジェクトを取得する(FAQ 件数・許可ドメイン等を付帯) |
| P-03 | ページング(カーソル)で整形して返す |

## リクエスト

#### Query Parameters

```json
{
  "cursor": "(共通仕様)",
```

| `limit`  | integer | 1 ページの取得件数(任意・共通ページング仕様)     |

## レスポンス

**レスポンス(200)**

```json
{
  "items": [
    {
      "id": "01HZ...",
      "name": "...",
      "status": "active|deleted",
      "createdAt": "...",
      "faqCount": 42,
      "widgetKey": "pk_live_...",
      "allowedDomains": ["example.com", "*.example.com"],
      "contactEmail": "contact@...",
      "contactEmailVerifiedAt": "..." | null
    }
  ],
  "nextCursor": null
}
```

| 項目 | 型 | 説明 |
|----|----|----|
| `items` | array\<object\> | 契約配下のプロジェクト一覧 |
| `　└ id` | string | プロジェクト ID |
| `　└ name` | string | プロジェクト名 |
| `　└ status` | enum | プロジェクト状態(`active` / `deleted`) |
| `　└ createdAt` | string (ISO 8601) | 作成日時 |
| `　└ faqCount` | integer | 当該プロジェクトの FAQ 件数 |
| `　└ widgetKey` | string | ウィジェット公開キー |
| `　└ allowedDomains` | array | 許可ドメインのリスト |
| `　└ contactEmail` | string | プロジェクト連絡先メールアドレス |
| `　└ contactEmailVerifiedAt` | string (ISO 8601) | 連絡先メール確認日時(未確認は `null`) |
| `nextCursor` | string | 次ページ取得用カーソル(末尾は `null`) |
