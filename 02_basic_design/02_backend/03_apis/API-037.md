# <span id="API-037"></span>API-037: ウィジェット起動

> **このページは ウィジェット起動 API の仕様を定義します。** 公開キーとドメインを検証しウィジェットセッションを発行する。

## 項目

| 項目 | 内容 |
|---|---|
| API ID | `API-037` |
| API名 | ウィジェット起動 |
| トレーサビリティID | [TR-041](../../00_traceability/index.md#TR-041) ・ [TR-043](../../00_traceability/index.md#TR-043) ・ [TR-061](../../00_traceability/index.md#TR-061) |
| エンドポイント | `/widget/v1/bootstrap` |
| HTTPメソッド | POST |
| 認証 | 不要(ドメイン検証) |
| 認可 | end_user(公開キー) |

## 処理概要

公開キーとドメインを検証しウィジェットセッションを発行する。

| 処理 ID | 処理内容                                               |
|---------|--------------------------------------------------------|
| P-01    | 公開キーを検証し、対象プロジェクトを特定する           |
| P-02    | 提示キーが現行公開キーに一致しない場合、旧キー(`T_PRJ_LEGACY_KEYS`)を照合し、猶予期限(`grace_until`)が未経過なら許可し、経過または不一致なら失効として拒否する(遅延失効) |
| P-03    | リクエスト元オリジンが許可ドメインに含まれるか検証する |
| P-04    | 対象プロジェクトの課金状態(サスペンション=支払い不能で停止中か)を確認する |
| P-05    | ウィジェットセッショントークンを発行する               |
| P-06    | プロジェクト表示設定を整形して返却する                 |

## リクエスト

#### 認証ヘッダ

本 API は認証ヘッダを必要とせず、公開キーとオリジンをリクエストボディで受け取りドメイン検証で認可する。

| ヘッダ | 要否 | 説明 |
|---|---|---|
| —(なし) | — | 公開キー(`publicKey`)・オリジン(`origin`)は本文で送出し、ドメイン検証で認可する |

#### Query Parameters

```json
{}
```

#### Path Parameters

```json
{}
```

#### Request Body

```json
{
  "publicKey": "pk_live_abc123...",
  "origin": "https://example.com"
}
```

| 項目        | 型     | 説明                                               |
|-------------|--------|----------------------------------------------------|
| `publicKey` | string | プロジェクトの公開キー。必須                       |
| `origin`    | string | リクエスト元オリジン(許可ドメイン検証に使用)。必須 |

## レスポンス

**レスポンス(200)**

```json
{
  "sessionToken": "wst_...",
  "expiresIn": 3600,
  "projectConfig": {
    "headerColor": "#06c",
    "placement": "bottom-right",
    "headerTitle": "お問い合わせ",
    "contactEmail": "contact@..."
  }
}
```

| 項目               | 型      | 説明                                     |
|--------------------|---------|------------------------------------------|
| `sessionToken`     | string  | 発行されたウィジェットセッショントークン |
| `expiresIn`        | integer | セッショントークンの有効期限(秒)         |
| `projectConfig`    | object  | ウィジェット表示用のプロジェクト設定     |
| `　└ headerColor`  | string  | ヘッダーの表示色                         |
| `　└ placement`    | string  | ウィジェットの表示位置                   |
| `　└ headerTitle`  | string  | ヘッダーに表示するタイトル               |
| `　└ contactEmail` | string  | 問い合わせ先メールアドレス               |

## バリデーション

入力値の検証規則は `## リクエスト` の各パラメータ説明を参照する。

## エラー

| HTTP ステータス | エラーコード         | 内容           |
|-----------------|----------------------|----------------|
| 401 | `WIDGET_KEY_INVALID` → [ERR-026](../../05_errors/ERR-026.md#ERR-026) | 公開キーが不正(旧キーが猶予期限超過で失効済み、または現行・旧いずれにも一致しない場合を含む) |
| 403 | `DOMAIN_NOT_ALLOWED` → [ERR-027](../../05_errors/ERR-027.md#ERR-027) | 許可ドメイン外 |
| 403 | `BILLING_ACCOUNT_SUSPENDED` → [ERR-004](../../05_errors/ERR-004.md#ERR-004) | サスペンション(支払い不能でプロジェクト課金停止中) |
