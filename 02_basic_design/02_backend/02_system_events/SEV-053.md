# <span id="SEV-053"></span>SEV-053: 既存お知らせへ集約

> **システム処理 SYS-028 のイベント「既存お知らせへ集約」を定義します。** 対応システム・対応業務UC・トリガー・呼出API・連携先と処理内容を記述します。

*種別 システムイベント設計 ・ ステータス ドラフト*

## 項目

| 項目 | 内容 |
|---|---|
| システムイベントID | `SEV-053` |
| イベント名 | 既存お知らせへ集約 |
| 対応システムID | [SYS-028](../01_system/SYS-028.md#SYS-028) |
| 対応業務UC | [UC-068](../../../01_requirements/04_business_usecases/UC-068.md#UC-068) |
| トリガー | 受信箱お知らせの生成要求発生時に、同一契約・同一イベント種別で一定時間内の既存お知らせが確認できたとき |
| 呼出API | — |
| 連携先 | 対象アカウント利用者の受信箱(集約結果の反映) |

## 処理

処理内容の正本は対応システム [SYS-028](../01_system/SYS-028.md#SYS-028) の §6 システムイベント一覧 を参照(二重管理を避けるため本ページには再掲しない)。本イベントは、同一契約・同一イベント種別で一定時間内に発生した既存のお知らせへ新規分をまとめ、受信箱が重複したお知らせで埋もれないよう 1 件へ集約する役割を担う。
