# <span id="SEV-043"></span>SEV-043: 配信状態更新

> **システム処理 SYS-023 のイベント「配信状態更新」を定義します。** 対応システム・対応業務UC・トリガー・呼出API・連携先と処理内容を記述します。

*種別 システムイベント設計 ・ ステータス ドラフト*

## 項目

| 項目 | 内容 |
|---|---|
| システムイベントID | `SEV-043` |
| イベント名 | 配信状態更新 |
| 対応システムID | [SYS-023](../01_system/SYS-023.md#SYS-023) |
| 対応業務UC | [UC-063](../../../01_requirements/04_business_usecases/UC-063.md#UC-063) |
| トリガー | メール配信事業者からの配信結果通知の受信(検証成功かつ新規受信) |
| 呼出API | [API-059](../03_apis/API-059.md#API-059) |
| 連携先 | 後段イベント(送信停止リスト登録) |

## 処理

処理内容の正本は対応システム [SYS-023](../01_system/SYS-023.md#SYS-023) の §6 システムイベント一覧 を参照(二重管理を避けるため本ページには再掲しない)。本イベントは、正当な送信元から発行された正規かつ新規の配信結果通知について、配信結果の種別(到達・不達・苦情・遅延)を判定し、対象通知の配信状態を判定結果へ更新する役割を担う。
