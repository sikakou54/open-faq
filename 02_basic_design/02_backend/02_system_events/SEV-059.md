# <span id="SEV-059"></span>SEV-059: ロック発動

> **システム処理 SYS-031 のイベント「ロック発動」を定義します。** 対応システム・対応業務UC・トリガー・呼出API・連携先と処理内容を記述します。

*種別 システムイベント設計 ・ ステータス ドラフト*

## 項目

| 項目 | 内容 |
|---|---|
| システムイベントID | `SEV-059` |
| イベント名 | ロック発動 |
| 対応システムID | [SYS-031](../01_system/SYS-031.md#SYS-031) |
| 対応業務UC | [UC-073](../../../01_requirements/04_business_usecases/UC-073.md#UC-073) |
| トリガー | 連続ログイン失敗回数が規定回数に到達 |
| 呼出API | [API-002](../03_apis/API-002.md#API-002) |
| 連携先 | 本人および必要な権限者への通知(ロック発動・救済導線案内) |

## 処理

処理内容の正本は対応システム [SYS-031](../01_system/SYS-031.md#SYS-031) の §6 システムイベント一覧 を参照(二重管理を避けるため本ページには再掲しない)。本イベントは連続失敗が規定回数に到達したアカウントを一定時間ロックし、本人および必要な権限者へロック発動を通知して救済導線を案内する役割を担う。
