# <span id="SEV-008"></span>SEV-008: 回答不可案内

> **システム処理 SYS-004 のイベント「回答不可案内」を定義します。** 対応システム・対応業務UC・トリガー・呼出API・連携先と処理内容を記述します。

*種別 システムイベント設計 ・ ステータス ドラフト*

## 項目

| 項目 | 内容 |
|---|---|
| システムイベントID | `SEV-008` |
| イベント名 | 回答不可案内 |
| 対応システムID | [SYS-004](../01_system/SYS-004.md#SYS-004) |
| 対応業務UC | [UC-054](../../../01_requirements/04_business_usecases/UC-054.md#UC-054) |
| トリガー | 未解決質問の記録完了 |
| 呼出API | — |
| 連携先 | ウィジェット利用者への回答不可案内(連絡先設定時は連絡先を併記) |

## 処理

処理内容の正本は対応システム [SYS-004](../01_system/SYS-004.md#SYS-004) の §6 システムイベント一覧 を参照(二重管理を避けるため本ページには再掲しない)。本イベントは、ウィジェット利用者へ回答できなかった旨を案内し、連絡先設定時は連絡先を併記して無言で失敗させない役割を担う(追跡用の管理識別子は提示しない)。
