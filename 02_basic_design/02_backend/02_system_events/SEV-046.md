# <span id="SEV-046"></span>SEV-046: 配信ログ記録

| 項目 | 内容 |

| システムイベントID | `SEV-046` |
| イベント名 | 配信ログ記録 |

## 処理

処理内容の正本は対応システム [SYS-024](../01_system/SYS-024.md#SYS-024) の §6 システムイベント一覧 を参照(二重管理を避けるため本ページには再掲しない)。本イベントは重要度に応じて送信したお知らせメールの送信結果を配信ログとして記録し、送信失敗時の再送(別ユースケース)へ引き渡せるようにする役割を担う。
