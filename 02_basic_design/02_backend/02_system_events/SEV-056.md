# <span id="SEV-056"></span>SEV-056: 監査記録

## 処理

処理内容の正本は対応システム [SYS-029](../01_system/SYS-029.md#SYS-029) の §6 システムイベント一覧 を参照(二重管理を避けるため本ページには再掲しない)。本イベントは、依存順に実施した物理削除の内容を監査記録として残し、削除の事実を後から追跡できるようにする役割を担う。
