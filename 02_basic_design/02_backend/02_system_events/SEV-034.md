# <span id="SEV-034"></span>SEV-034: 利用量サマリ反映

> **システム処理 SYS-018 のイベント「利用量サマリ反映」を定義します。** 対応システム・対応業務UC・トリガー・呼出API・連携先と処理内容を記述します。

*種別 システムイベント設計 ・ ステータス ドラフト*

## 項目

| 項目 | 内容 |
|---|---|
| システムイベントID | `SEV-034` |
| イベント名 | 利用量サマリ反映 |
| 対応システムID | [SYS-018](../01_system/SYS-018.md#SYS-018) |
| 対応業務UC | [UC-056](../../../01_requirements/04_business_usecases/UC-056.md#UC-056) |
| トリガー | 利用量集計・閾値判定の完了 |
| 呼出API | [API-041](../03_apis/API-041.md#API-041) |
| 連携先 | 管理画面の利用量サマリ(ダッシュボード・利用量画面) |

## 処理

処理内容の正本は対応システム [SYS-018](../01_system/SYS-018.md#SYS-018) の §6 システムイベント一覧 を参照(二重管理を避けるため本ページには再掲しない)。本イベントは集計結果を管理画面向けの利用量サマリへ短い遅延で反映し、利用者がダッシュボードや利用量画面で最新の利用状況を確認できる状態にする役割を担う。
