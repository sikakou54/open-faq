# <span id="SEV-032"></span>SEV-032: フォールバックアラート通知

> **システム処理 SYS-017 のイベント「フォールバックアラート通知」を定義します。** 対応システム・対応業務UC・トリガー・呼出API・連携先と処理内容を記述します。

*種別 システムイベント設計 ・ ステータス ドラフト*

## 項目

| 項目 | 内容 |
|---|---|
| システムイベントID | `SEV-032` |
| イベント名 | フォールバックアラート通知 |
| 対応システムID | [SYS-017](../01_system/SYS-017.md#SYS-017) |
| 対応業務UC | [UC-052](../../../01_requirements/04_business_usecases/UC-052.md#UC-052) |
| トリガー | しきい値の取得が長期に失敗しフォールバックで動作している間 |
| 呼出API | — |
| 連携先 | 運用へのアラート通知 |

## 処理

処理内容の正本は対応システム [SYS-017](../01_system/SYS-017.md#SYS-017) の §6 システムイベント一覧 を参照(二重管理を避けるため本ページには再掲しない)。本イベントは、しきい値設定の取得が長期に行えずグローバル既定値で推論を継続している間に、フォールバック動作中であることを運用へアラート通知する結果通知イベントの役割を担う。
