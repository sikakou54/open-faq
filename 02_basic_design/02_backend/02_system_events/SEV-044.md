# <span id="SEV-044"></span>SEV-044: 送信停止リスト登録

> **システム処理 SYS-023 のイベント「送信停止リスト登録」を定義します。** 対応システム・対応業務UC・トリガー・呼出API・連携先と処理内容を記述します。

*種別 システムイベント設計 ・ ステータス ドラフト*

## 項目

| 項目 | 内容 |
|---|---|
| システムイベントID | `SEV-044` |
| イベント名 | 送信停止リスト登録 |
| 対応システムID | [SYS-023](../01_system/SYS-023.md#SYS-023) |
| 対応業務UC | [UC-063](../../../01_requirements/04_business_usecases/UC-063.md#UC-063) |
| トリガー | 配信結果が不達(バウンス)または苦情と判定されたこと |
| 呼出API | [API-059](../03_apis/API-059.md#API-059) |
| 連携先 | 監査記録・配信事業者への受領応答 |

## 処理

処理内容の正本は対応システム [SYS-023](../01_system/SYS-023.md#SYS-023) の §6 システムイベント一覧 を参照(二重管理を避けるため本ページには再掲しない)。本イベントは、不達・苦情と判定された宛先を全契約横断の送信停止リストへ登録し、送信元ドメインの評判悪化を防ぐため以降の送信を抑制対象とする役割を担う。
