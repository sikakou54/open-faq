# <span id="SEV-047"></span>SEV-047: システム通知生成

## 処理

処理内容の正本は対応システム [SYS-025](../01_system/SYS-025.md#SYS-025) の §6 システムイベント一覧 を参照(二重管理を避けるため本ページには再掲しない)。本イベントは、解決した対象アカウント利用者の受信箱へ、イベント種別に応じた内容のシステム通知のお知らせを生成する役割を担う。
