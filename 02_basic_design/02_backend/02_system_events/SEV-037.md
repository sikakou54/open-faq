# <span id="SEV-037"></span>SEV-037: 受付停止

> **システム処理 SYS-020 のイベント「受付停止」を定義します。** 対応システム・対応業務UC・トリガー・呼出API・連携先と処理内容を記述します。

*種別 システムイベント設計 ・ ステータス ドラフト*

## 項目

| 項目 | 内容 |
|---|---|
| システムイベントID | `SEV-037` |
| イベント名 | 受付停止 |
| 対応システムID | [SYS-020](../01_system/SYS-020.md#SYS-020) |
| 対応業務UC | [UC-058](../../../01_requirements/04_business_usecases/UC-058.md#UC-058) |
| トリガー | ウィジェット利用者の質問送信時に上限到達を判定したとき |
| 呼出API | [API-046](../03_apis/API-046.md#API-046) |
| 連携先 | 後段イベント(安全な定型文応答) |

## 処理

処理内容の正本は対応システム [SYS-020](../01_system/SYS-020.md#SYS-020) の §6 システムイベント一覧 を参照(二重管理を避けるため本ページには再掲しない)。本イベントは当月質問数が月次上限件数に到達したと判定したとき、支払方法の有無に関わらず当該プロジェクトのウィジェット新規質問受付を停止する役割を担う。
