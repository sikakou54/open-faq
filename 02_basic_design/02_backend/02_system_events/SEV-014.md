# <span id="SEV-014"></span>SEV-014: 許可ドメイン上での起動

> **システム処理 SYS-007 のイベント「許可ドメイン上での起動」を定義します。** 対応システム・対応業務UC・トリガー・呼出API・連携先と処理内容を記述します。

*種別 システムイベント設計 ・ ステータス ドラフト*

## 項目

| 項目 | 内容 |
|---|---|
| システムイベントID | `SEV-014` |
| イベント名 | 許可ドメイン上での起動 |
| 対応システムID | [SYS-007](../01_system/SYS-007.md#SYS-007) |
| 対応業務UC | [UC-062](../../../01_requirements/04_business_usecases/UC-062.md#UC-062) |
| トリガー | 許可ドメイン照合で読み込み元が許可ドメインに一致したこと |
| 呼出API | [API-037](../03_apis/API-037.md#API-037) |
| 連携先 | 後段処理(対象プロジェクト範囲に限定したFAQ対応の提供) |

## 処理

処理内容の正本は対応システム [SYS-007](../01_system/SYS-007.md#SYS-007) の §6 システムイベント一覧 を参照(二重管理を避けるため本ページには再掲しない)。本イベントは、読み込み元が許可ドメインに一致する場合のみウィジェットを動作させ、対象プロジェクトの範囲内でFAQ・回答を取り扱う役割を担う。
