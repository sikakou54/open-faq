# <span id="SEV-001"></span>SEV-001: メンバー数集計・しきい値評価

> **システム処理 SYS-001 のイベント「メンバー数集計・しきい値評価」を定義します。** 対応システム・対応業務UC・トリガー・呼出API・連携先と処理内容を記述します。

*種別 システムイベント設計 ・ ステータス ドラフト*

## 項目

| 項目 | 内容 |
|---|---|
| システムイベントID | `SEV-001` |
| イベント名 | メンバー数集計・しきい値評価 |
| 対応システムID | [SYS-001](../01_system/SYS-001.md#SYS-001) |
| 対応業務UC | [UC-049](../../../01_requirements/04_business_usecases/UC-049.md#UC-049) |
| トリガー | メンバー数の変動契機 / 定期集計スケジュール(日次) |
| 呼出API | — |
| 連携先 | 後段イベント(アラート通知生成・記録) |

## 処理

処理内容の正本は対応システム [SYS-001](../01_system/SYS-001.md#SYS-001) の §6 システムイベント一覧 を参照(二重管理を避けるため本ページには再掲しない)。本イベントは契約ごとの最新メンバー数を集計し、しきい値への接近と短期間での急増を評価して、検知時に後段の通知生成イベントへ引き渡す役割を担う。
