# <span id="SEV-024"></span>SEV-024: お知らせ閲覧範囲の本人限定

> **システム処理 SYS-013 のイベント「お知らせ閲覧範囲の本人限定」を定義します。** 対応システム・対応業務UC・トリガー・呼出API・連携先と処理内容を記述します。

*種別 システムイベント設計 ・ ステータス ドラフト*

## 項目

| 項目 | 内容 |
|---|---|
| システムイベントID | `SEV-024` |
| イベント名 | お知らせ閲覧範囲の本人限定 |
| 対応システムID | [SYS-013](../01_system/SYS-013.md#SYS-013) |
| 対応業務UC | [UC-086](../../../01_requirements/04_business_usecases/UC-086.md#UC-086) |
| トリガー | お知らせ表示のアクセス受信 |
| 呼出API | [API-048](../03_apis/API-048.md#API-048) |
| 連携先 | 後段処理(本人宛お知らせの表示 / 識別できない閲覧者への非公開) |

## 処理

処理内容の正本は対応システム [SYS-013](../01_system/SYS-013.md#SYS-013) の §6 システムイベント一覧 を参照(二重管理を避けるため本ページには再掲しない)。本イベントは、お知らせの表示要求の閲覧者がアカウント利用者本人かを確認し、本人宛のお知らせのみを抽出して表示する役割を担う。本人として識別できない閲覧者には一切公開しない。
