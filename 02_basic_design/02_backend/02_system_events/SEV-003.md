# <span id="SEV-003"></span>SEV-003: プロジェクト数集計・急増判定

> **システム処理 SYS-002 のイベント「プロジェクト数集計・急増判定」を定義します。** 対応システム・対応業務UC・トリガー・呼出API・連携先と処理内容を記述します。

*種別 システムイベント設計 ・ ステータス ドラフト*

## 項目

| 項目 | 内容 |
|---|---|
| システムイベントID | `SEV-003` |
| イベント名 | プロジェクト数集計・急増判定 |
| 対応システムID | [SYS-002](../01_system/SYS-002.md#SYS-002) |
| 対応業務UC | [UC-050](../../../01_requirements/04_business_usecases/UC-050.md#UC-050) |
| トリガー | プロジェクトの新規作成、または定期監視スケジュール |
| 呼出API | — |
| 連携先 | 後段イベント([SEV-004](SEV-004.md#SEV-004) オーナー通知) |

## 処理

処理内容の正本は対応システム [SYS-002](../01_system/SYS-002.md#SYS-002) の §6 システムイベント一覧 を参照(二重管理を避けるため本ページには再掲しない)。本イベントは対象契約のプロジェクト数を集計し、想定外の急増に該当するかを判定して、該当時に後段の通知イベントへ引き継ぐ役割を担う。
