# <span id="SEV-050"></span>SEV-050: 通知メール送信・配信記録

> **システム処理 SYS-026 のイベント「通知メール送信・配信記録」を定義します。** 対応システム・対応業務UC・トリガー・呼出API・連携先と処理内容を記述します。

*種別 システムイベント設計 ・ ステータス ドラフト*

## 項目

| 項目 | 内容 |
|---|---|
| システムイベントID | `SEV-050` |
| イベント名 | 通知メール送信・配信記録 |
| 対応システムID | [SYS-026](../01_system/SYS-026.md#SYS-026) |
| 対応業務UC | [UC-066](../../../01_requirements/04_business_usecases/UC-066.md#UC-066) |
| トリガー | 受信箱お知らせ生成後、通知宛先を解決できたとき |
| 呼出API | [API-058](../03_apis/API-058.md#API-058) |
| 連携先 | メール配信記録 |

## 処理

処理内容の正本は対応システム [SYS-026](../01_system/SYS-026.md#SYS-026) の §6 システムイベント一覧 を参照(二重管理を避けるため本ページには再掲しない)。本イベントは対象メンバーへ割当変更通知メールを送信し、その送信結果を配信記録として残す役割を担う。宛先を解決できない場合は本イベントを実行しない。
