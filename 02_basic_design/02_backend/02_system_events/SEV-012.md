# <span id="SEV-012"></span>SEV-012: 取込失敗

> **システム処理 SYS-006 のイベント「取込失敗」を定義します。** 対応システム・対応業務UC・トリガー・呼出API・連携先と処理内容を記述します。

*種別 システムイベント設計 ・ ステータス ドラフト*

## 項目

| 項目 | 内容 |
|---|---|
| システムイベントID | `SEV-012` |
| イベント名 | 取込失敗 |
| 対応システムID | [SYS-006](../01_system/SYS-006.md#SYS-006) |
| 対応業務UC | [UC-061](../../../01_requirements/04_business_usecases/UC-061.md#UC-061) |
| トリガー | 受信した通知の契約・決済状態への反映失敗 |
| 呼出API | — |
| 連携先 | 取込失敗の運用通知 / 再処理対象としての記録 |

## 処理

処理内容の正本は対応システム [SYS-006](../01_system/SYS-006.md#SYS-006) の §6 システムイベント一覧 を参照(二重管理を避けるため本ページには再掲しない)。本イベントは、取込に失敗した通知を失敗として記録・通知し、後から再処理できる状態を確保する役割を担う。
