# <span id="SEV-016"></span>SEV-016: 認証関連通知の送信

> **システム処理 SYS-008 のイベント「認証関連通知の送信」を定義します。** 対応システム・対応業務UC・トリガー・呼出API・連携先と処理内容を記述します。

*種別 システムイベント設計 ・ ステータス ドラフト*

## 項目

| 項目 | 内容 |
|---|---|
| システムイベントID | `SEV-016` |
| イベント名 | 認証関連通知の送信 |
| 対応システムID | [SYS-008](../01_system/SYS-008.md#SYS-008) |
| 対応業務UC | [UC-069](../../../01_requirements/04_business_usecases/UC-069.md#UC-069) |
| トリガー | オプトアウト不可判定で送信対象とされたこと |
| 呼出API | [API-058](../03_apis/API-058.md#API-058) |
| 連携先 | メール配信 IF・受信箱・通知ログ |

## 処理

処理内容の正本は対応システム [SYS-008](../01_system/SYS-008.md#SYS-008) の §6 システムイベント一覧 を参照(二重管理を避けるため本ページには再掲しない)。本イベントは、配信用の連携先を介して対象のアカウント利用者へ認証関連通知を送信し、本人の受信箱へ配置したうえで送信結果(成否)を記録する役割を担う。一時的な送信障害時は再送して確実な到達を図る。
