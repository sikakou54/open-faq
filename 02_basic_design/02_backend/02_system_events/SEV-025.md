# <span id="SEV-025"></span>SEV-025: お知らせ受信箱の利用者別保持

> **システム処理 SYS-014 のイベント「お知らせ受信箱の利用者別保持」を定義します。** 対応システム・対応業務UC・トリガー・呼出API・連携先と処理内容を記述します。

*種別 システムイベント設計 ・ ステータス ドラフト*

## 項目

| 項目 | 内容 |
|---|---|
| システムイベントID | `SEV-025` |
| イベント名 | お知らせ受信箱の利用者別保持 |
| 対応システムID | [SYS-014](../01_system/SYS-014.md#SYS-014) |
| 対応業務UC | [UC-087](../../../01_requirements/04_business_usecases/UC-087.md#UC-087) |
| トリガー | お知らせ配信の発生 |
| 呼出API | — |
| 連携先 | 後段処理(本人限定での受信箱保持) |

## 処理

処理内容の正本は対応システム [SYS-014](../01_system/SYS-014.md#SYS-014) の §6 システムイベント一覧 を参照(二重管理を避けるため本ページには再掲しない)。本イベントは、お知らせ配信時に対象となるアカウント利用者ごとに受信者と受信箱を生成し、各受信箱の閲覧を本人に限定して本人専用の受信状態として保持する役割を担う。
