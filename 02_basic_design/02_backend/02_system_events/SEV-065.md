# <span id="SEV-065"></span>SEV-065: 保持期間超過データの論理削除

> **システム処理 SYS-034 のイベント「保持期間超過データの論理削除」を定義します。** 対応システム・対応業務UC・トリガー・呼出API・連携先と処理内容を記述します。

*種別 システムイベント設計 ・ ステータス ドラフト*

## 項目

| 項目 | 内容 |
|---|---|
| システムイベントID | `SEV-065` |
| イベント名 | 保持期間超過データの論理削除 |
| 対応システムID | [SYS-034](../01_system/SYS-034.md#SYS-034) |
| 対応業務UC | — |
| トリガー | 日次の定期処理として保持期間超過データを抽出した契機 |
| 呼出API | — |
| 連携先 | 後段イベント(削除実行結果の記録) |

## 処理

処理内容の正本は対応システム [SYS-034](../01_system/SYS-034.md#SYS-034) の §6 システムイベント一覧 を参照(二重管理を避けるため本ページには再掲しない)。本イベントは保持期間(1 年)を超過した質問ログ・未解決質問・通知ログ・お知らせ受信箱を論理削除する役割を担う。
