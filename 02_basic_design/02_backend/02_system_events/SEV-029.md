# <span id="SEV-029"></span>SEV-029: 行単位取り込み

> **システム処理 SYS-016 のイベント「行単位取り込み」を定義します。** 対応システム・対応業務UC・トリガー・呼出API・連携先と処理内容を記述します。

*種別 システムイベント設計 ・ ステータス ドラフト*

## 項目

| 項目 | 内容 |
|---|---|
| システムイベントID | `SEV-029` |
| イベント名 | 行単位取り込み |
| 対応システムID | [SYS-016](../01_system/SYS-016.md#SYS-016) |
| 対応業務UC | [UC-051](../../../01_requirements/04_business_usecases/UC-051.md#UC-051) |
| トリガー | FAQ 一括取り込み依頼の受付による取り込みジョブ起動 |
| 呼出API | [API-028](../03_apis/API-028.md#API-028) |
| 連携先 | 後段イベント(完了通知) |

## 処理

処理内容の正本は対応システム [SYS-016](../01_system/SYS-016.md#SYS-016) の §6 システムイベント一覧 を参照(二重管理を避けるため本ページには再掲しない)。本イベントは取り込み対象を行ごとに読み込み、新規登録 / 既存 FAQ の上書き / 行失敗を判定して反映し、成功件数・失敗件数と失敗明細を集計して後段の完了通知イベントへ引き渡す役割を担う。
