# <span id="SEV-058"></span>SEV-058: 再ログイン誘導

> **システム処理 SYS-030 のイベント「再ログイン誘導」を定義します。** 対応システム・対応業務UC・トリガー・呼出API・連携先と処理内容を記述します。

*種別 システムイベント設計 ・ ステータス ドラフト*

## 項目

| 項目 | 内容 |
|---|---|
| システムイベントID | `SEV-058` |
| イベント名 | 再ログイン誘導 |
| 対応システムID | [SYS-030](../01_system/SYS-030.md#SYS-030) |
| 対応業務UC | [UC-072](../../../01_requirements/04_business_usecases/UC-072.md#UC-072) |
| トリガー | セッション無効化の完了 |
| 呼出API | — |
| 連携先 | アカウント利用者(再ログイン画面) |

## 処理

処理内容の正本は対応システム [SYS-030](../01_system/SYS-030.md#SYS-030) の §6 システムイベント一覧 を参照(二重管理を避けるため本ページには再掲しない)。本イベントは、セッション失効を明示するメッセージを返し、アカウント利用者を再ログインへ誘導する役割を担う。
