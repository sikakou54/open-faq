# <span id="SEV-045"></span>SEV-045: 受信箱お知らせ生成

> **システム処理 SYS-024 のイベント「受信箱お知らせ生成」を定義します。** 対応システム・対応業務UC・トリガー・呼出API・連携先と処理内容を記述します。

*種別 システムイベント設計 ・ ステータス ドラフト*

## 項目

| 項目 | 内容 |
|---|---|
| システムイベントID | `SEV-045` |
| イベント名 | 受信箱お知らせ生成 |
| 対応システムID | [SYS-024](../01_system/SYS-024.md#SYS-024) |
| 対応業務UC | [UC-064](../../../01_requirements/04_business_usecases/UC-064.md#UC-064) |
| トリガー | 配信予定日時の到来 / 運営による即時配信の指示(宛先解決後) |
| 呼出API | [API-048](../03_apis/API-048.md#API-048) |
| 連携先 | 後段イベント(お知らせメール送信・配信ログ記録) |

## 処理

処理内容の正本は対応システム [SYS-024](../01_system/SYS-024.md#SYS-024) の §6 システムイベント一覧 を参照(二重管理を避けるため本ページには再掲しない)。本イベントは宛先範囲から解決した各アカウント利用者の受信箱に運営お知らせを生成し、対象者が一覧でお知らせを確認できるようにする役割を担う。
