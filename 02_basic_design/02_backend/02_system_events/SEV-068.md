# <span id="SEV-068"></span>SEV-068: 再処理上限到達のエスカレーション

> **システム処理 SYS-035 のイベント「再処理上限到達のエスカレーション」を定義します。** 対応システム・対応業務UC・トリガー・呼出API・連携先と処理内容を記述します。

*種別 システムイベント設計 ・ ステータス ドラフト*

## 項目

| 項目 | 内容 |
|---|---|
| システムイベントID | `SEV-068` |
| イベント名 | 再処理上限到達のエスカレーション |
| 対応システムID | [SYS-035](../01_system/SYS-035.md#SYS-035) |
| 対応業務UC | [UC-061](../../../01_requirements/04_business_usecases/UC-061.md#UC-061) |
| トリガー | 再処理を繰り返しても成功せず上限回数に達したこと |
| 呼出API | — |
| 連携先 | 運用者への通知([MSG-013](../../06_messages/MSG-013.md#MSG-013)) |

## 処理

処理内容の正本は対応システム [SYS-035](../01_system/SYS-035.md#SYS-035) の §6 システムイベント一覧 を参照(二重管理を避けるため本ページには再掲しない)。本イベントは再処理を繰り返しても成功せず上限回数に達した通知を、運用者へエスカレーション通知する役割を担う。
