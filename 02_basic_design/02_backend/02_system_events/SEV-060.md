# <span id="SEV-060"></span>SEV-060: ロック解除

> **システム処理 SYS-031 のイベント「ロック解除」を定義します。** 対応システム・対応業務UC・トリガー・呼出API・連携先と処理内容を記述します。

*種別 システムイベント設計 ・ ステータス ドラフト*

## 項目

| 項目 | 内容 |
|---|---|
| システムイベントID | `SEV-060` |
| イベント名 | ロック解除 |
| 対応システムID | [SYS-031](../01_system/SYS-031.md#SYS-031) |
| 対応業務UC | [UC-073](../../../01_requirements/04_business_usecases/UC-073.md#UC-073) |
| トリガー | ロック期間の経過(自動解除) / 権限者による手動解除 |
| 呼出API | — |
| 連携先 | 権限者への解除完了通知、ログイン試行の再受付 |

## 処理

処理内容の正本は対応システム [SYS-031](../01_system/SYS-031.md#SYS-031) の §6 システムイベント一覧 を参照(二重管理を避けるため本ページには再掲しない)。本イベントは時間経過または権限者の操作でロックを解除し、連続失敗回数を初期化してログイン試行を再受付する役割を担う。手動解除時は権限者へ解除完了を通知する。
