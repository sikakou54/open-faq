# <span id="SEV-030"></span>SEV-030: 完了通知

> **システム処理 SYS-016 のイベント「完了通知」を定義します。** 対応システム・対応業務UC・トリガー・呼出API・連携先と処理内容を記述します。

*種別 システムイベント設計 ・ ステータス ドラフト*

## 項目

| 項目 | 内容 |
|---|---|
| システムイベントID | `SEV-030` |
| イベント名 | 完了通知 |
| 対応システムID | [SYS-016](../01_system/SYS-016.md#SYS-016) |
| 対応業務UC | [UC-051](../../../01_requirements/04_business_usecases/UC-051.md#UC-051) |
| トリガー | 全行の処理後の成功・失敗件数集計の完了 |
| 呼出API | — |
| 連携先 | 依頼したアカウント利用者(取り込み完了の通知先) |

## 処理

処理内容の正本は対応システム [SYS-016](../01_system/SYS-016.md#SYS-016) の §6 システムイベント一覧 を参照(二重管理を避けるため本ページには再掲しない)。本イベントは全行の処理後に集計した成功件数・失敗件数と部分失敗の明細を、依頼したアカウント利用者へ取り込み完了として知らせる役割を担う。
