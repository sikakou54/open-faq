# <span id="SEV-055"></span>SEV-055: 依存順物理削除

> **システム処理 SYS-029 のイベント「依存順物理削除」を定義します。** 対応システム・対応業務UC・トリガー・呼出API・連携先と処理内容を記述します。

*種別 システムイベント設計 ・ ステータス ドラフト*

## 項目

| 項目 | 内容 |
|---|---|
| システムイベントID | `SEV-055` |
| イベント名 | 依存順物理削除 |
| 対応システムID | [SYS-029](../01_system/SYS-029.md#SYS-029) |
| 対応業務UC | [UC-071](../../../01_requirements/04_business_usecases/UC-071.md#UC-071) |
| トリガー | 90 日経過の削除対象を抽出したとき |
| 呼出API | — |
| 連携先 | 後段イベント(監査記録) |

## 処理

処理内容の正本は対応システム [SYS-029](../01_system/SYS-029.md#SYS-029) の §6 システムイベント一覧 を参照(二重管理を避けるため本ページには再掲しない)。本イベントは、論理削除から 90 日を経過した対象を保持義務のあるデータを除いたうえで依存関係の順序に従って物理削除し、復元できない状態にする役割を担う。
