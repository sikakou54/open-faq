# <span id="SEV-039"></span>SEV-039: 請求確定・記録

> **システム処理 SYS-021 のイベント「請求確定・記録」を定義します。** 対応システム・対応業務UC・トリガー・呼出API・連携先と処理内容を記述します。

*種別 システムイベント設計 ・ ステータス ドラフト*

## 項目

| 項目 | 内容 |
|---|---|
| システムイベントID | `SEV-039` |
| イベント名 | 請求確定・記録 |
| 対応システムID | [SYS-021](../01_system/SYS-021.md#SYS-021) |
| 対応業務UC | [UC-059](../../../01_requirements/04_business_usecases/UC-059.md#UC-059) |
| トリガー | 月次請求確定処理での契約単位の請求金額確定 |
| 呼出API | [API-043](../03_apis/API-043.md#API-043) |
| 連携先 | 後段イベント(確定通知送信) |

## 処理

処理内容の正本は対応システム [SYS-021](../01_system/SYS-021.md#SYS-021) の §6 システムイベント一覧 を参照(二重管理を避けるため本ページには再掲しない)。本イベントは、プロジェクトごとの算定結果を契約単位へ集約した請求金額を、当該契約・当該月が未確定の場合に限り確定し記録する役割を担う。確定済み・対象なしの契約は確定を行わず冪等にスキップする。
