# <span id="SEV-042"></span>SEV-042: 通常状態復帰

> **システム処理 SYS-022 のイベント「通常状態復帰」を定義します。** 対応システム・対応業務UC・トリガー・呼出API・連携先と処理内容を記述します。

*種別 システムイベント設計 ・ ステータス ドラフト*

## 項目

| 項目 | 内容 |
|---|---|
| システムイベントID | `SEV-042` |
| イベント名 | 通常状態復帰 |
| 対応システムID | [SYS-022](../01_system/SYS-022.md#SYS-022) |
| 対応業務UC | [UC-060](../../../01_requirements/04_business_usecases/UC-060.md#UC-060) |
| トリガー | 再決済成功 / 解除の通知受信 |
| 呼出API | — |
| 連携先 | 契約状態の更新(通常状態への即時復帰) |

## 処理

処理内容の正本は対応システム [SYS-022](../01_system/SYS-022.md#SYS-022) の §6 システムイベント一覧 を参照(二重管理を避けるため本ページには再掲しない)。本イベントは猶予中・サスペンション中のいずれであっても、再決済成功または解除の通知を受けた契約を通常状態へ即時復帰させる役割を担う。
