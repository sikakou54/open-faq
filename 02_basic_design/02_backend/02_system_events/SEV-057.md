# <span id="SEV-057"></span>SEV-057: セッション無効化

> **システム処理 SYS-030 のイベント「セッション無効化」を定義します。** 対応システム・対応業務UC・トリガー・呼出API・連携先と処理内容を記述します。

*種別 システムイベント設計 ・ ステータス ドラフト*

## 項目

| 項目 | 内容 |
|---|---|
| システムイベントID | `SEV-057` |
| イベント名 | セッション無効化 |
| 対応システムID | [SYS-030](../01_system/SYS-030.md#SYS-030) |
| 対応業務UC | [UC-072](../../../01_requirements/04_business_usecases/UC-072.md#UC-072) |
| トリガー | 失効優先順位とタイムアウト(無操作・絶対)による失効判定の成立 |
| 呼出API | — |
| 連携先 | 後段イベント(再ログイン誘導) |

## 処理

処理内容の正本は対応システム [SYS-030](../01_system/SYS-030.md#SYS-030) の §6 システムイベント一覧 を参照(二重管理を避けるため本ページには再掲しない)。本イベントは、タイムアウトで失効に該当したセッションを無効化し、以降の操作で改めて認証が求められる状態へ遷移させる役割を担う。
