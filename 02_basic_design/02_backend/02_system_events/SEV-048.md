# <span id="SEV-048"></span>SEV-048: 通知メール送信・配信ログ記録

> **システム処理 SYS-025 のイベント「通知メール送信・配信ログ記録」を定義します。** 対応システム・対応業務UC・トリガー・呼出API・連携先と処理内容を記述します。

*種別 システムイベント設計 ・ ステータス ドラフト*

## 項目

| 項目 | 内容 |
|---|---|
| システムイベントID | `SEV-048` |
| イベント名 | 通知メール送信・配信ログ記録 |
| 対応システムID | [SYS-025](../01_system/SYS-025.md#SYS-025) |
| 対応業務UC | [UC-065](../../../01_requirements/04_business_usecases/UC-065.md#UC-065) |
| トリガー | システム通知生成後のメール対象契機の判定 |
| 呼出API | [API-058](../03_apis/API-058.md#API-058) |
| 連携先 | 配信ログ(送信結果の記録) |

## 処理

処理内容の正本は対応システム [SYS-025](../01_system/SYS-025.md#SYS-025) の §6 システムイベント一覧 を参照(二重管理を避けるため本ページには再掲しない)。本イベントは、メール対象の契機で対象アカウント利用者へ通知メールを送信し、送信結果を配信ログとして記録する役割を担う。
