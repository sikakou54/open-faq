# <span id="SEV-062"></span>SEV-062: 再認証・停止時制限適用

> **システム処理 SYS-032 のイベント「再認証・停止時制限適用」を定義します。** 対応システム・対応業務UC・トリガー・呼出API・連携先と処理内容を記述します。

*種別 システムイベント設計 ・ ステータス ドラフト*

## 項目

| 項目 | 内容 |
|---|---|
| システムイベントID | `SEV-062` |
| イベント名 | 再認証・停止時制限適用 |
| 対応システムID | [SYS-032](../01_system/SYS-032.md#SYS-032) |
| 対応業務UC | [UC-074](../../../01_requirements/04_business_usecases/UC-074.md#UC-074) |
| トリガー | 一斉無効化後に配下利用者の次回操作が到達した時 |
| 呼出API | [API-003](../03_apis/API-003.md#API-003) |
| 連携先 | — |

## 処理

処理内容の正本は対応システム [SYS-032](../01_system/SYS-032.md#SYS-032) の §6 システムイベント一覧 を参照(二重管理を避けるため本ページには再掲しない)。本イベントは無効化後の次回操作に対して再認証を要求し、再認証後の操作に停止時のアクセス制限を適用する役割を担う。
