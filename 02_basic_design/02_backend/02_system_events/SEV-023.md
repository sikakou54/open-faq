# <span id="SEV-023"></span>SEV-023: 利用根拠を失ったメンバーの利用停止

> **システム処理 SYS-012 のイベント「利用根拠を失ったメンバーの利用停止」を定義します。** 対応システム・対応業務UC・トリガー・呼出API・連携先と処理内容を記述します。

*種別 システムイベント設計 ・ ステータス ドラフト*

## 項目

| 項目 | 内容 |
|---|---|
| システムイベントID | `SEV-023` |
| イベント名 | 利用根拠を失ったメンバーの利用停止 |
| 対応システムID | [SYS-012](../01_system/SYS-012.md#SYS-012) |
| 対応業務UC | [UC-079](../../../01_requirements/04_business_usecases/UC-079.md#UC-079) |
| トリガー | 前段イベント(メンバー割当の一括解除)の完了 |
| 呼出API | — |
| 連携先 | 利用根拠を失ったメンバーの関連データの消去対象化(後段の保持期間経過後の消去) |

## 処理

処理内容の正本は対応システム [SYS-012](../01_system/SYS-012.md#SYS-012) の §6 システムイベント一覧 を参照(二重管理を避けるため本ページには再掲しない)。本イベントは、割当解除後に他のプロジェクトへ有効な割当を持たないメンバー(オーナーを除く)を利用できない状態にし、その関連データを所定の保持期間の経過後に消去対象として扱う役割を担う。
