# <span id="SEV-033"></span>SEV-033: 利用量集計・閾値判定

> **システム処理 SYS-018 のイベント「利用量集計・閾値判定」を定義します。** 対応システム・対応業務UC・トリガー・呼出API・連携先と処理内容を記述します。

*種別 システムイベント設計 ・ ステータス ドラフト*

## 項目

| 項目 | 内容 |
|---|---|
| システムイベントID | `SEV-033` |
| イベント名 | 利用量集計・閾値判定 |
| 対応システムID | [SYS-018](../01_system/SYS-018.md#SYS-018) |
| 対応業務UC | [UC-056](../../../01_requirements/04_business_usecases/UC-056.md#UC-056) |
| トリガー | ウィジェットへの質問が対象プロジェクトに到達した時 |
| 呼出API | [API-040](../03_apis/API-040.md#API-040) |
| 連携先 | 後段イベント(利用量サマリ反映)・アラート通知 / 受付停止を担当する業務処理 |

## 処理

処理内容の正本は対応システム [SYS-018](../01_system/SYS-018.md#SYS-018) の §6 システムイベント一覧 を参照(二重管理を避けるため本ページには再掲しない)。本イベントは質問到達を契機に当月の質問数を集計し、上限オンのプロジェクトでは利用率を算出してアラート閾値・上限到達の各契機を判定し、検知時に通知 / 受付停止を担当する業務処理へ引き渡す役割を担う。
