# <span id="SEV-026"></span>SEV-026: 退会・無効化確定時の受信箱削除

> **システム処理 SYS-014 のイベント「退会・無効化確定時の受信箱削除」を定義します。** 対応システム・対応業務UC・トリガー・呼出API・連携先と処理内容を記述します。

*種別 システムイベント設計 ・ ステータス ドラフト*

## 項目

| 項目 | 内容 |
|---|---|
| システムイベントID | `SEV-026` |
| イベント名 | 退会・無効化確定時の受信箱削除 |
| 対応システムID | [SYS-014](../01_system/SYS-014.md#SYS-014) |
| 対応業務UC | [UC-087](../../../01_requirements/04_business_usecases/UC-087.md#UC-087) |
| トリガー | 退会申請・アカウント無効化の確定 |
| 呼出API | [API-056](../03_apis/API-056.md#API-056) |
| 連携先 | 後段処理(受信者・受信箱の削除と削除完了確認) |

## 処理

処理内容の正本は対応システム [SYS-014](../01_system/SYS-014.md#SYS-014) の §6 システムイベント一覧 を参照(二重管理を避けるため本ページには再掲しない)。本イベントは、退会・無効化の確定を契機に対象利用者を特定し、当該利用者の受信箱と受信者を削除して、孤立した個人データを残さない状態へ収束させる役割を担う。一時的な障害で削除が完了しなかった場合は再度削除を試みる。
