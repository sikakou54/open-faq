# <span id="SEV-052"></span>SEV-052: 配信状態更新

> **システム処理 SYS-027 のイベント「配信状態更新」を定義します。** 対応システム・対応業務UC・トリガー・呼出API・連携先と処理内容を記述します。

*種別 システムイベント設計 ・ ステータス ドラフト*

## 項目

| 項目 | 内容 |
|---|---|
| システムイベントID | `SEV-052` |
| イベント名 | 配信状態更新 |
| 対応システムID | [SYS-027](../01_system/SYS-027.md#SYS-027) |
| 対応業務UC | [UC-067](../../../01_requirements/04_business_usecases/UC-067.md#UC-067) |
| トリガー | 通知再送の結果確定 / 再送上限到達・抑制該当の検知 |
| 呼出API | — |
| 連携先 | 通知配信ログ(配信状態・再送回数の記録) |

## 処理

処理内容の正本は対応システム [SYS-027](../01_system/SYS-027.md#SYS-027) の §6 システムイベント一覧 を参照(二重管理を避けるため本ページには再掲しない)。本イベントは再送結果に応じて配信状態と再送回数を更新し、再送上限到達分・抑制該当分を確定失敗として記録する役割を担う。
