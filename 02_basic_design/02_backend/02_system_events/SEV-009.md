# <span id="SEV-009"></span>SEV-009: 未解決質問の記録

> **システム処理 SYS-005 のイベント「未解決質問の記録」を定義します。** 対応システム・対応業務UC・トリガー・呼出API・連携先と処理内容を記述します。

*種別 システムイベント設計 ・ ステータス ドラフト*

## 項目

| 項目 | 内容 |
|---|---|
| システムイベントID | `SEV-009` |
| イベント名 | 未解決質問の記録 |
| 対応システムID | [SYS-005](../01_system/SYS-005.md#SYS-005) |
| 対応業務UC | [UC-055](../../../01_requirements/04_business_usecases/UC-055.md#UC-055) |
| トリガー | 未解決質問の登録契機(システムが回答できなかった、または利用者が未解決と申告した等)の発生 |
| 呼出API | [API-039](../03_apis/API-039.md#API-039) |
| 連携先 | 後段処理(FAQ 改善・サポート対応の追跡起点) |

## 処理

処理内容の正本は対応システム [SYS-005](../01_system/SYS-005.md#SYS-005) の §6 システムイベント一覧 を参照(二重管理を避けるため本ページには再掲しない)。本イベントは、質問内容・未解決理由・発生日時・関連プロジェクト・関連する質問のやり取り・追跡用識別子を未解決質問へ記録する役割を担う。
