# <span id="SEV-041"></span>SEV-041: サスペンション移行

> **システム処理 SYS-022 のイベント「サスペンション移行」を定義します。** 対応システム・対応業務UC・トリガー・呼出API・連携先と処理内容を記述します。

*種別 システムイベント設計 ・ ステータス ドラフト*

## 項目

| 項目 | 内容 |
|---|---|
| システムイベントID | `SEV-041` |
| イベント名 | サスペンション移行 |
| 対応システムID | [SYS-022](../01_system/SYS-022.md#SYS-022) |
| 対応業務UC | [UC-060](../../../01_requirements/04_business_usecases/UC-060.md#UC-060) |
| トリガー | 猶予経過を判定する定期処理の起動(猶予経過かつ再決済未成立) |
| 呼出API | — |
| 連携先 | 契約状態の更新(ウィジェット応答の機能停止・管理操作の限定) |

## 処理

処理内容の正本は対応システム [SYS-022](../01_system/SYS-022.md#SYS-022) の §6 システムイベント一覧 を参照(二重管理を避けるため本ページには再掲しない)。本イベントは定期処理の判定結果を受け、猶予期間を過ぎても再決済が成立していない契約をサスペンションへ移行する役割を担う。
