# <span id="SEV-035"></span>SEV-035: 上限アラートお知らせ生成

> **システム処理 SYS-019 のイベント「上限アラートお知らせ生成」を定義します。** 対応システム・対応業務UC・トリガー・呼出API・連携先と処理内容を記述します。

*種別 システムイベント設計 ・ ステータス ドラフト*

## 項目

| 項目 | 内容 |
|---|---|
| システムイベントID | `SEV-035` |
| イベント名 | 上限アラートお知らせ生成 |
| 対応システムID | [SYS-019](../01_system/SYS-019.md#SYS-019) |
| 対応業務UC | [UC-057](../../../01_requirements/04_business_usecases/UC-057.md#UC-057) |
| トリガー | 当月質問数が設定済みアラート閾値へ当月初めて到達したことの検知 |
| 呼出API | [API-046](../03_apis/API-046.md#API-046) |
| 連携先 | 後段イベント(アラートメール送信・結果記録) |

## 処理

処理内容の正本は対応システム [SYS-019](../01_system/SYS-019.md#SYS-019) の §6 システムイベント一覧 を参照(二重管理を避けるため本ページには再掲しない)。本イベントは当月初回到達を判定したうえで、オーナーと当該プロジェクトの有効なメンバーを宛先として特定・重複排除し、各宛先の受信箱に上限アラートのお知らせを生成する役割を担う。
