# <span id="SEV-007"></span>SEV-007: 未解決質問登録

> **システム処理 SYS-004 のイベント「未解決質問登録」を定義します。** 対応システム・対応業務UC・トリガー・呼出API・連携先と処理内容を記述します。

*種別 システムイベント設計 ・ ステータス ドラフト*

## 項目

| 項目 | 内容 |
|---|---|
| システムイベントID | `SEV-007` |
| イベント名 | 未解決質問登録 |
| 対応システムID | [SYS-004](../01_system/SYS-004.md#SYS-004) |
| 対応業務UC | [UC-054](../../../01_requirements/04_business_usecases/UC-054.md#UC-054) |
| トリガー | 登録済み FAQ で回答できないと判定された質問の登録要求 |
| 呼出API | [API-039](../03_apis/API-039.md#API-039) |
| 連携先 | 未解決質問・質問ログの記録(後段の回答不可案内へ引き継ぐ) |

## 処理

処理内容の正本は対応システム [SYS-004](../01_system/SYS-004.md#SYS-004) の §6 システムイベント一覧 を参照(二重管理を避けるため本ページには再掲しない)。本イベントは、回答できなかった質問を追跡用の管理識別子とともに未解決質問として記録し、後続の改善追跡の起点を作る役割を担う。
