# <span id="SEV-038"></span>SEV-038: 定型文応答

> **システム処理 SYS-020 のイベント「定型文応答」を定義します。** 対応システム・対応業務UC・トリガー・呼出API・連携先と処理内容を記述します。

*種別 システムイベント設計 ・ ステータス ドラフト*

## 項目

| 項目 | 内容 |
|---|---|
| システムイベントID | `SEV-038` |
| イベント名 | 定型文応答 |
| 対応システムID | [SYS-020](../01_system/SYS-020.md#SYS-020) |
| 対応業務UC | [UC-058](../../../01_requirements/04_business_usecases/UC-058.md#UC-058) |
| トリガー | 受付停止後にウィジェットへ応答を返すとき |
| 呼出API | [API-046](../03_apis/API-046.md#API-046) |
| 連携先 | エンドユーザー向け FAQ ウィジェット |

## 処理

処理内容の正本は対応システム [SYS-020](../01_system/SYS-020.md#SYS-020) の §6 システムイベント一覧 を参照(二重管理を避けるため本ページには再掲しない)。本イベントは受付停止時に、問い合わせ先を案内する安全な定型文をウィジェット利用者へ返し、これ以上の質問入力・送信を受け付けない状態とする役割を担う。
