# <span id="SEV-021"></span>SEV-021: 外部露出箇所のサニタイズ完了

> **システム処理 SYS-011 のイベント「外部露出箇所のサニタイズ完了」を定義します。** 対応システム・対応業務UC・トリガー・呼出API・連携先と処理内容を記述します。

*種別 システムイベント設計 ・ ステータス ドラフト*

## 項目

| 項目 | 内容 |
|---|---|
| システムイベントID | `SEV-021` |
| イベント名 | 外部露出箇所のサニタイズ完了 |
| 対応システムID | [SYS-011](../01_system/SYS-011.md#SYS-011) |
| 対応業務UC | [UC-077](../../../01_requirements/04_business_usecases/UC-077.md#UC-077) |
| トリガー | ウィジェット利用者の入力を含む通知の生成時、外部露出箇所の特定後 |
| 呼出API | [API-058](../03_apis/API-058.md#API-058) |
| 連携先 | メール配信IF(安全な値で構成した通知の配信を委ねる) |

## 処理

処理内容の正本は対応システム [SYS-011](../01_system/SYS-011.md#SYS-011) の §6 システムイベント一覧 を参照(二重管理を避けるため本ページには再掲しない)。本イベントは、外部露出箇所を安全な固定値・サニタイズ済みの値で構成し終えた通知を、配信処理へ引き渡す役割を担う。
