# <span id="SEV-061"></span>SEV-061: セッション一斉無効化

> **システム処理 SYS-032 のイベント「セッション一斉無効化」を定義します。** 対応システム・対応業務UC・トリガー・呼出API・連携先と処理内容を記述します。

*種別 システムイベント設計 ・ ステータス ドラフト*

## 項目

| 項目 | 内容 |
|---|---|
| システムイベントID | `SEV-061` |
| イベント名 | セッション一斉無効化 |
| 対応システムID | [SYS-032](../01_system/SYS-032.md#SYS-032) |
| 対応業務UC | [UC-074](../../../01_requirements/04_business_usecases/UC-074.md#UC-074) |
| トリガー | 対象契約が停止(手動停止 / 規約違反停止)状態へ遷移した時 |
| 呼出API | — |
| 連携先 | 後段イベント(再認証・停止時制限適用) |

## 処理

処理内容の正本は対応システム [SYS-032](../01_system/SYS-032.md#SYS-032) の §6 システムイベント一覧 を参照(二重管理を避けるため本ページには再掲しない)。本イベントは停止された契約に属する配下利用者の有効なセッションを特定し、特定した全セッションを速やかに一斉無効化する中核の役割を担う。
