# <span id="SEV-015"></span>SEV-015: オプトアウト不可判定

> **システム処理 SYS-008 のイベント「オプトアウト不可判定」を定義します。** 対応システム・対応業務UC・トリガー・呼出API・連携先と処理内容を記述します。

*種別 システムイベント設計 ・ ステータス ドラフト*

## 項目

| 項目 | 内容 |
|---|---|
| システムイベントID | `SEV-015` |
| イベント名 | オプトアウト不可判定 |
| 対応システムID | [SYS-008](../01_system/SYS-008.md#SYS-008) |
| 対応業務UC | [UC-069](../../../01_requirements/04_business_usecases/UC-069.md#UC-069) |
| トリガー | 認証関連通知の契機(メール確認・パスワード再設定・ロックアウト等)の発生 |
| 呼出API | — |
| 連携先 | 後段処理(認証関連通知の送信) |

## 処理

処理内容の正本は対応システム [SYS-008](../01_system/SYS-008.md#SYS-008) の §6 システムイベント一覧 を参照(二重管理を避けるため本ページには再掲しない)。本イベントは、認証関連通知を受信拒否(オプトアウト)の対象外と判定し、利用者の受信拒否設定にかかわらず送信対象とする役割を担う。
