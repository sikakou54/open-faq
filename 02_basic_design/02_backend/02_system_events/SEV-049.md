# <span id="SEV-049"></span>SEV-049: 受信箱お知らせ生成

## 処理

処理内容の正本は対応システム [SYS-026](../01_system/SYS-026.md#SYS-026) の §6 システムイベント一覧 を参照(二重管理を避けるため本ページには再掲しない)。本イベントは対象メンバーと変更内容を特定し、対象メンバーの受信箱に運営お知らせ種別・通常重要度の割当変更お知らせを生成する役割を担う。
