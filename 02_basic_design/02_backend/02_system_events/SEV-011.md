# <span id="SEV-011"></span>SEV-011: 取込完了

> **システム処理 SYS-006 のイベント「取込完了」を定義します。** 対応システム・対応業務UC・トリガー・呼出API・連携先と処理内容を記述します。

*種別 システムイベント設計 ・ ステータス ドラフト*

## 項目

| 項目 | 内容 |
|---|---|
| システムイベントID | `SEV-011` |
| イベント名 | 取込完了 |
| 対応システムID | [SYS-006](../01_system/SYS-006.md#SYS-006) |
| 対応業務UC | [UC-061](../../../01_requirements/04_business_usecases/UC-061.md#UC-061) |
| トリガー | 受信検証・重複排除を通過した通知の契約・決済状態への反映成功 |
| 呼出API | — |
| 連携先 | 契約・サブスクリプション・請求書への状態反映 |

## 処理

処理内容の正本は対応システム [SYS-006](../01_system/SYS-006.md#SYS-006) の §6 システムイベント一覧 を参照(二重管理を避けるため本ページには再掲しない)。本イベントは、検証を通過した通知の内容をサービス内の契約状態へ正しく反映し、取込完了として記録する役割を担う。
