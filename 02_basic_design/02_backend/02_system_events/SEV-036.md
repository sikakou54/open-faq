# <span id="SEV-036"></span>SEV-036: アラートメール送信・結果記録

> **システム処理 SYS-019 のイベント「アラートメール送信・結果記録」を定義します。** 対応システム・対応業務UC・トリガー・呼出API・連携先と処理内容を記述します。

*種別 システムイベント設計 ・ ステータス ドラフト*

## 項目

| 項目 | 内容 |
|---|---|
| システムイベントID | `SEV-036` |
| イベント名 | アラートメール送信・結果記録 |
| 対応システムID | [SYS-019](../01_system/SYS-019.md#SYS-019) |
| 対応業務UC | [UC-057](../../../01_requirements/04_business_usecases/UC-057.md#UC-057) |
| トリガー | 上限アラートお知らせ生成の完了 |
| 呼出API | [API-058](../03_apis/API-058.md#API-058) |
| 連携先 | 通知再送(メール送信失敗時の再送対象) |

## 処理

処理内容の正本は対応システム [SYS-019](../01_system/SYS-019.md#SYS-019) の §6 システムイベント一覧 を参照(二重管理を避けるため本ページには再掲しない)。本イベントは各宛先へ上限アラートのメールを送信し、その送信結果を記録する役割を担う。メール送信が失敗した場合は受信箱のお知らせを生成済みのまま失敗を記録し、後続の通知再送の対象とする。
