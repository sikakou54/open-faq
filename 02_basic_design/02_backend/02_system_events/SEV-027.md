# <span id="SEV-027"></span>SEV-027: 遷移時の未読件数取得・反映

> **システム処理 SYS-015 のイベント「遷移時の未読件数取得・反映」を定義します。** 対応システム・対応業務UC・トリガー・呼出API・連携先と処理内容を記述します。

*種別 システムイベント設計 ・ ステータス ドラフト*

## 項目

| 項目 | 内容 |
|---|---|
| システムイベントID | `SEV-027` |
| イベント名 | 遷移時の未読件数取得・反映 |
| 対応システムID | [SYS-015](../01_system/SYS-015.md#SYS-015) |
| 対応業務UC | [UC-088](../../../01_requirements/04_business_usecases/UC-088.md#UC-088) |
| トリガー | 管理ダッシュボードへの遷移 |
| 呼出API | [API-051](../03_apis/API-051.md#API-051) |
| 連携先 | 未読件数の表示反映、後段イベント(滞在中の定期的な未読件数最新化) |

## 処理

処理内容の正本は対応システム [SYS-015](../01_system/SYS-015.md#SYS-015) の §6 システムイベント一覧 を参照(二重管理を避けるため本ページには再掲しない)。本イベントは、管理ダッシュボードへの遷移を契機に当人宛の未読お知らせ件数を取得し、未読なしを含めて表示へ反映する役割を担う。
