# <span id="SEV-054"></span>SEV-054: お知らせ新規生成

> **システム処理 SYS-028 のイベント「お知らせ新規生成」を定義します。** 対応システム・対応業務UC・トリガー・呼出API・連携先と処理内容を記述します。

*種別 システムイベント設計 ・ ステータス ドラフト*

## 項目

| 項目 | 内容 |
|---|---|
| システムイベントID | `SEV-054` |
| イベント名 | お知らせ新規生成 |
| 対応システムID | [SYS-028](../01_system/SYS-028.md#SYS-028) |
| 対応業務UC | [UC-068](../../../01_requirements/04_business_usecases/UC-068.md#UC-068) |
| トリガー | 受信箱お知らせの生成要求発生時に、同一契約・同一イベント種別で一定時間内の既存お知らせが無いとき |
| 呼出API | — |
| 連携先 | 対象アカウント利用者の受信箱(新規お知らせの反映) |

## 処理

処理内容の正本は対応システム [SYS-028](../01_system/SYS-028.md#SYS-028) の §6 システムイベント一覧 を参照(二重管理を避けるため本ページには再掲しない)。本イベントは、一定時間内に集約対象となる既存のお知らせが無い場合に新たにお知らせを生成し、対象アカウント利用者の受信箱へ反映する役割を担う。窓を外れた発生や異なるイベント種別も本イベントで別のお知らせとして生成される。
