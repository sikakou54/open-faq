# <span id="SEV-022"></span>SEV-022: メンバー割当の一括解除

> **システム処理 SYS-012 のイベント「メンバー割当の一括解除」を定義します。** 対応システム・対応業務UC・トリガー・呼出API・連携先と処理内容を記述します。

*種別 システムイベント設計 ・ ステータス ドラフト*

## 項目

| 項目 | 内容 |
|---|---|
| システムイベントID | `SEV-022` |
| イベント名 | メンバー割当の一括解除 |
| 対応システムID | [SYS-012](../01_system/SYS-012.md#SYS-012) |
| 対応業務UC | [UC-079](../../../01_requirements/04_business_usecases/UC-079.md#UC-079) |
| トリガー | プロジェクト削除の確定([API-018](../03_apis/API-018.md#API-018)) |
| 呼出API | [API-023](../03_apis/API-023.md#API-023) |
| 連携先 | 後段イベント(利用根拠を失ったメンバーの利用停止) |

## 処理

処理内容の正本は対応システム [SYS-012](../01_system/SYS-012.md#SYS-012) の §6 システムイベント一覧 を参照(二重管理を避けるため本ページには再掲しない)。本イベントは、プロジェクト削除の確定を契機に当該プロジェクトに紐づくメンバーの割当を一括で解除し、後段の利用根拠判定へつなげる役割を担う。
