# <span id="SEV-019"></span>SEV-019: 契約単位レート制限の評価

> **システム処理 SYS-010 のイベント「契約単位レート制限の評価」を定義します。** 対応システム・対応業務UC・トリガー・呼出API・連携先と処理内容を記述します。

*種別 システムイベント設計 ・ ステータス ドラフト*

## 項目

| 項目 | 内容 |
|---|---|
| システムイベントID | `SEV-019` |
| イベント名 | 契約単位レート制限の評価 |
| 対応システムID | [SYS-010](../01_system/SYS-010.md#SYS-010) |
| 対応業務UC | [UC-076](../../../01_requirements/04_business_usecases/UC-076.md#UC-076) |
| トリガー | 各機能へのリクエスト受信(全 API 横断・ゲートウェイ層) |
| 呼出API | — |
| 連携先 | 後段処理(リクエスト許可 / 上限超過リクエストの抑制) |

## 処理

処理内容の正本は対応システム [SYS-010](../01_system/SYS-010.md#SYS-010) の §6 システムイベント一覧 を参照(二重管理を避けるため本ページには再掲しない)。本イベントは、リクエスト元の契約・機能種別に適用する上限を取得し、契約単位で適用状況を評価して上限の範囲内か超過かを判定する役割を担う。
