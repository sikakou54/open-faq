# <span id="SEV-040"></span>SEV-040: 確定通知送信

> **システム処理 SYS-021 のイベント「確定通知送信」を定義します。** 対応システム・対応業務UC・トリガー・呼出API・連携先と処理内容を記述します。

*種別 システムイベント設計 ・ ステータス ドラフト*

## 項目

| 項目 | 内容 |
|---|---|
| システムイベントID | `SEV-040` |
| イベント名 | 確定通知送信 |
| 対応システムID | [SYS-021](../01_system/SYS-021.md#SYS-021) |
| 対応業務UC | [UC-059](../../../01_requirements/04_business_usecases/UC-059.md#UC-059) |
| トリガー | 請求確定の完了 |
| 呼出API | [API-058](../03_apis/API-058.md#API-058) |
| 連携先 | メール配信 IF / 受信箱お知らせ生成 |

## 処理

処理内容の正本は対応システム [SYS-021](../01_system/SYS-021.md#SYS-021) の §6 システムイベント一覧 を参照(二重管理を避けるため本ページには再掲しない)。本イベントは、確定した請求についてオーナーへ確定通知メールを送信し配信結果を記録するとともに、受信箱へ「請求確定」のお知らせ生成へ引き渡す役割を担う。配信に失敗した場合は請求自体は確定済みとし、失敗として記録する。
