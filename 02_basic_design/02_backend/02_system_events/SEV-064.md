# <span id="SEV-064"></span>SEV-064: 整合性違反通知

> **システム処理 SYS-033 のイベント「整合性違反通知」を定義します。** 対応システム・対応業務UC・トリガー・呼出API・連携先と処理内容を記述します。

*種別 システムイベント設計 ・ ステータス ドラフト*

## 項目

| 項目 | 内容 |
|---|---|
| システムイベントID | `SEV-064` |
| イベント名 | 整合性違反通知 |
| 対応システムID | [SYS-033](../01_system/SYS-033.md#SYS-033) |
| 対応業務UC | [UC-075](../../../01_requirements/04_business_usecases/UC-075.md#UC-075) |
| トリガー | 監査ログ整合性検証で整合性違反を検出 |
| 呼出API | — |
| 連携先 | 担当者(アラート通知) |

## 処理

処理内容の正本は対応システム [SYS-033](../01_system/SYS-033.md#SYS-033) の §6 システムイベント一覧 を参照(二重管理を避けるため本ページには再掲しない)。本イベントは整合性違反(改ざん・欠落の疑い)を検出した対象を特定し担当者へアラート通知する役割を担う。検証は中断せず残りの全件走査を継続する。
