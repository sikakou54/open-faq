# <span id="TBL-001"></span>M_USER(ユーザーマスタ)

オーナー・メンバーを含む**すべてのユーザー**の認証情報(メール / パスワード / 表示名)を一元保持するユーザーマスタです。オーナーかどうかの判定は、対象プロジェクトの `M_PROJECTS.owner_user_id` 参照で行い(立場はプロジェクトごとに決まる)、本テーブルには所有・課金属性を持ちません。アカウント状態の意味は [状態モデル §1](../../08_state-model.md#1-アカウント状態) を参照します。

|----|
| TBL-001 |
