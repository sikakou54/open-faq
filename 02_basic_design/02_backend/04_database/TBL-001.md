# <span id="TBL-001"></span>M_USER(ユーザーマスタ)

オーナー・メンバーを含む**すべてのユーザー**の認証情報(メール / パスワード / 表示名)を一元保持するユーザーマスタです。オーナーかどうかの判定は、対象プロジェクトの `M_PROJECTS.owner_user_id` 参照で行い(立場はプロジェクトごとに決まる)、本テーブルには所有・課金属性を持ちません。新規サインアップ直後は `status='pending_activation'`(メール確認待ち)とし、メール確認の完了で `active` に確定します。



| ID | 物理名 | 論理名 | 区分 |
|----|----|----|----|
| TBL-001 | `M_USER` | ユーザーマスタ | マスタ |
