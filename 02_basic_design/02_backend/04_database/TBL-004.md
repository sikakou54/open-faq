### <span id="settings"></span>`settings` JSON 構造(ウィジェット表示設定)

`settings`(JSON 文字列)はウィジェットの見た目・表示文言を保持する。画面 [SCR-011 ウィジェット設定](../../01_frontend/01_screens/SCR-011.md#SCR-011)(主色 IT-05 / 表示位置 IT-10 / 見出し IT-11 / 初期メッセージ IT-12)で設定し、[API-037 ウィジェット起動](../03_apis/API-037.md#API-037) の `projectConfig` へ整形して返却する。各キーは API 層でバリデーションする(下限・上限・列挙)。未設定キーは既定値を適用する。

| キー | 型 | 必須 | 既定値 | 制約 / 説明 | SCR-011 項目 | API-037 `projectConfig` 対応 |
|----|----|----|----|----|----|----|
| `headerColor` | string | 任意 | `"#5E6AD2"`(設計値) | ウィジェット主色。`#` + 6 桁 HEX(`^#[0-9A-Fa-f]{6}$`) | IT-05 主色 | `headerColor` |
| `placement` | string(enum) | 任意 | `"bottom-right"`(設計値) | 表示位置。`bottom-left`(左下)/ `bottom-right`(右下) | IT-10 表示位置 | `placement` |
| `headerTitle` | string | 任意 | `""`(未設定時はウィジェット既定文言) | ウィジェット展開時のヘッダー見出し。0〜60 文字(設計値・SCR-011 §4#9 を正本とする) | IT-11 見出し | `headerTitle` |
| `welcomeMessage` | string | 任意 | `""`(未設定時はウィジェット既定文言) | ウィジェットを開いた際にボットが最初に送る初期メッセージ。0〜200 文字(設計値・SCR-011 §4#10 を正本とする) | IT-12 初期メッセージ | (起動時にウィジェットへ反映) |

> [!NOTE]
> 連絡先メール(API-037 `projectConfig.contactEmail`)は `settings` JSON ではなく `contact_email` カラムから返却する。`settings` には公開キー(`widget_public_key` カラム)・連絡先(`contact_email` カラム)を重複保持しない。`placement` の enum 値(`bottom-left` / `bottom-right`)・文字数上限は API 層と SCR-011 の整合のための設計値であり、詳細設計で確定する。
