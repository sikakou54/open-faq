# <span id="TBL-018"></span>T_BILL_SUBS(課金サブスクリプション)

Stripe のサブスクリプションと連動する、課金アカウント(ユーザー)単位の課金サブスクリプション情報を保持します。

| ID |
|----|
| TBL-018 |

| No | 論理名 | 物理名 | データ型 | 桁数 | NULL | PK | FK | UNIQUE | DEFAULT | 制約 |
|---:|----|----|----|---:|----|----|----|----|----|----|
| 1 | ID | `id` | TEXT | \- | NO | ○ |  |  |  |  |

| 3 | Stripe サブスク ID | `stripe_sub_id` | TEXT | \- | YES |  |  | ○ |  |  |
| 4 | 状態 | `status` | TEXT | \- | NO |  |  |  |  | `status IN ('active','past_due','canceled','unpaid','incomplete')` |
| 5 | プラン ID | `plan_id` | TEXT | \- | NO |  |  |  |  | 課金プロバイダ側のプラン / 価格識別子を格納する。階層プランを提供しない MVP でも、従量課金に用いる単一プラン / 価格の識別子を保持するため NOT NULL(既定値なし)を維持する |

### <span id="インデックス"></span>インデックス

| No | インデックス名 | 対象カラム | UNIQUE | 用途 |
|---:|----|----|----|----|
| 1 | `idx_billing_subs_account` | `billing_account_id` |  | 課金アカウント検索 |
| 2 | `idx_billing_subs_status` | `status` |  | 状態別集計 |
| 3 | `idx_billing_subs_valid` | `valid` WHERE `valid = 0` |  | 論理削除済み参照(物理削除対象外) |
| 4 | `idx_billing_subs_grace_started_at` | `grace_started_at` WHERE `grace_started_at IS NOT NULL` |  | 猶予経過判定バッチの抽出 |
