# <span id="TBL-032"></span>T_BILLING_WEBHOOK_LOG(課金Webhook受信ログ)

トランザクション 課金 一定期間保持

課金プロバイダから受信した決済・契約状態通知を、検証結果・取込状態とともに記録します。重複受信の検出と取込失敗の再処理に用います。

### <span id="コード値区分値"></span>コード値・区分値

| カラム名 | 値 | 意味 |
|----------|------|------|
| `status` | `received` | 受信済み(未取込) |
| `status` | `processed` | 取込完了 |
| `status` | `failed` | 取込失敗(再処理対象) |
| `status` | `skipped` | 重複のため取込スキップ |
| `signature_valid` | `1` | 署名検証成功 |
| `signature_valid` | `0` | 署名検証失敗(不正受信) |
| `provider` | `stripe` | Stripe(課金プロバイダ。現状唯一) |

> [!NOTE]
> 課金プロバイダは現状 Stripe のみのため `provider` の許可値は `stripe` のみとする(設計値)。将来プロバイダを追加する場合は許可値を拡張する。
