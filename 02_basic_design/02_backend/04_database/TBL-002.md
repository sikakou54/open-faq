# <span id="TBL-002"></span>M_CONTRACT(契約マスタ)

マスタ 新規

**契約を管理する契約マスタ**です。`id` が契約境界キー本体であり、`M_PROJECTS` など契約スコープのテーブルはこの契約を親に持ちます。**オーナー判定**は `user_id` と一致するユーザーを当該契約のオーナーとみなします(1 契約 = 1 オーナー)。
