# <span id="TBL-023"></span>T_WITHDRAW_REQ(退会記録)

トランザクション

アカウント単位の即時退会の実行記録を保持します。退会日時・退会者(本人)・理由と、アカウント・請求データの削除予定日(退会日 + 7 年)を記録します。
