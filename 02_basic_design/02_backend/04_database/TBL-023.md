# <span id="TBL-023"></span>T_WITHDRAW_REQ(退会記録)

アカウント単位の即時退会の実行記録を保持します。退会日時・退会者(本人)・理由と、アカウント・請求データの削除予定日([システム仕様書 §4](../../07_system-spec.md#4-データ保持期間削除猶予) に基づく削除予定日)を記録します。
