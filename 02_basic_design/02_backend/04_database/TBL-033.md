### <span id="error_summary"></span>`error_summary` JSON 構造(失敗明細)

`error_summary`(JSON 文字列)は、取込に失敗した行の明細を保持する。画面のエラー一覧表示に用いる。

| 項目 | 規約 |
|----|----|
| 全体の型 | JSON オブジェクト。例 `{ "errors": [ { "row": 12, "code": "CSV_INVALID", "message": "..." } ] }` |
| `errors` | 配列。失敗行ごとの明細を要素に持つ |
| `errors[].row` | 整数。CSV の行番号(ヘッダ行を除く 1 始まり) |
| `errors[].code` | 文字列。失敗理由のエラーコード(例 `CSV_INVALID`) |
| `errors[].message` | 文字列。失敗理由の説明 |
| 最大件数 | 詳細設計で定義する(設計値) |
| 既定値 | `NULL`(失敗行なし) |
