# <span id="TBL-021"></span>T_ANNOUNCE_RCPT(お知らせ受信者)

お知らせの実配信先(受信者ユーザー)・配信集計・監査情報を保持します。受信者はアカウント利用者本人(`M_USER`)ごとに 1 件を保持します。



| ID | 物理名 | 論理名 | 区分 |
|----|----|----|----|
| TBL-021 | `T_ANNOUNCE_RCPT` | お知らせ受信者 | トランザクション |

### <span id="カラム定義"></span>カラム定義

| No | 論理名 | 物理名 | データ型 | 桁数 | NULL | PK | FK | UNIQUE | DEFAULT | 制約 |
|---:|----|----|----|---:|----|----|----|----|----|----|
| 1 | ID | `id` | TEXT | \- | NO | ○ |  |  |  |  |
| 2 | お知らせ ID | `announcement_id` | TEXT | \- | NO |  | `M_SERVICE_ANNOUNCE(id)` |  |  |  |
| 3 | 受信者ユーザー ID | `user_id` | TEXT | \- | NO |  | `M_USER(id)` |  |  | 受信者となるアカウント利用者本人 |
| 4 | 送信日時 | `sent_at` | TEXT | \- | NO |  |  |  |  |  |
| 5 | 配信状態 | `delivery_status` | TEXT | \- | NO |  |  |  |  | `delivery_status IN ('pending','delivered','failed')` |
| 6 | 既読日時 | `read_at` | TEXT | \- | YES |  |  |  |  |  |

### <span id="インデックス"></span>インデックス

| No | インデックス名 | 対象カラム | UNIQUE | 用途 |
|---:|----|----|----|----|
| 1 | `idx_ar_announcement` | `announcement_id` |  | お知らせ別 |
| 2 | `idx_ar_user` | `(user_id, announcement_id)` |  | ユーザー別逆引き |

### <span id="コード値区分値"></span>コード値・区分値

| カラム名          | 値          | 意味     |
|-------------------|-------------|----------|
| `delivery_status` | `pending`   | 配信待ち |
| `delivery_status` | `delivered` | 配信完了 |
| `delivery_status` | `failed`    | 配信失敗 |
