### <span id="カラム定義"></span>カラム定義

| No | 論理名 | 物理名 | データ型 | 桁数 | NULL | PK | FK | UNIQUE | DEFAULT | 制約 |

| 4 | 重要度 | `importance` | TEXT | \- | NO |  |  |  |  | `importance IN ('low','normal','high','critical')` |
