| ID | 物理名 | 論理名 | 区分 |
|----|----|----|----|
| TBL-010 | `M_SERVICE_ANNOUNCE` | お知らせ(Control Plane) | マスタ |

### <span id="カラム定義"></span>カラム定義

| No | 論理名 | 物理名 | データ型 | 桁数 | NULL | PK | FK | UNIQUE | DEFAULT | 制約 |
|---:|----|----|----|---:|----|----|----|----|----|----|
| 1 | ID | `id` | TEXT | \- | NO | ○ |  |  |  |  |
| 2 | タイトル | `title` | TEXT | \- | NO |  |  |  |  | `length(title) BETWEEN 1 AND 200` |
| 3 | 本文(HTML) | `body_html` | TEXT | \- | NO |  |  |  |  |  |
| 4 | 重要度 | `importance` | TEXT | \- | NO |  |  |  |  | `importance IN ('low','normal','high','critical')` |
| 5 | 公開日時 | `published_at` | TEXT | \- | NO |  |  |  |  |  |
| 6 | 取り下げ日時 | `retracted_at` | TEXT | \- | YES |  |  |  |  |  |
| 7 | 作成者種別 | `created_by_type` | TEXT | \- | YES |  |  |  |  | `created_by_type IN ('owner','project_user')` |
| 8 | 作成者 ID | `created_by_id` | TEXT | \- | YES |  |  |  |  | `created_by_type` に応じ `M_USER(id)` / `M_PRJ_USERS(id)` を指す(論理参照) |
| 9 | 作成日時 | `created_at` | TEXT | \- | NO |  |  |  |  |  |
| 10 | CTA 文言 | `action_label` | TEXT | \- | YES |  |  |  |  | 行動導線(CTA)ボタンの表示文言。配信時に受信箱 `T_INBOX_MSG.action_label` へ複写される供給元 |
| 11 | CTA 遷移先 | `action_url` | TEXT | \- | YES |  |  |  |  | 行動導線(CTA)ボタンの押下時遷移先 URL/画面参照。配信時に受信箱 `T_INBOX_MSG.action_url` へ複写される供給元 |
