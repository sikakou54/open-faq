## シーケンス図

```mermaid
sequenceDiagram
  actor U as 利用者
  participant S as 画面 SCR-029
  participant API as API
  participant DB as DB
  U->>S: 登録を完了するを押下(IT-10)
  S->>S: 全項目を再検証(EV-02〜EV-09)
  S->>API: POST /auth/invitations/{token}/activate(API-AUTH-008)
  alt 検証 OK ・ トークン有効
    API->>DB: M_USER(U 氏名・PWハッシュ・active) ・ M_PRJ_USERS(U valid=1) ・ T_TERMS_AGREE(C×2) ・ T_ACCESS_TOKENS(U 消費)
    DB-->>API: 更新結果(同一トランザクション)
    API-->>S: 200 有効化完了
    S-->>U: IT-13 完了画面を表示
  else 400 フィールドエラー
    API-->>S: 400
    S-->>U: フィールドエラー表示(フォーム維持)
  else 410 トークン無効 / 期限切れ
    API-->>S: 410
    S-->>U: IT-11 エラー画面を表示
  end
```
