## シーケンス図

```mermaid
sequenceDiagram
  actor U as ウィジェット利用者
  participant W as ウィジェット
  participant API as API
  participant DB as DB
  Note over W: EV-05 の応答が type=unanswered
  W-->>U: 回答できなかった旨をシステム返信で表示
  W->>API: POST /widget/v1/inquiries(API-WGT-003)
  API->>DB: T_INQUIRIES(CR) 質問ログ・未解決質問
  DB-->>API: 登録結果
  alt 成功
    API-->>W: 200 登録完了
    W-->>U: 連絡先メール(IT-08)を案内・問い合わせ ID は非表示
  else 失敗
    API-->>W: エラー
    W-->>U: エラー表示・再試行案内(連打防止)
  end
```
