## シーケンス図

```mermaid
sequenceDiagram
  actor U as 利用者
  participant S as 画面 SCR-028 FAQ 一覧
  participant API as API
  participant DB as DB
  U->>S: 画面を開く(初期表示)
  S->>API: GET /faqs(API-FAQ-001)
  API->>DB: M_FAQS(R)
  DB-->>API: 一覧データ
  API-->>S: 200 一覧(0 件もありうる)

  else 0 件

  end
```
