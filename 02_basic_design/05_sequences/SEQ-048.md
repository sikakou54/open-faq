## シーケンス図

```mermaid
sequenceDiagram
  actor U as 利用者
  participant S as モーダル SCR-024
  participant API as API
  participant DB as DB
  participant M as メール配信IF
  U->>S: 「招待メールを送信する」を押下(IT-10)
  S->>S: 入力バリデーション
  alt 検証エラー
    S-->>U: インラインエラー(中止)
  else 検証 OK
    S->>API: POST /projects/{id}/members(API-MBR-002)
    API->>DB: M_PRJ_USERS(C 予約割当・ユーザー) ・ T_ACCESS_TOKENS(C 活性化 7日)
    alt 成功
      DB-->>API: 作成結果
      API->>M: 招待メール送信を依頼
      M-->>API: 受付
      API-->>S: 200
      S-->>U: モーダルを閉じ一覧を更新
    else 重複(FR-036)/ 403 / エラー
      API-->>S: 4xx
      S-->>U: エラー表示(入力保持)
    end
  end
```
