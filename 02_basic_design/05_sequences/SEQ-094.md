## シーケンス図

```mermaid
sequenceDiagram
  autonumber
  participant SCH as スケジューラ / 配信イベント
  participant D as 配信処理
  participant DB as DB (D1)
  participant MAIL as メール配信 IF
  actor U as アカウント利用者
  SCH->>D: 配信予定日時到来 / 即時配信
  D->>DB: M_SERVICE_ANNOUNCE(R) 対象お知らせを取得
  DB-->>D: 宛先範囲・件名・本文・重要度
  D->>D: 宛先範囲(全契約 / 特定契約 / 特定プロジェクト)を解決
  alt 宛先あり
    D->>DB: T_INBOX_MSG(C) 「運営お知らせ」生成
    opt メール送信対象
      D->>MAIL: お知らせメールを送信(重要度に応じて)
      MAIL-->>U: お知らせメール
      D->>DB: H_NOTIF_LOGS(C) 配信ログ記録
    end
  else 宛先なし
    D-->>SCH: 正常終了(配信なし)
  end
```
