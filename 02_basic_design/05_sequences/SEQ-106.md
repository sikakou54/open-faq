## シーケンス図

```mermaid
sequenceDiagram
  autonumber
  participant E as お知らせ生成要求
  participant AG as 集約処理
  participant DB as DB (D1)
  E->>AG: 受信箱お知らせの生成を要求(契約 / イベント種別)
  AG->>DB: T_INBOX_MSG(R) 同一契約・同一種別の 60 分以内を照会
  DB-->>AG: 既存お知らせ(あり / なし)
  alt 60 分以内に既存あり
    AG->>DB: T_INBOX_MSG(U) 既存を更新(1 件へ集約)
  else 既存なし
    AG->>DB: T_INBOX_MSG(C) 新規お知らせを生成
  end
  DB-->>AG: 反映結果
```
