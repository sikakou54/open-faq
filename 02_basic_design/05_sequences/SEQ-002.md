## シーケンス図

```mermaid
sequenceDiagram
  autonumber
  actor U as 利用者
  participant S as 画面 SCR-001
  participant API as API /v1
  participant DB as DB (D1)
  participant TS as Turnstile(外部)
  U->>S: メール・パスワード入力 → ログイン
  S->>S: IT-01・IT-02 を再検証
  opt 連続失敗後
    S->>TS: Turnstile チャレンジ
    TS-->>S: トークン
  end
  S->>API: POST /auth/login
  API->>DB: M_CONTRACT(R) ・ M_PRJ_USERS(R) 認証
  DB-->>API: 照合結果・ロック / 停止状態
  alt 認証成功
    API->>DB: T_SESSIONS(C)
    DB-->>API: セッション確立
    API-->>S: 200 + Cookie + 種別・再同意要否
    S-->>U: 規約再同意 SCR-025 / オーナー SCR-026 / メンバー SCR-028 へ遷移
  else 認証失敗 / ロック / 停止
    API-->>S: 401 / 423 / 400 / 403
    S-->>U: 共通エラー(IT-06)/ ロックアウト警告(IT-07)/ Turnstile(IT-08)
  end
```
