## シーケンス図

```mermaid
sequenceDiagram
  autonumber
  participant OP as 契約管理(システム)
  participant SV as セッション無効化処理
  participant DB as DB (D1)
  participant U as 配下利用者
  OP->>SV: 停止 / 規約違反停止イベントを連携
  SV->>DB: M_CONTRACT(R) 停止状態を確認
  DB-->>SV: 契約状態(停止)
  SV->>DB: T_SESSIONS(D) 配下全セッションを速やかに無効化
  DB-->>SV: 反映結果
  Note over SV: 進行中の長時間処理は完了まで継続(中断しない)
  U->>SV: 次回操作
  SV-->>U: 再認証を要求 / 停止時アクセス制限を適用
```
