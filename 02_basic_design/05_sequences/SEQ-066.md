## シーケンス図

```mermaid
sequenceDiagram
  actor U as 利用者
  participant S as 画面 SCR-024 退会申請
  participant API as API
  participant DB as DB
  U->>S: 確認ダイアログの「OK」を押下
  S->>API: POST /auth/re-auth(再認証)(API-AUTH-005)
  alt 再認証失敗
    API-->>S: 認証エラー
    S-->>U: エラー表示・確認ダイアログへ戻る
  else 再認証成功
    API-->>S: 再認証 OK
    S->>API: POST /withdrawal-requests(API-TRM-005)
    alt 成功
      API->>DB: T_WITHDRAW_REQ(C) ・ M_CONTRACT(U status=deleted_pending)
      DB-->>API: 登録・更新結果
      API-->>S: 200 申請完了
      S-->>U: SCR-028 個人設定へ遷移
    else 申請エラー
      API-->>S: エラー
      S-->>U: エラー表示・確認ダイアログへ戻る
    end
  end
```
