## シーケンス図

```mermaid
sequenceDiagram
  actor U as 利用者
  participant S as モーダル SCR-028
  participant API as API
  participant DB as DB
  participant M as メール配信IF
  U->>S: 「保存」を押下(IT-10)
  S->>S: 全項目を検証
  alt バリデーションエラー
    S-->>U: インラインエラー(送信中断)
  else 検証 OK
    S->>API: PATCH /projects/{id}(版を添付)(API-PRJ-003)
    API->>DB: M_PROJECTS(U) ・ M_ALLOWED_DOMAINS(CUD)
    alt 成功
      DB-->>API: 更新結果
      opt 連絡先メール変更時
        API->>M: 確認メール送信を依頼
      end
      API-->>S: 200 更新完了
      S-->>U: モーダルを閉じ一覧を更新(IT-07 確認待ち)
    else 版衝突(楽観ロック)
      API-->>S: 版不一致
      S-->>U: 衝突を表示・再読み込みを促す
    else 認可エラー / その他
      API-->>S: 403 / エラー
      S-->>U: 権限不足 / トースト表示
    end
  end
```
