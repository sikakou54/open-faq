## シーケンス図

```mermaid
sequenceDiagram
  actor U as ウィジェット利用者
  participant W as ウィジェット
  participant API as API
  participant AI as AI推論IF
  participant DB as DB
  U->>W: 「送信」を押下(IT-06)
  W->>API: POST /widget/v1/ask(API-WGT-002)
  API->>AI: 質問・登録 FAQ で回答生成
  AI-->>API: 回答 / 未解決判定
  API->>DB: T_INQUIRIES(CR) 質問ログ
  DB-->>API: 記録結果
  alt 回答あり
    API-->>W: 200 AI 回答
    W-->>U: AI 回答(IT-07)を会話欄に追加
  else 未解決(type=unanswered)
    API-->>W: 200 未解決 → EV-06 へ
  else 受付制限(429)
    API-->>W: 429 → EV-07 へ
  else 処理エラー
    API-->>W: エラー → EV-08 へ
  end
```
