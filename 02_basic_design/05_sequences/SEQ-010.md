## シーケンス図

```mermaid
sequenceDiagram
  actor U as 利用者
  participant S as モーダル SCR-028
  participant API as API
  participant DB as DB
  U->>S: 編集モードで開く
  S->>API: GET /projects/{id}(API-PRJ-003)
  API->>DB: M_PROJECTS(R) ・ M_ALLOWED_DOMAINS(R)
  DB-->>API: プロジェクト現値
  alt 成功
    API-->>S: 200 現値
    S-->>U: 各欄へ展開・編集専用項目を表示
  else 失敗
    API-->>S: エラー
    S-->>U: トースト表示・モーダルを閉じる
  end
```
