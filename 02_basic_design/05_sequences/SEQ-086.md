## シーケンス図

```mermaid
sequenceDiagram
  actor U as 利用者
  participant S as 画面 SCR-029 設定
  participant API as API
  participant DB as DB
  U->>S: 「変更を保存」を押下(IT-02)
  S->>S: 入力を最終検証
  alt バリデーションエラー
    S-->>U: 該当項目にエラー表示(送信しない)
  else 検証 OK
    S->>API: 契約設定更新(API-AUTH-015)
    API->>DB: M_CONTRACT(U)
    alt 成功
      DB-->>API: 更新結果
      API-->>S: 200 更新完了
      S-->>U: 成功トースト・IT-01/IT-06 を再描画
    else API エラー
      API-->>S: エラー
      S-->>U: エラートースト(入力値を保持)
    end
  end
```
