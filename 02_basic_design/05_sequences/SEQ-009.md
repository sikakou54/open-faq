## シーケンス図

```mermaid
sequenceDiagram
  actor U as 利用者
  participant S as 画面 SCR-004 プロジェクト
  participant API as API
  participant DB as DB
  U->>S: 画面を開く(初期表示)
  S->>API: GET /projects(API-PRJ-001)
  API->>DB: M_PROJECTS(R) ・ M_ALLOWED_DOMAINS(R)
  DB-->>API: 一覧データ
  API-->>S: 200 一覧(0 件もありうる)
  alt 1 件以上
    S-->>U: 一覧と件数を表示
  else 0 件
    S-->>U: 空状態(IT-07)を表示
  end
```
