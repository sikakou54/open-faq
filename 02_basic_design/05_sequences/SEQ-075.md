## シーケンス図

```mermaid
sequenceDiagram
  actor U as 利用者
  participant S as 画面 SCR-024
  participant API as API
  participant DB as DB
  U->>S: 確認リンクから開く(初期表示)
  S->>API: POST /auth/contact-verifications/{token}(API-AUTH-009)

  alt 成功(200)
    S-->>U: IT-01 確認完了画面を表示
  else 期限切れ / 無効(410)

  else 使用済み(409)
    S-->>U: IT-03 既使用画面を表示
  end
```
