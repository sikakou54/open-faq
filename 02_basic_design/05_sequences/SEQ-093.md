## シーケンス図

```mermaid
sequenceDiagram
  autonumber
  participant SCH as スケジューラ
  participant B as 請求確定バッチ
  participant DB as DB (D1)
  participant MAIL as メール配信 IF
  actor O as オーナー
  SCH->>B: 月初起動(対象月)
  B->>DB: T_USAGE_METER(R) プロジェクト単位で月次集計
  DB-->>B: 利用量(課金対象を区別)
  B->>DB: M_PRJ_QUOTA_LIMITS(R) 無料枠・超過単価を参照
  DB-->>B: 算定基礎
  B->>DB: M_CONTRACT(R) 契約単位へ集約
  DB-->>B: 契約別請求金額
  alt 当該契約・当該月が未確定
    B->>DB: T_BILL_INVOICES(C) 請求書を確定
    B->>MAIL: 月次請求確定通知を送信
    MAIL-->>O: 請求確定メール
    B->>DB: H_NOTIF_LOGS(C) 配信ログ記録
    B->>DB: T_INBOX_MSG(C) 「請求確定」お知らせ生成
  else 確定済み / 対象なし
    B-->>SCH: 当該契約をスキップ(冪等)
  end
```
