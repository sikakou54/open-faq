## シーケンス図

```mermaid
sequenceDiagram
  actor U as 利用者
  participant S as 画面 SCR-029
  participant API as API
  participant DB as DB
  U->>S: 招待リンクから開く(初期表示)
  S->>API: POST /auth/invitations/{token}/preview(API-AUTH-007)
  API->>DB: T_ACCESS_TOKENS(R トークン検証)
  DB-->>API: トークン状態・招待情報
  API-->>S: 成功 / 410 / 使用済み
  alt 成功
    S-->>U: 招待情報(IT-02)・フォームを表示
  else 無効 / 期限切れ(410)
    S-->>U: IT-11 エラー画面を表示
  else 使用済み
    S-->>U: IT-12 既使用エラー画面を表示
  end
```
