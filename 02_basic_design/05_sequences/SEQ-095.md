## シーケンス図

```mermaid
sequenceDiagram
  autonumber
  participant E as 運用イベント発生元
  participant G as システム通知生成処理
  participant DB as DB (D1)
  participant MAIL as メール配信 IF
  actor U as アカウント利用者
  E->>G: 運用イベント発生(event_type)
  G->>G: サブ契機判定 / 宛先解決
  alt 宛先あり
    G->>DB: T_INBOX_MSG(C) 「システム通知」生成
    alt メール対象契機
      G->>MAIL: システム通知メールを送信
      MAIL-->>U: システム通知メール
      G->>DB: H_NOTIF_LOGS(C) 配信ログ記録
    else メール対象外
      G-->>U: 受信箱のみで通知
    end
  else 宛先なし
    G-->>E: 正常終了(通知なし)
  end
```
