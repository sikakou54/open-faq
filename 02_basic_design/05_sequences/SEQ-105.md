## シーケンス図

```mermaid
sequenceDiagram
  autonumber
  participant E as しきい値変更イベント
  participant PR as しきい値伝播処理
  participant DB as DB (D1)
  participant C as しきい値キャッシュ(抽象)
  participant AI as AI 推論処理(API-AI-001)
  E->>PR: AI しきい値変更を通知
  PR->>DB: M_PROJECTS(R) 変更後のしきい値を取得
  DB-->>PR: しきい値(信頼度 / 関連度)
  PR->>C: しきい値を反映(短時間で伝播)
  AI->>C: 推論時に最新しきい値を参照
  alt 取得成功
    C-->>AI: 最新しきい値
  else 取得不能
    AI->>AI: 直近取得値 / 既定値(0.60 / 0.50)で継続
    AI-->>E: フォールバック動作中をアラート通知
  end
  AI->>AI: 確信度・関連度で 回答 / 回答不能 を判定
```
