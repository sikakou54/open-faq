## シーケンス図

```mermaid
sequenceDiagram
  autonumber
  actor U as ウィジェット利用者
  participant W as ウィジェット(送信検証)
  participant J as 受付停止判定処理
  participant DB as DB (D1)
  U->>W: 質問を送信
  W->>J: 受付可否を判定
  J->>DB: M_PRJ_QUOTA_LIMITS(R) / T_USAGE_METER(R) 参照
  DB-->>J: 上限件数・当月利用量
  alt 100% 到達(上限オン)
    J-->>W: 受付停止(安全な定型文・送信無効化)
    W-->>U: 連絡先案内の定型文を表示
  else 未到達 / 上限オフ
    J-->>W: 受付可
    W-->>U: 通常どおり受付
  end
```

- 本図は基本設計レベルの抽象度（利用者 / 画面 / API / DB / 外部・バッチ・通知、`テーブル名(CRUD)` 表記）で記述する。
