## シーケンス図

```mermaid
sequenceDiagram
  autonumber
  participant E as 割当変更イベント発生元
  participant N as 割当変更通知処理
  participant DB as DB (D1)
  participant MAIL as メール配信 IF
  actor M as メンバー
  E->>N: 割当の追加 / 変更 / 剥奪イベント
  N->>N: 対象メンバー・変更内容を特定
  N->>DB: T_INBOX_MSG(C) 「運営お知らせ」(通常)生成
  alt 宛先解決可
    N->>MAIL: 割当変更通知メールを送信
    MAIL-->>M: 割当変更通知メール
    N->>DB: H_NOTIF_LOGS(C) 配信ログ記録
  else 宛先解決不可
    N-->>E: 受信箱のみ生成(メールなし)
  end
```
