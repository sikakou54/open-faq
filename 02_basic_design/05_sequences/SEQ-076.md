## シーケンス図

```mermaid
sequenceDiagram
  actor U as 利用者
  participant S as 画面 SCR-025 プライバシーポリシー閲覧
  participant API as API
  participant DB as DB
  U->>S: 画面を開く(初期表示)
  S->>API: GET /privacy/current(API-TRM-002)
  API->>DB: M_TERMS_VER(R)
  DB-->>API: 最新版全文
  alt ログイン済み
    API->>DB: T_TERMS_AGREE(R)
    DB-->>API: 同意有無
    API-->>S: 200 全文 + 同意状態
    S-->>U: 全文(IT-02)・同意状態バッジ(IT-01)を表示
  else 未ログイン
    API-->>S: 200 全文
    S-->>U: 全文(IT-02)のみ表示(バッジ非表示)
  end
```

## 備考

- 本図は基本設計レベルの抽象度（利用者 / 画面 / API / DB / 外部・バッチ・通知、`テーブル名(CRUD)` 表記）で記述する。
- 図の出典は業務ユースケース [UC-196](../../01_requirements/04_business_usecases/UC-196.md#UC-196)。画面イベントとの対応は UC-196 を参照。
