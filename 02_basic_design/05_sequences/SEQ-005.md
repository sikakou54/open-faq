## シーケンス図

```mermaid
sequenceDiagram
  actor U as 利用者
  participant S as 画面 SCR-003
  participant API as API
  participant DB as DB
  participant ML as メール配信IF
  U->>S: 再送を押下(IT-05、カウントダウン完了後)
  S->>API: POST /auth/password-reset-request(API-AUTH-004)
  API->>DB: M_CONTRACT(R) ・ M_PRJ_USERS(R) 対象照合
  opt 対象が存在する場合のみ
    API->>ML: 再設定リンクメール送信依頼
    ML-->>API: 受理
  end
  API-->>S: 200 一律応答
  S-->>U: カウントダウンをリセットし IT-05 を非活性化
```
