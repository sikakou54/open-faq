## シーケンス図

```mermaid
sequenceDiagram
  actor U as 利用者
  participant S as モーダル SCR-028
  participant API as API
  participant DB as DB
  U->>S: 「プロジェクトを削除」を押下(IT-11)
  S->>S: 削除確認名称(IT-13)の一致を確認
  S->>API: POST /auth/re-auth(L3 再認証)(API-AUTH-005)
  alt 再認証失敗
    API-->>S: 認証エラー
    S-->>U: 「パスワードが正しくありません」を表示
  else 再認証成功
    API-->>S: 再認証 OK
    S->>API: DELETE /projects/{id}(API-PRJ-003)
    API->>DB: M_PROJECTS(D 論理削除) ・ M_PRJ_USERS(D) ・ M_USER(U 無効化)
    alt 成功
      DB-->>API: 削除結果
      API-->>S: 200 削除完了
      S-->>U: モーダルを閉じ一覧を更新
    else 認可エラー / その他
      API-->>S: 403 / エラー
      S-->>U: 権限不足 / トースト表示
    end
  end
```
