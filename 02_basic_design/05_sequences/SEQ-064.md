# <span id="SEQ-064"></span>SEQ-064: 初期表示

## 項目

シーケンス図の対応ユースケースと、図に登場する画面・API・テーブルを示します。

| 項目 | 内容 |

## シーケンス図

```mermaid
sequenceDiagram
  actor U as 利用者
  participant S as 画面 SCR-029
  participant API as API
  participant DB as DB
  U->>S: 画面を開く(初期表示)
  alt トークンなし(SCR-002 完了後)
    S-->>U: 送信済み状態(IT-01/02/03/04/05)を表示
  else トークンあり(確認リンク)
    S->>API: POST /auth/email-verifications/{token}(API-AUTH-006)
    API->>DB: M_CONTRACT(RU メール確認状態)
    DB-->>API: 検証・更新結果
    API-->>S: 成功 / 失敗
    alt 確認成功
      S-->>U: 確認成功(IT-06・IT-08)を表示
    else 期限切れ / 使用済み
      S-->>U: 確認失敗(IT-07・IT-09)を表示
    end
  end
```

## 備考

- 本図は基本設計レベルの抽象度（利用者 / 画面 / API / DB / 外部・バッチ・通知、`テーブル名(CRUD)` 表記）で記述する。
- 図の出典は業務ユースケース [UC-151](../../01_requirements/02_business_usecases/UC-151.md#UC-151)。画面イベントとの対応は UC-151 を参照。
