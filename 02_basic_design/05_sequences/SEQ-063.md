## シーケンス図

```mermaid
sequenceDiagram
  actor U as 利用者
  participant S as 画面 SCR-028 お知らせ詳細
  participant API as API
  participant DB as DB
  U->>S: 画面を開く(初期表示)
  S->>API: GET /me/announcements(API-ANN-001)
  API->>DB: M_SERVICE_ANNOUNCE(R) ・ T_ANNOUNCE_RCPT(R)
  DB-->>API: 本文・メタ・既読状態
  API-->>S: 200 本文・メタ情報
  alt 未読(初回表示)
    S->>API: POST /me/announcements/{id}/read(API-ANN-002)
    API->>DB: T_ANNOUNCE_RCPT(U read_at)
    DB-->>API: 既読化結果
    API-->>S: 200 既読日時
    S-->>U: 本文・メタ表示・既読日時(IT-06)更新
  else 既読済
    S-->>U: 本文・メタ・既読日時(IT-06)を表示
  end
```
