## シーケンス図

```mermaid
sequenceDiagram
  actor U as 利用者
  participant S as モーダル SCR-025 CSV インポート
  participant API as API
  participant J as ジョブ(UC-SYSTEM-001)
  U->>S: 「読み込みを開始」を押下(IT-08)
  S->>API: POST /faqs/import(CSV)(API-FAQ-004)
  API->>J: 取込ジョブを受け付け
  API-->>S: 202 + jobId
  loop 進捗ポーリング
    S->>API: ジョブ進捗を取得(jobId)
    API-->>S: 処理済み件数 / 総件数
    S-->>U: 進捗バー(IT-05)を更新
  end
  alt 全件成功
    S-->>U: 成功メッセージ・モーダルを閉じ FAQ 一覧を再読み込み
  else 部分失敗
    S-->>U: 成功分を反映・エラー一覧(IT-06)に失敗行を表示
  end
```
