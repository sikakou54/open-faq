## シーケンス図

```mermaid
sequenceDiagram
  participant U as アカウント利用者
  participant V as 画面(SCR-029)
  participant A as API(API-FAQ-002)
  participant D as DB
  U->>V: 保存を押下
  V->>V: 入力を検証
  alt バリデーションエラー
    V-->>U: エラー箇所を表示(保存しない)
  else 検証 OK
    alt 新規作成
      V->>A: FAQ 新規作成を要求
    else 既存編集
      V->>A: FAQ 更新を要求(版を添付)
    end
    A->>D: FAQ(C/U)
    alt 成功
      D-->>A: 保存結果
      A-->>V: 成功
      V-->>U: FAQ 一覧(SCR-028)へ遷移
    else 版衝突(楽観ロック)
      A-->>V: 版不一致
      V-->>U: IT-11 を表示・再読み込みを促す
    else 件数上限超過
      A-->>V: 上限超過
      V-->>U: エラーを表示(保存しない)
    end
  end
```
