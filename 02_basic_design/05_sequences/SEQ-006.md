## シーケンス図

```mermaid
sequenceDiagram
  actor U as 利用者
  participant S as 画面 SCR-003
  participant API as API
  participant DB as DB
  U->>S: 再設定リンクから段階 2 を開く(トークン付き)
  S->>API: トークン検証要求(API-AUTH-010 前提)
  API->>DB: M_CONTRACT(R) / M_PRJ_USERS(R) トークン照合
  DB-->>API: トークン状態(有効 / 無効)
  API-->>S: 検証結果
  alt トークン有効
    S-->>U: 段階 2 フォーム(IT-06/07/08)を表示
  else 無効 / 期限切れ
    S-->>U: IT-09 エラーアラート・再送リンクを表示
  end
```
