## シーケンス図

```mermaid
sequenceDiagram
  autonumber
  participant SCH as スケジューラ
  participant B as 削除バッチ
  participant DB as DB (D1)
  SCH->>B: 日次起動
  B->>DB: 論理削除データを走査(90 日経過を判定)
  DB-->>B: 物理削除対象の一覧
  alt 対象あり
    B->>DB: 物理削除(FAQ / 質問 等 従属データ)
    B->>DB: 物理削除(M_USER → M_CONTRACT)
    DB-->>B: 削除結果
    B->>DB: H_AUDIT_LOGS(C) 監査記録
  else 対象なし
    B-->>SCH: 正常終了(削除なし)
  end
```
