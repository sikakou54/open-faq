## シーケンス図

```mermaid
sequenceDiagram
  actor U as 利用者
  participant S as 画面 SCR-028 FAQ 一覧
  participant API as API
  participant DB as DB
  U->>S: 「削除する」を押下(IT-12)
  S-->>U: 確認ダイアログ(件数・警告 FR-174)
  alt キャンセル
    U->>S: キャンセル
    S-->>U: 状態不変(選択・一括操作バー維持)
  else 確認 OK
    U->>S: 削除を確認
    S->>API: 選択件数分の削除を実行 DELETE /faqs/{id}(API-FAQ-002)
    API->>DB: M_FAQS(D 論理削除)
    DB-->>API: 削除結果
    API-->>S: 削除完了
    S->>API: GET /faqs(再取得)(API-FAQ-001)
    API->>DB: M_FAQS(R)
    DB-->>API: 最新一覧
    API-->>S: 200 一覧
    S-->>U: 一覧更新・選択解除
  end
```
