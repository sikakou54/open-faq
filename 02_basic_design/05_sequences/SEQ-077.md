## 概要

利用量と上限画面を開いたとき、当月の質問数利用量と月次上限・無料枠の設定を取得して表示する。取得できた場合は利用量と上限 ON/OFF を反映し、集計前または取得失敗時は空状態を表示する。

## シーケンス図

```mermaid
sequenceDiagram
  autonumber
  actor U as オーナー / メンバー
  participant Screen as 利用量と上限(プロジェクト単位)
  participant Server as サーバー
  U->>Screen: 画面を開く(初期表示)
  Screen->>Server: 当月利用量・上限設定の取得要求
  Server->>Server: 認証・データ取得
  alt 取得成功
    Server-->>Screen: 当月利用量・月次上限・無料枠
    Screen-->>U: 利用量と上限 ON/OFF を表示
  else 集計前 / 取得失敗
    Server-->>Screen: 利用量なし
    Screen-->>U: 空状態を表示
  end
```
