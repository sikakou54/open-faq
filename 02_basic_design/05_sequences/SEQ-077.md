## シーケンス図

```mermaid
sequenceDiagram
  actor U as 利用者
  participant S as 画面 SCR-026 利用量と上限
  participant API as API
  participant DB as DB
  U->>S: 画面を開く(初期表示)
  S->>API: GET /usage(プロジェクト・当月)(API-BIL-001)
  API->>DB: T_USAGE_METER(R)
  DB-->>API: 当月利用量
  API-->>S: 200 利用量
  S->>API: GET /projects/{id}/quota-limits(API-BIL-006)
  API->>DB: M_PRJ_QUOTA_LIMITS(R)
  DB-->>API: 月次上限・無料枠
  API-->>S: 200 上限設定
  alt 取得成功
    S-->>U: 当月利用・上限 ON/OFF 表示(IT-03 / IT-04)
  else 集計前 / 取得失敗
    S-->>U: 空状態(IT-06)
  end
```
