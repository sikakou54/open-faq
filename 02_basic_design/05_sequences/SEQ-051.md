## シーケンス図

```mermaid
sequenceDiagram
  actor U as 利用者
  participant S as モーダル SCR-024
  participant API as API
  participant DB as DB
  U->>S: 確認ダイアログで「外す」を押下(IT-13)

    DB-->>API: 解除結果

    S-->>U: モーダルを閉じ一覧を更新
  else 403 / 失敗
    API-->>S: 4xx
    S-->>U: エラー表示(モーダル保持)
  end
```
