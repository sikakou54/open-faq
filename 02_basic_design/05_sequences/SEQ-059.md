## シーケンス図

```mermaid
sequenceDiagram
  actor U as 利用者
  participant S as 画面 SCR-026 お知らせ一覧
  participant API as API
  participant DB as DB
  U->>S: 「既読化する」を押下(IT-13)
  S->>API: POST /me/announcements/read(選択行)(API-ANN-003)
  API->>DB: T_ANNOUNCE_RCPT(U read_at)
  DB-->>API: 既読化結果
  alt 成功
    API-->>S: 200 既読化完了
    S->>API: GET /me/announcements(再取得)(API-ANN-001)
    API->>DB: M_SERVICE_ANNOUNCE(R) ・ T_ANNOUNCE_RCPT(R)
    DB-->>API: 最新一覧
    API-->>S: 200 一覧
    S-->>U: 一覧・未読件数更新・選択解除
  else 失敗
    API-->>S: エラー
    S-->>U: エラーをトースト表示
  end
```
