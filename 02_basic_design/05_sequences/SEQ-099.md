## シーケンス図

```mermaid
sequenceDiagram
  autonumber
  participant W as ウィジェット(質問送信)
  participant A as 利用量集計処理
  participant DB as DB (D1)
  participant S as 管理画面サマリ
  W->>A: 質問リクエスト到達
  A->>DB: T_USAGE_METER(CU) 当月質問数を計測(課金対象を区別)
  A->>DB: M_PRJ_QUOTA_LIMITS(R) 上限・選択閾値を参照
  DB-->>A: 上限・選択閾値
  alt 上限オン
    A->>A: 利用率を算出し閾値 / 100% / 125% を判定
    A-->>A: 発火契機を担当 UC へ引き渡し
  else 上限オフ
    A-->>A: 計測のみ(判定なし)
  end
  A->>S: 利用量サマリへ反映(5 分以内)
```
