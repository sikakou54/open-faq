## シーケンス図

```mermaid
sequenceDiagram
  actor U as 利用者
  participant S as 画面 SCR-025
  participant API as API
  participant DB as DB
  U->>S: 同意して続行を押下(IT-04)
  opt 利用規約が改定対象
    S->>API: 利用規約 同意(API-TRM-003)
    API->>DB: T_TERMS_AGREE(C)
    DB-->>API: 記録結果
    API-->>S: 200
  end
  opt プライバシーポリシーが改定対象
    S->>API: プライバシーポリシー 同意(API-TRM-004)
    API->>DB: T_TERMS_AGREE(C)
    DB-->>API: 記録結果
    API-->>S: 200
  end
  alt 全て成功
    S-->>U: 割込み前の操作へ戻る
  else 失敗
    S-->>U: エラーを表示しモーダルを維持
  end
```

- 本図は基本設計レベルの抽象度（利用者 / 画面 / API / DB / 外部・バッチ・通知、`テーブル名(CRUD)` 表記）で記述する。
- 図の出典は業務ユースケース [UC-169](../../01_requirements/04_business_usecases/UC-169.md#UC-169)。画面イベントとの対応は UC-169 を参照。
