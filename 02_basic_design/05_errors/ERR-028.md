|----|----|----|----|----|----|

## <span id="def"></span>1. 定義

| 項目 | 値 |
|----|----|
| エラーコード | `PAYMENT_METHOD_DECLINED` |
| 分類コード(taxonomy) | `E-BILL-PAYMENT-FAILED` |
| HTTP ステータス | 402 |
| エラー分類 | 業務(課金) |

## <span id="msg"></span>2. メッセージ

カードが拒否された

## <span id="cond"></span>3. 発生条件・表示先・復旧手段

| 観点 | 内容 |
|----|----|
| 発生条件 | 支払方法の登録・更新で、カード会社により決済が拒否されたとき。 |
| 表示先 | 画面トースト(支払方法画面のエラー表示) |
| 復旧手段 | 別の有効なカードを登録するか、カード会社へ確認のうえ再度登録する。 |
