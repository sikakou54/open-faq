# <span id="ERR-024"></span>ERR-024: CSV_INVALID

> **このページは API レスポンスで返すエラー `CSV_INVALID` の HTTP ステータス・分類・メッセージ・業務ユースケースID・API ID を定義します。**

| ID | 業務ユースケースID | API ID |
|----|----|----|
| ERR-024 | [UC-027](../../01_requirements/04_business_usecases/UC-027.md#UC-027) ・ [UC-046](../../01_requirements/04_business_usecases/UC-046.md#UC-046) | [API-028](../02_backend/03_apis/API-028.md#API-028) |

| エラーコード | HTTPステータス | エラー分類 |
|----|----|----|
| `CSV_INVALID` | 415 / 400 | 入力検証 |
*種別 エラー定義 ・ 分類 入力検証 ・ ステータス ドラフト*

## <span id="def"></span>1. 定義

| 項目 | 値 |
|----|----|
| エラーコード | `CSV_INVALID` |
| 分類コード(taxonomy) | `E-INPUT-CSV-INVALID` |
| HTTP ステータス | 415 / 400 |
| エラー分類 | 入力検証 |

## <span id="msg"></span>2. メッセージ

CSV 以外 / 不正形式

## <span id="cond"></span>3. 発生条件・表示先・復旧手段

| 観点 | 内容 |
|----|----|
| 発生条件 | FAQ CSV インポートで、CSV 以外のファイル形式、または列構成・内容が不正なファイルをアップロードしたとき。 |
| 表示先 | 画面トースト(インポート画面のエラー表示) |
| 復旧手段 | 所定の CSV フォーマットに整えたファイルをアップロードし直す。 |
