## <span id="msg"></span>2. メッセージ

## <span id="cond"></span>3. 発生条件・表示先・復旧手段

| 観点 | 内容 |
|----|----|
| 発生条件 | ウィジェットセッショントークン(`wst_*`)が無効 / 期限切れのまま認証付きウィジェット API を呼び出したとき。ウィジェットは本エラー検知時に自動的に [API-037](../02_backend/03_apis/API-037.md#API-037) で再 bootstrap してから再送する(透過的再確立)。 |
| 表示先 | 画面には表示しない(ウィジェットが透過的に再確立し再送するため。再送も失敗した場合のみ処理エラー EM-03 を表示) |
| 復旧手段 | ウィジェットが [API-037](../02_backend/03_apis/API-037.md#API-037) でセッションを再取得して自動再送する。 |
