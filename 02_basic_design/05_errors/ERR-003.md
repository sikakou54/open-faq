## <span id="def"></span>1. 定義

| 項目 | 値 |
|----|----|
| エラーコード | `LOCKED_OUT` |
| 分類コード(taxonomy) | `E-AUTH-LOCKED` |
| HTTP ステータス | 423 |
| エラー分類 | 認証 |

## <span id="msg"></span>2. メッセージ

アカウントロック中

## <span id="cond"></span>3. 発生条件・表示先・復旧手段

| 観点 | 内容 |
|----|----|
| 発生条件 | ログインAPIで、連続したログイン失敗によりアカウントがロックされている状態でログインを試みたとき。 |
| 表示先 | 画面トースト |
| 復旧手段 | ロック解除まで時間をおいてから再ログインする。または案内に従いパスワード再設定を行う。 |
