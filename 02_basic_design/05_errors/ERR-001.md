# <span id="ERR-001"></span>ERR-001: VALIDATION_ERROR

> **このページは API レスポンスで返すエラー `VALIDATION_ERROR` の HTTP ステータス・分類・メッセージ・対応 API / EVT を定義します。**

*種別 エラー定義 ・ 分類 入力検証 ・ ステータス ドラフト*

## <span id="def"></span>1. 定義

| 項目 | 値 |
|----|----|
| エラーコード | `VALIDATION_ERROR` |
| 分類コード(taxonomy) | — |
| HTTP ステータス | 400 |
| エラー分類 | 入力検証 |

## <span id="msg"></span>2. メッセージ

- 入力値の検証エラー
- 氏名長 / パスワード強度 / 規約同意未チェック
- パスワード強度不足等
- パスワード強度不足 / 不一致
- メールアドレス形式不正
- `name` / `allowedDomains` 必須違反
- 質問 500 字 / 回答 5,000 字超過ほか
- 50 件超過 / 不正な `status`
- メンバーが `projectId` 未指定
- 100 件超過
