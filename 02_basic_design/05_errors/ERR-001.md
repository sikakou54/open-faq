## <span id="msg"></span>2. メッセージ

入力値の検証エラー。発生する API・入力ごとに表示メッセージが異なる。どの API・どの入力で何が表示されるかは「3. 発生条件・表示先・復旧手段」の発生条件表で対応づける。

## <span id="cond"></span>3. 発生条件・表示先・復旧手段

| 観点 | 内容 |
|----|----|
| 発生条件 | 各 API での入力検証違反。発生 API・入力と表示内容は下表のとおり。 |
| 表示先 | 画面トースト(該当入力欄のインラインエラー併用) |
| 復旧手段 | 指摘された入力項目を正しい値に修正し再送信する |
