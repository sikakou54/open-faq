# <span id="EVT-216"></span>EVT-216: 請求・重要通知メールを入力

入力値をリアルタイムでバリデーションする。
<ul>
<li>正常: 入力を受け付ける</li>
<li>エラー: メール形式不正の場合、IT-01 下部にインラインエラーを表示し IT-02 を無効化する</li>
</ul>
