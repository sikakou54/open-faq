# <span id="EVT-042"></span>EVT-042: 「プロジェクトを削除」を押下

> **画面 SCR-005 のイベント「「プロジェクトを削除」を押下」を定義します。** 対応画面・対応業務UC・対象項目・呼出API・遷移先と、処理内容を記述します。

*版数 v1.0 ・ 更新 2026-06-21 ・ 再構成 P3*

## 項目

| 項目 | 内容 |
|---|---|
| 画面イベントID | `EVT-042` |
| イベント名 | 「プロジェクトを削除」を押下 |
| 対応画面ID | [SCR-005](../01_screens/SCR-005.md#SCR-005) |
| 対応業務UC | [UC-042](../../01_requirements/02_business_usecases/UC-042.md#UC-042) |
| 対象項目ID | [IT-11](../01_screens/SCR-005.md#IT-11) |
| 呼出API | [プロジェクト更新・削除](../03_apis/API-project.md#API-PRJ-003) |
| 遷移先 | [SCR-004](../01_screens/SCR-004.md#SCR-004) |

## 処理

<ul>
<li>削除確認名称(IT-13)が現プロジェクト名と完全一致していない場合は処理を中断する(ボタンは無効状態のため通常到達しない)</li>
<li>再認証(L3 パスワード再認証)を要求する</li>
<li>成功時: <a href="../03_apis/API-project.md#API-PRJ-003">プロジェクト更新・削除</a> API(DELETE)でプロジェクトを論理削除する。メンバー割当を解除し、他に有効割当を持たないメンバーのアカウントを無効化する。モーダルを閉じ SCR-004 の一覧を更新する</li>
<li>失敗時(再認証エラー): 「パスワードが正しくありません」をインラインで表示する</li>
<li>失敗時(その他): トーストでエラーを表示する</li>
</ul>

## 備考

再構成 P3 で旧画面イベント `SCR-004-001/EV-10`(現 SCR-005 §6 の `EV-10`)から導出。処理内容の正本は SCR-005 §6 と本ページ。
