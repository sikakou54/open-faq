# <span id="EVT-076"></span>EVT-076: 初期表示

> **画面 SCR-009 のイベント「初期表示」を定義します。** 対応画面・対応業務UC・対象項目・呼出API・遷移先と、処理内容を記述します。

*版数 v1.0 ・ 更新 2026-06-21 ・ 再構成 P3*

| 項目 | 内容 |
|---|---|
| 画面イベントID | `EVT-076` |

| 対応画面ID | [SCR-009](../01_screens/SCR-009.md#SCR-009) |

## 処理

<ul>
<li>既存編集: <a href="../03_apis/API-faq.md#API-FAQ-009">FAQ 個別取得</a> で現値をロードし各欄へ展開。登録元未解決質問が存在する場合は IT-07 を表示</li>
<li>新規作成(未解決質問起点): 未解決質問の質問文を IT-03 へ初期反映し、その他の欄は空</li>
<li>新規作成(手動): 全欄空フォームを表示</li>
<li>権限なし(URL 直アクセス): 権限不足メッセージを表示して操作不可</li>
</ul>
