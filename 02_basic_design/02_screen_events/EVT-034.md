# <span id="EVT-034"></span>EVT-034: 初期表示(編集モード)

> **画面 SCR-005 のイベント「初期表示(編集モード)」を定義します。** 対応画面・対応業務UC・対象項目・呼出API・遷移先と、処理内容を記述します。

*版数 v1.0 ・ 更新 2026-06-21 ・ 再構成 P3*

## 項目

| 項目 | 内容 |
|---|---|
| 画面イベントID | `EVT-034` |
| イベント名 | 初期表示(編集モード) |
| 対応画面ID | [SCR-005](../01_screens/SCR-005.md#SCR-005) |
| 対応業務UC | [UC-034](../../01_requirements/02_business_usecases/UC-034.md#UC-034) |
| 対象項目ID | — |
| 呼出API | [プロジェクト更新・削除](../03_apis/API-018.md#API-018) |
| 遷移先 | — |

## 処理

<ul>
<li>成功時: <a href="../03_apis/API-018.md#API-018">プロジェクト更新・削除</a> API(GET)で現値を取得し各入力欄に設定する。見出し(IT-01)を「プロジェクトを編集」に設定し、編集モード専用項目(IT-02 / IT-07 / IT-10 / IT-11)を表示する</li>
<li>失敗時: 取得エラーをトーストで表示しモーダルを閉じる</li>
</ul>
