# <span id="EVT-133"></span>EVT-133: 初期表示

> **画面 SCR-015 のイベント「初期表示」を定義します。** 対応画面・対応業務UC・対象項目・呼出API・遷移先と、処理内容を記述します。

*版数 v1.0 ・ 更新 2026-06-21 ・ 再構成 P3*

## 項目

| 項目 | 内容 |
|---|---|
| 画面イベントID | `EVT-133` |
| イベント名 | 初期表示 |
| 対応画面ID | [SCR-015](../01_screens/SCR-015.md#SCR-015) |
| 対応業務UC | [UC-133](../../01_requirements/02_business_usecases/UC-133.md#UC-133) |
| 対象項目ID | — |
| 呼出API | [利用規約 最新版取得](../03_apis/API-052.md#API-052) |
| 遷移先 | — |

## 処理

<ul>
<li><a href="../03_apis/API-052.md#API-052">利用規約 最新版取得</a> を呼び出し、最新版全文(IT-02)を表示する</li>
<li>ログイン済みの場合: <code>T_TERMS_AGREE</code> をサーバーサイドで照合して同意状態バッジ(IT-01)を表示する(同意済み: 緑バッジ / 未同意: 赤バッジ + 「再同意へ」リンク)</li>
<li>未ログインの場合: 同意状態バッジは非表示とする</li>
</ul>

## 備考

再構成 P3 で旧画面イベント `SCR-010/EV-01`(現 SCR-015 §6 の `EV-01`)から導出。処理内容の正本は SCR-015 §6 と本ページ。
