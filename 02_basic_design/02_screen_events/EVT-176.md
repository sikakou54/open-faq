# <span id="EVT-176"></span>EVT-176: メールアドレスを入力

> **画面 SCR-022 のイベント「メールアドレスを入力」を定義します。** 対応画面・対応業務UC・対象項目・呼出API・遷移先と、処理内容を記述します。

*版数 v1.0 ・ 更新 2026-06-21 ・ 再構成 P3*

## 項目

| 項目 | 内容 |
|---|---|
| 画面イベントID | `EVT-176` |
| イベント名 | メールアドレスを入力 |
| 対応画面ID | [SCR-022](../01_screens/SCR-022.md#SCR-022) |
| 対応業務UC | [UC-176](../../01_requirements/02_business_usecases/UC-176.md#UC-176) |
| 対象項目ID | [IT-02](../01_screens/SCR-022.md#IT-02) |
| 呼出API | — |
| 遷移先 | — |

## 処理

<ul>
<li>成功時: メールアドレス形式として正当であればバリデーションエラーを表示しない</li>
<li>失敗時: 空欄またはメールアドレス形式でない場合、入力欄下部にバリデーションエラーを表示する</li>
</ul>

## 備考
