# <span id="EVT-152"></span>EVT-152: 「メールを再送する」を押下

> **画面 SCR-018 のイベント「「メールを再送する」を押下」を定義します。** 対応画面・対応業務UC・対象項目・呼出API・遷移先と、処理内容を記述します。

*版数 v1.0 ・ 更新 2026-06-21 ・ 再構成 P3*

## 項目

| 項目 | 内容 |
|---|---|
| 画面イベントID | `EVT-152` |
| イベント名 | 「メールを再送する」を押下 |
| 対応画面ID | [SCR-018](../01_screens/SCR-018.md#SCR-018) |
| 対応業務UC | [UC-152](../../01_requirements/02_business_usecases/UC-152.md#UC-152) |
| 対象項目ID | [IT-04](../01_screens/SCR-018.md#IT-04) |
| 呼出API | [新規登録(確認メール再送)](../03_apis/API-001.md#API-001) |
| 遷移先 | — |

## 処理

<a href="../03_apis/API-001.md#API-001">新規登録(確認メール再送)</a> API を呼び出し確認メールを再送する。<ul>
<li>成功時: ボタンを非活性化してカウントダウン(レート制限 5 分)を表示する</li>
<li>レート制限中: ボタンは非活性のまま(カウントダウン終了で再活性)</li>
<li>失敗時: エラーメッセージを表示する</li>
</ul>

## 備考

再構成 P3 で旧画面イベント `SCR-013/EV-02`(現 SCR-018 §6 の `EV-02`)から導出。処理内容の正本は SCR-018 §6 と本ページ。
