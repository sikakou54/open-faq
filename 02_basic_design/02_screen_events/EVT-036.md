# <span id="EVT-036"></span>EVT-036: 許可ドメインを入力

> **画面 SCR-005 のイベント「許可ドメインを入力」を定義します。** 対応画面・対応業務UC・対象項目・呼出API・遷移先と、処理内容を記述します。

*版数 v1.0 ・ 更新 2026-06-21 ・ 再構成 P3*

## 項目

| 項目 | 内容 |
|---|---|
| 画面イベントID | `EVT-036` |
| イベント名 | 許可ドメインを入力 |
| 対応画面ID | [SCR-005](../01_screens/SCR-005.md#SCR-005) |
| 対応業務UC | [UC-036](../../01_requirements/02_business_usecases/UC-036.md#UC-036) |
| 対象項目ID | [IT-04](../01_screens/SCR-005.md#IT-04) |
| 呼出API | — |
| 遷移先 | — |

## 処理

Enter またはカンマでタグを追加する。追加時にドメイン形式(完全一致または <code>*.example.com</code> 形式・IP アドレス・プロトコル指定不可)を検証し、不正な場合はインラインエラーを表示してタグを追加しない

## 備考

再構成 P3 で旧画面イベント `SCR-004-001/EV-04`(現 SCR-005 §6 の `EV-04`)から導出。処理内容の正本は SCR-005 §6 と本ページ。
