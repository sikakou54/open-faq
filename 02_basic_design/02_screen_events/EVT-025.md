# <span id="EVT-025"></span>EVT-025: 「新しいパスワードを設定する」を押下

> **画面 SCR-003 のイベント「「新しいパスワードを設定する」を押下」を定義します。** 対応画面・対応業務UC・対象項目・呼出API・遷移先と、処理内容を記述します。

*版数 v1.0 ・ 更新 2026-06-21 ・ 再構成 P3*

## 項目

| 項目 | 内容 |
|---|---|
| 画面イベントID | `EVT-025` |
| イベント名 | 「新しいパスワードを設定する」を押下 |
| 対応画面ID | [SCR-003](../01_screens/SCR-003.md#SCR-003) |
| 対応業務UC | [UC-025](../../01_requirements/02_business_usecases/UC-025.md#UC-025) |
| 対象項目ID | [IT-08](../01_screens/SCR-003.md#IT-08) |
| 呼出API | [パスワード再設定確定](../03_apis/API-auth.md#API-AUTH-010) |
| 遷移先 | — |

## 処理

<ul>
<li>IT-06 の強度要件(FR-006: 12 文字以上・英大小文字・数字・記号 3 種類以上)を検証する</li>
<li>IT-07 の一致を検証する</li>
<li>検証失敗時: 不足要件メッセージ / 不一致エラーを表示してリクエストを中断する</li>
<li>検証成功時: <a href="../03_apis/API-auth.md#API-AUTH-010">パスワード再設定確定</a> API を発行する。成功時はパスワードハッシュを更新し当該ユーザーの全セッションを失効させる</li>
<li>完了後: IT-10 完了画面(「ログインしてください」案内 + IT-10 ボタン)を表示する</li>
</ul>

## 備考

再構成 P3 で旧画面イベント `SCR-003/EV-07`(現 SCR-003 §6 の `EV-07`)から導出。処理内容の正本は SCR-003 §6 と本ページ。
