# <span id="EVT-011"></span>EVT-011: 業種を選択

> **画面 SCR-002 のイベント「業種を選択」を定義します。** 対応画面・対応業務UC・対象項目・呼出API・遷移先と、処理内容を記述します。

*版数 v1.0 ・ 更新 2026-06-21 ・ 再構成 P3*

## 項目

| 項目 | 内容 |
|---|---|
| 画面イベントID | `EVT-011` |
| イベント名 | 業種を選択 |
| 対応画面ID | [SCR-002](../01_screens/SCR-002.md#SCR-002) |
| 対応業務UC | [UC-011](../../01_requirements/02_business_usecases/UC-011.md#UC-011) |
| 対象項目ID | [IT-04](../01_screens/SCR-002.md#IT-04) |
| 呼出API | — |
| 遷移先 | — |

## 処理

高規制業界(金融 / 医療等)を選択した場合、提供範囲外である旨の注意メッセージとサポート窓口の案内をフォーム上にインライン表示する。登録ボタンは非活性にせず、登録自体は許容する。それ以外の選択では追加表示なし

## 備考

再構成 P3 で旧画面イベント `SCR-002/EV-05`(現 SCR-002 §6 の `EV-05`)から導出。処理内容の正本は SCR-002 §6 と本ページ。
