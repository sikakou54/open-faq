# <span id="EVT-142"></span>EVT-142: 「既読化する」を押下

> **画面 SCR-016 のイベント「「既読化する」を押下」を定義します。** 対応画面・対応業務UC・対象項目・呼出API・遷移先と、処理内容を記述します。

*版数 v1.0 ・ 更新 2026-06-21 ・ 再構成 P3*

## 項目

| 項目 | 内容 |
|---|---|
| 画面イベントID | `EVT-142` |
| イベント名 | 「既読化する」を押下 |
| 対応画面ID | [SCR-016](../01_screens/SCR-016.md#SCR-016) |
| 対応業務UC | [UC-142](../../01_requirements/02_business_usecases/UC-142.md#UC-142) |
| 対象項目ID | [IT-13](../01_screens/SCR-016.md#IT-13) |
| 呼出API | [お知らせ一括既読](../03_apis/API-inbox.md#API-ANN-003) |
| 遷移先 | — |

## 処理

<ul>
<li>選択行を <a href="../03_apis/API-inbox.md#API-ANN-003">お知らせ一括既読</a> API で既読化する</li>
<li>成功時: 一覧を再取得し未読件数(IT-04)を更新する。選択状態を解除し一括操作バー(IT-12)を非表示にする</li>
<li>失敗時: エラーをトーストで表示する</li>
</ul>
