# <span id="EVT-190"></span>EVT-190: 「登録を完了する」を押下

> **画面 SCR-023 のイベント「「登録を完了する」を押下」を定義します。** 対応画面・対応業務UC・対象項目・呼出API・遷移先と、処理内容を記述します。

*版数 v1.0 ・ 更新 2026-06-21 ・ 再構成 P3*

## 項目

| 項目 | 内容 |
|---|---|
| 画面イベントID | `EVT-190` |
| イベント名 | 「登録を完了する」を押下 |
| 対応画面ID | [SCR-023](../01_screens/SCR-023.md#SCR-023) |
| 対応業務UC | [UC-190](../../01_requirements/02_business_usecases/UC-190.md#UC-190) |
| 対象項目ID | [IT-10](../01_screens/SCR-023.md#IT-10) |
| 呼出API | [メンバーアカウント有効化](../03_apis/API-auth.md#API-AUTH-008) |
| 遷移先 | — |

## 処理

<ul>
<li>全項目のクライアント側バリデーション(EV-02〜EV-09 の各検証)を実行し、エラーがある場合は送信せずエラーを表示する</li>
<li>バリデーション通過後、<a href="../03_apis/API-auth.md#API-AUTH-008">メンバーアカウント有効化</a> API を実行(同一トランザクション: 予約 <code>M_USER</code> への氏名・パスワードハッシュ設定・<code>status='active'</code> 化 / <code>M_PRJ_USERS</code> の <code>valid=1</code> 化 / <code>T_TERMS_AGREE</code> 2 件登録 / トークン消費)</li>
<li>成功時: IT-13 完了画面を表示する</li>
<li>HTTP 400(フィールドエラー)時: フィールド単位のエラーメッセージを表示し、入力フォームを操作可能なまま維持する</li>
<li>HTTP 410(トークン期限切れ / 無効)時: IT-11 トークン無効 / 期限切れエラー画面を表示する</li>
</ul>

## 備考

再構成 P3 で旧画面イベント `SCR-018/EV-10`(現 SCR-023 §6 の `EV-10`)から導出。処理内容の正本は SCR-023 §6 と本ページ。
