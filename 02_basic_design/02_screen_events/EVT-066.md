# <span id="EVT-066"></span>EVT-066: 行を選択

> **画面 SCR-008 のイベント「行を選択」を定義します。** 対応画面・対応業務UC・対象項目・呼出API・遷移先と、処理内容を記述します。

*版数 v1.0 ・ 更新 2026-06-21 ・ 再構成 P3*

## 項目

| 項目 | 内容 |
|---|---|
| 画面イベントID | `EVT-066` |
| イベント名 | 行を選択 |
| 対応画面ID | [SCR-008](../01_screens/SCR-008.md#SCR-008) |
| 対応業務UC | [UC-066](../../01_requirements/02_business_usecases/UC-066.md#UC-066) |
| 対象項目ID | [IT-10](../01_screens/SCR-008.md#IT-10) |
| 呼出API | — |
| 遷移先 | — |

## 処理

<ul>
<li>チェックを入れると対象 FAQ を選択状態にし、1 件以上選択時に IT-12 の一括操作バーを下部に表示する</li>
<li>チェックを外すと選択を解除し、0 件になった時は一括操作バーを非表示にする</li>
<li>最大 50 件まで選択可(FR-173)</li>
</ul>

## 備考

再構成 P3 で旧画面イベント `SCR-006/EV-05`(現 SCR-008 §6 の `EV-05`)から導出。処理内容の正本は SCR-008 §6 と本ページ。
