# <span id="EVT-195"></span>EVT-195: 「閉じる」を押下

> **画面 SCR-024 のイベント「「閉じる」を押下」を定義します。** 対応画面・対応業務UC・対象項目・呼出API・遷移先と、処理内容を記述します。

*版数 v1.0 ・ 更新 2026-06-21 ・ 再構成 P3*

## 項目

| 項目 | 内容 |
|---|---|
| 画面イベントID | `EVT-195` |
| イベント名 | 「閉じる」を押下 |
| 対応画面ID | [SCR-024](../01_screens/SCR-024.md#SCR-024) |
| 対応業務UC | [UC-195](../../01_requirements/02_business_usecases/UC-195.md#UC-195) |
| 対象項目ID | [IT-04](../01_screens/SCR-024.md#IT-04) |
| 呼出API | — |
| 遷移先 | — |

## 処理

<ul>
<li>window.close() を実行してタブ / ウィンドウを閉じる</li>
<li>close() が無効な場合(直接 URL アクセス等)は静的な完了メッセージをそのまま表示し続ける(認証コンソールへの遷移は行わない)</li>
</ul>
