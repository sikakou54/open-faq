# <span id="EVT-204"></span>EVT-204: 「今月の利用上限」を入力

> **画面 SCR-027 のイベント「「今月の利用上限」を入力」を定義します。** 対応画面・対応業務UC・対象項目・呼出API・遷移先と、処理内容を記述します。

*版数 v1.0 ・ 更新 2026-06-21 ・ 再構成 P3*

## 項目

| 項目 | 内容 |
|---|---|
| 画面イベントID | `EVT-204` |
| イベント名 | 「今月の利用上限」を入力 |
| 対応画面ID | [SCR-027](../01_screens/SCR-027.md#SCR-027) |
| 対応業務UC | [UC-204](../../01_requirements/02_business_usecases/UC-204.md#UC-204) |
| 対象項目ID | [IT-02](../01_screens/SCR-027.md#IT-02) |
| 呼出API | — |
| 遷移先 | — |

## 処理

<ul>
<li>成功: 入力値をもとに課金対象件数・最大課金額をリアルタイムで計算し計算式に反映する</li>
<li>失敗(範囲外・非整数): 入力欄にエラーを表示し保存ボタンを無効化する</li>
</ul>
