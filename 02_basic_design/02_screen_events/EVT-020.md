# <span id="EVT-020"></span>EVT-020: 「再設定リンクを送信」を押下

> **画面 SCR-003 のイベント「「再設定リンクを送信」を押下」を定義します。** 対応画面・対応業務UC・対象項目・呼出API・遷移先と、処理内容を記述します。

*版数 v1.0 ・ 更新 2026-06-21 ・ 再構成 P3*

## 項目

| 項目 | 内容 |
|---|---|
| 画面イベントID | `EVT-020` |
| イベント名 | 「再設定リンクを送信」を押下 |
| 対応画面ID | [SCR-003](../01_screens/SCR-003.md#SCR-003) |
| 対応業務UC | [UC-020](../../01_requirements/02_business_usecases/UC-020.md#UC-020) |
| 対象項目ID | [IT-03](../01_screens/SCR-003.md#IT-03) |
| 呼出API | [パスワード再設定要求](../03_apis/API-004.md#API-004) |
| 遷移先 | — |

## 処理

<ul>
<li>IT-02 の形式バリデーションを実行する</li>
<li>形式不正時: IT-02 直下にエラーメッセージを表示してリクエストを中断</li>
<li>形式正常時: <a href="../03_apis/API-004.md#API-004">パスワード再設定要求</a> API を発行する(存在有無は漏らさず一律応答)</li>
<li>応答受取後: IT-04 送信済み案内と IT-05 再送ボタン(カウントダウン付き)を表示する</li>
</ul>

## 備考

再構成 P3 で旧画面イベント `SCR-003/EV-02`(現 SCR-003 §6 の `EV-02`)から導出。処理内容の正本は SCR-003 §6 と本ページ。
