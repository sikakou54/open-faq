# <span id="EVT-217"></span>EVT-217: 「変更を保存」を押下

> **画面 SCR-029 のイベント「「変更を保存」を押下」を定義します。** 対応画面・対応業務UC・対象項目・呼出API・遷移先と、処理内容を記述します。

*版数 v1.0 ・ 更新 2026-06-21 ・ 再構成 P3*

## 項目

| 項目 | 内容 |
|---|---|
| 画面イベントID | `EVT-217` |
| イベント名 | 「変更を保存」を押下 |
| 対応画面ID | [SCR-029](../01_screens/SCR-029.md#SCR-029) |
| 対応業務UC | [UC-217](../../01_requirements/02_business_usecases/UC-217.md#UC-217) |
| 対象項目ID | [IT-02](../01_screens/SCR-029.md#IT-02) |
| 呼出API | [契約設定更新](../03_apis/API-015.md#API-015) |
| 遷移先 | — |

## 処理

入力値を最終バリデーションのうえ、<a href="../03_apis/API-015.md#API-015">契約設定更新</a> を呼び出して連絡先メール・タイムゾーンを更新する。
<ul>
<li>成功: 成功トースト(「設定を更新しました」)を表示し、IT-01・IT-06 を更新後の値で再描画する</li>
<li>エラー(バリデーション): エラー内容を該当項目下部に表示する</li>
<li>エラー(API): エラートーストを表示し、入力値を保持する</li>
</ul>
