# <span id="EVT-140"></span>EVT-140: 行を選択

> **画面 SCR-016 のイベント「行を選択」を定義します。** 対応画面・対応業務UC・対象項目・呼出API・遷移先と、処理内容を記述します。

*版数 v1.0 ・ 更新 2026-06-21 ・ 再構成 P3*

## 項目

| 項目 | 内容 |
|---|---|
| 画面イベントID | `EVT-140` |
| イベント名 | 行を選択 |
| 対応画面ID | [SCR-016](../01_screens/SCR-016.md#SCR-016) |
| 対応業務UC | [UC-140](../../01_requirements/02_business_usecases/UC-140.md#UC-140) |
| 対象項目ID | [IT-11](../01_screens/SCR-016.md#IT-11) |
| 呼出API | — |
| 遷移先 | — |

## 処理

<ul>
<li>チェックをオンにした行を一括操作の対象として選択状態にする(最大 100 件: FR-173)</li>
<li>1 件以上選択時: 一括操作バー(IT-12)を表示する</li>
<li>すべての選択を解除した場合: 一括操作バー(IT-12)を非表示にする</li>
</ul>

## 備考

再構成 P3 で旧画面イベント `SCR-011/EV-05`(現 SCR-016 §6 の `EV-05`)から導出。処理内容の正本は SCR-016 §6 と本ページ。
