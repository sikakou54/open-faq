# <span id="EVT-203"></span>EVT-203: 上限設定トグルを切り替え

> **画面 SCR-027 のイベント「上限設定トグルを切り替え」を定義します。** 対応画面・対応業務UC・対象項目・呼出API・遷移先と、処理内容を記述します。

*版数 v1.0 ・ 更新 2026-06-21 ・ 再構成 P3*

## 項目

| 項目 | 内容 |
|---|---|
| 画面イベントID | `EVT-203` |
| イベント名 | 上限設定トグルを切り替え |
| 対応画面ID | [SCR-027](../01_screens/SCR-027.md#SCR-027) |
| 対応業務UC | [UC-203](../../01_requirements/02_business_usecases/UC-203.md#UC-203) |
| 対象項目ID | [IT-01](../01_screens/SCR-027.md#IT-01) |
| 呼出API | — |
| 遷移先 | — |

## 処理

<ul>
<li>OFF に切り替え: IT-02(件数入力)・IT-04(アラート設定)を非活性化し、全アラート閾値を未選択状態にする</li>
<li>ON に切り替え: IT-02・IT-04 を活性化し、課金計算式を再描画する</li>
</ul>

## 備考

再構成 P3 で旧画面イベント `SCR-021-001/EV-02`(現 SCR-027 §6 の `EV-02`)から導出。処理内容の正本は SCR-027 §6 と本ページ。
