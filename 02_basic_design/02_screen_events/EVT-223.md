# <span id="EVT-223"></span>EVT-223: ランチャーバッジを押下

> **画面 SCR-030 のイベント「ランチャーバッジを押下」を定義します。** 対応画面・対応業務UC・対象項目・呼出API・遷移先と、処理内容を記述します。

*版数 v1.0 ・ 更新 2026-06-21 ・ 再構成 P3*

## 項目

| 項目 | 内容 |
|---|---|
| 画面イベントID | `EVT-223` |
| イベント名 | ランチャーバッジを押下 |
| 対応画面ID | [SCR-030](../01_screens/SCR-030.md#SCR-030) |
| 対応業務UC | [UC-223](../../01_requirements/02_business_usecases/UC-223.md#UC-223) |
| 対象項目ID | [IT-01](../01_screens/SCR-030.md#IT-01) |
| 呼出API | [ウィジェット起動](../03_apis/API-037.md#API-037) |
| 遷移先 | — |

## 処理

<ul>
<li>バッジを非表示にしチャット UI を展開する(キーボード操作 Enter / Space キーによる起動も同等に扱う)</li>
<li><a href="../03_apis/API-037.md#API-037">ウィジェット起動</a> API でセッションを確立し、ウィジェット設定(タイトル・連絡先メール等)を取得する</li>
<li>失敗時: エラーメッセージを会話欄に表示し、再試行案内を行う(連打防止付き)</li>
</ul>

## 備考

再構成 P3 で旧画面イベント `SCR-WIDGET/EV-02`(現 SCR-030 §6 の `EV-02`)から導出。処理内容の正本は SCR-030 §6 と本ページ。
