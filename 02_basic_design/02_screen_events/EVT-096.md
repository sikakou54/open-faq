# <span id="EVT-096"></span>EVT-096: 初期表示

> **画面 SCR-011 のイベント「初期表示」を定義します。** 対応画面・対応業務UC・対象項目・呼出API・遷移先と、処理内容を記述します。

*版数 v1.0 ・ 更新 2026-06-21 ・ 再構成 P3*

| 項目 | 内容 |

| 画面イベントID | `EVT-096` |

## 処理

<ul>
<li>当該プロジェクトの公開キー・ウィジェット設定(主色・表示位置・見出し・初期メッセージ等)・埋め込みコードを取得し、各セクション(IT-02 / IT-05 / IT-08 / IT-10 / IT-11 / IT-12)へ表示する</li>
<li>ローテーション猶予中に旧キー使用を検知している場合は旧キー使用中バッジ(IT-03)を表示する</li>
</ul>
