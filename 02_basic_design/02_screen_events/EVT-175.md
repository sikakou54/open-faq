# <span id="EVT-175"></span>EVT-175: 表示名を入力

> **画面 SCR-022 のイベント「表示名を入力」を定義します。** 対応画面・対応業務UC・対象項目・呼出API・遷移先と、処理内容を記述します。

*版数 v1.0 ・ 更新 2026-06-21 ・ 再構成 P3*

## 項目

| 項目 | 内容 |
|---|---|
| 画面イベントID | `EVT-175` |
| イベント名 | 表示名を入力 |
| 対応画面ID | [SCR-022](../01_screens/SCR-022.md#SCR-022) |
| 対応業務UC | [UC-175](../../01_requirements/02_business_usecases/UC-175.md#UC-175) |
| 対象項目ID | [IT-01](../01_screens/SCR-022.md#IT-01) |
| 呼出API | — |
| 遷移先 | — |

## 処理

<ul>
<li>成功時: 1〜100 文字の範囲内であればバリデーションエラーを表示しない</li>
<li>失敗時: 空欄または 101 文字以上の場合、入力欄下部にバリデーションエラーを表示する</li>
</ul>

## 備考

再構成 P3 で旧画面イベント `SCR-017/EV-03`(現 SCR-022 §6 の `EV-03`)から導出。処理内容の正本は SCR-022 §6 と本ページ。
