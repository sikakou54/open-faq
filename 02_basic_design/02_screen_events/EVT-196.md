# <span id="EVT-196"></span>EVT-196: 初期表示

> **画面 SCR-025 のイベント「初期表示」を定義します。** 対応画面・対応業務UC・対象項目・呼出API・遷移先と、処理内容を記述します。

*版数 v1.0 ・ 更新 2026-06-21 ・ 再構成 P3*

## 項目

| 項目 | 内容 |
|---|---|
| 画面イベントID | `EVT-196` |
| イベント名 | 初期表示 |
| 対応画面ID | [SCR-025](../01_screens/SCR-025.md#SCR-025) |
| 対応業務UC | [UC-196](../../01_requirements/02_business_usecases/UC-196.md#UC-196) |
| 対象項目ID | — |
| 呼出API | [プライバシーポリシー 最新版取得](../03_apis/API-terms.md#API-TRM-002) |
| 遷移先 | [SCR-020](../01_screens/SCR-020.md#SCR-020) |

## 処理

<ul>
<li><a href="../03_apis/API-terms.md#API-TRM-002">プライバシーポリシー 最新版取得</a> API で最新版全文を取得し IT-02 に表示</li>
<li>ログイン済みの場合: 同意状態バッジ(IT-01)を取得・表示する。同意済みは緑バッジ(同意日付き)、未同意は赤バッジ(SCR-020 へのリンク付き)で表示</li>
<li>未ログインの場合: 同意状態バッジを非表示とし、全文のみ表示</li>
</ul>

## 備考

再構成 P3 で旧画面イベント `SCR-020/EV-01`(現 SCR-025 §6 の `EV-01`)から導出。処理内容の正本は SCR-025 §6 と本ページ。
