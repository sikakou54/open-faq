# <span id="EVT-228"></span>EVT-228: 受付制限(429)を受信

> **画面 SCR-030 のイベント「受付制限(429)を受信」を定義します。** 対応画面・対応業務UC・対象項目・呼出API・遷移先と、処理内容を記述します。

*版数 v1.0 ・ 更新 2026-06-21 ・ 再構成 P3*

## 項目

| 項目 | 内容 |
|---|---|
| 画面イベントID | `EVT-228` |
| イベント名 | 受付制限(429)を受信 |
| 対応画面ID | [SCR-030](../01_screens/SCR-030.md#SCR-030) |
| 対応業務UC | [UC-228](../../01_requirements/02_business_usecases/UC-228.md#UC-228) |
| 対象項目ID | [IT-09](../01_screens/SCR-030.md#IT-09) |
| 呼出API | [ウィジェット質問送信](../03_apis/API-038.md#API-038) |
| 遷移先 | — |

## 処理

<ul>
<li><a href="../03_apis/API-038.md#API-038">ウィジェット質問送信</a> API から質問数上限到達または支払方法ゲートによる 429 を受信した場合に発生する(EV-05 の結果として続いて処理される)</li>
<li>受付停止メッセージをシステム返信として会話欄に追加表示する(IT-09)</li>
<li>確認済みプロジェクト連絡先メールが設定済みの場合は連絡先メールを案内表示する(IT-08)</li>
<li>質問入力欄(IT-05)および送信ボタン(IT-06)を無効化する</li>
</ul>

## 備考

再構成 P3 で旧画面イベント `SCR-WIDGET/EV-07`(現 SCR-030 §6 の `EV-07`)から導出。処理内容の正本は SCR-030 §6 と本ページ。
