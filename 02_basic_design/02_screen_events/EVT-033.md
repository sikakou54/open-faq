# <span id="EVT-033"></span>EVT-033: 初期表示(新規モード)

> **画面 SCR-005 のイベント「初期表示(新規モード)」を定義します。** 対応画面・対応業務UC・対象項目・呼出API・遷移先と、処理内容を記述します。

*版数 v1.0 ・ 更新 2026-06-21 ・ 再構成 P3*

## 項目

| 項目 | 内容 |
|---|---|
| 画面イベントID | `EVT-033` |
| イベント名 | 初期表示(新規モード) |
| 対応画面ID | [SCR-005](../01_screens/SCR-005.md#SCR-005) |
| 対応業務UC | [UC-033](../../01_requirements/02_business_usecases/UC-033.md#UC-033) |
| 対象項目ID | — |
| 呼出API | — |
| 遷移先 | — |

## 処理

空の入力フォームを表示する。見出し(IT-01)を「新規プロジェクトを作成」に設定し、「プロジェクトを作成」ボタン(IT-09)のみ表示する。編集モード専用項目(IT-02 / IT-07 / IT-10 / IT-11)は非表示とする

## 備考

再構成 P3 で旧画面イベント `SCR-004-001/EV-01`(現 SCR-005 §6 の `EV-01`)から導出。処理内容の正本は SCR-005 §6 と本ページ。
