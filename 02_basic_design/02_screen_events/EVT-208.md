# <span id="EVT-208"></span>EVT-208: 初期表示

> **画面 SCR-028 のイベント「初期表示」を定義します。** 対応画面・対応業務UC・対象項目・呼出API・遷移先と、処理内容を記述します。

*版数 v1.0 ・ 更新 2026-06-21 ・ 再構成 P3*

## 項目

| 項目 | 内容 |
|---|---|
| 画面イベントID | `EVT-208` |
| イベント名 | 初期表示 |
| 対応画面ID | [SCR-028](../01_screens/SCR-028.md#SCR-028) |
| 対応業務UC | [UC-208](../../01_requirements/02_business_usecases/UC-208.md#UC-208) |
| 対象項目ID | — |
| 呼出API | [請求サマリ取得](../03_apis/API-billing.md#API-BIL-003) ・ [請求履歴取得](../03_apis/API-billing.md#API-BIL-004) |
| 遷移先 | — |

## 処理

<ul>
<li><a href="../03_apis/API-billing.md#API-BIL-003">請求サマリ取得</a> API で請求見込み・次回請求日・請求状態・プロジェクト別内訳を取得し IT-01〜IT-05 へ表示する</li>
<li><a href="../03_apis/API-billing.md#API-BIL-004">請求履歴取得</a> API で請求月・金額・状態・PDF リンクを取得し IT-07 へ表示する</li>
<li>請求状態が支払い失敗または支払方法未登録の場合: IT-09 バナーを表示し、原因・影響・復旧手順・復旧 CTA を示す</li>
</ul>

## 備考

再構成 P3 で旧画面イベント `SCR-022/EV-01`(現 SCR-028 §6 の `EV-01`)から導出。処理内容の正本は SCR-028 §6 と本ページ。
