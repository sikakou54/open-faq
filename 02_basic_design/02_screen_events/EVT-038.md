# <span id="EVT-038"></span>EVT-038: 「プロジェクトを作成」を押下

> **画面 SCR-005 のイベント「「プロジェクトを作成」を押下」を定義します。** 対応画面・対応業務UC・対象項目・呼出API・遷移先と、処理内容を記述します。

*版数 v1.0 ・ 更新 2026-06-21 ・ 再構成 P3*

## 項目

| 項目 | 内容 |
|---|---|
| 画面イベントID | `EVT-038` |
| イベント名 | 「プロジェクトを作成」を押下 |
| 対応画面ID | [SCR-005](../01_screens/SCR-005.md#SCR-005) |
| 対応業務UC | [UC-038](../../01_requirements/02_business_usecases/UC-038.md#UC-038) |
| 対象項目ID | [IT-09](../01_screens/SCR-005.md#IT-09) |
| 呼出API | [プロジェクト新規作成](../03_apis/API-project.md#API-PRJ-002) |
| 遷移先 | [SCR-004](../01_screens/SCR-004.md#SCR-004) |

## 処理

<ul>
<li>全項目のバリデーションを実行し、エラーがある場合は対象欄にインラインエラーを表示して送信を中断する</li>
<li>成功時: <a href="../03_apis/API-project.md#API-PRJ-002">プロジェクト新規作成</a> API を呼び出しプロジェクトを作成する。オーナーを当該プロジェクトのメンバーとして自動登録し、ウィジェット公開キーを発行する。モーダルを閉じ SCR-004 の一覧を更新する</li>
<li>失敗時(重複名): プロジェクト名欄にインラインエラー「このプロジェクト名は既に使用されています」を表示する</li>
<li>失敗時(その他): トーストでエラーを表示する</li>
</ul>

## 備考

再構成 P3 で旧画面イベント `SCR-004-001/EV-06`(現 SCR-005 §6 の `EV-06`)から導出。処理内容の正本は SCR-005 §6 と本ページ。
