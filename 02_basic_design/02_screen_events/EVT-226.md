# <span id="EVT-226"></span>EVT-226: 「送信」を押下

> **画面 SCR-030 のイベント「「送信」を押下」を定義します。** 対応画面・対応業務UC・対象項目・呼出API・遷移先と、処理内容を記述します。

*版数 v1.0 ・ 更新 2026-06-21 ・ 再構成 P3*

## 項目

| 項目 | 内容 |
|---|---|
| 画面イベントID | `EVT-226` |
| イベント名 | 「送信」を押下 |
| 対応画面ID | [SCR-030](../01_screens/SCR-030.md#SCR-030) |
| 対応業務UC | [UC-226](../../01_requirements/02_business_usecases/UC-226.md#UC-226) |
| 対象項目ID | [IT-06](../01_screens/SCR-030.md#IT-06) |
| 呼出API | [ウィジェット質問送信](../03_apis/API-038.md#API-038) |
| 遷移先 | — |

## 処理

<ul>
<li><a href="../03_apis/API-038.md#API-038">ウィジェット質問送信</a> API で質問を送信し、AI 回答を同じ会話欄に追加表示する</li>
<li>受付制限中は送信ボタンが無効化されているため操作不可</li>
<li>回答結果が未解決(<code>type=unanswered</code>)の場合は続けて EV-06 の処理が発生する</li>
<li>受付制限(429)を受信した場合は続けて EV-07 の処理が発生する</li>
<li>処理エラーの場合は続けて EV-08 の処理が発生する</li>
</ul>

## 備考

再構成 P3 で旧画面イベント `SCR-WIDGET/EV-05`(現 SCR-030 §6 の `EV-05`)から導出。処理内容の正本は SCR-030 §6 と本ページ。
