# <span id="EVT-225"></span>EVT-225: 質問を入力

> **画面 SCR-030 のイベント「質問を入力」を定義します。** 対応画面・対応業務UC・対象項目・呼出API・遷移先と、処理内容を記述します。

*版数 v1.0 ・ 更新 2026-06-21 ・ 再構成 P3*

## 項目

| 項目 | 内容 |
|---|---|
| 画面イベントID | `EVT-225` |
| イベント名 | 質問を入力 |
| 対応画面ID | [SCR-030](../01_screens/SCR-030.md#SCR-030) |
| 対応業務UC | [UC-225](../../01_requirements/02_business_usecases/UC-225.md#UC-225) |
| 対象項目ID | [IT-05](../01_screens/SCR-030.md#IT-05) |
| 呼出API | — |
| 遷移先 | — |

## 処理

テキストエリアに質問文を入力する。受付制限中(<code>EV-07</code> 後)は入力欄が無効化されているため操作不可

## 備考

再構成 P3 で旧画面イベント `SCR-WIDGET/EV-04`(現 SCR-030 §6 の `EV-04`)から導出。処理内容の正本は SCR-030 §6 と本ページ。
