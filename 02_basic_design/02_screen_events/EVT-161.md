# <span id="EVT-161"></span>EVT-161: 契約名を入力

> **画面 SCR-019 のイベント「契約名を入力」を定義します。** 対応画面・対応業務UC・対象項目・呼出API・遷移先と、処理内容を記述します。

*版数 v1.0 ・ 更新 2026-06-21 ・ 再構成 P3*

## 項目

| 項目 | 内容 |
|---|---|
| 画面イベントID | `EVT-161` |
| イベント名 | 契約名を入力 |
| 対応画面ID | [SCR-019](../01_screens/SCR-019.md#SCR-019) |
| 対応業務UC | [UC-161](../../01_requirements/02_business_usecases/UC-161.md#UC-161) |
| 対象項目ID | [IT-05](../01_screens/SCR-019.md#IT-05) |
| 呼出API | — |
| 遷移先 | — |

## 処理

<ul>
<li>入力値をリアルタイムで契約名と照合する</li>
<li>一致する場合: 「退会を申請する」ボタンの無効化を解除する</li>
<li>一致しない場合: 「退会を申請する」ボタンを無効化する</li>
</ul>

## 備考

再構成 P3 で旧画面イベント `SCR-014/EV-06`(現 SCR-019 §6 の `EV-06`)から導出。処理内容の正本は SCR-019 §6 と本ページ。
