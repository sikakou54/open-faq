# <span id="EVT-227"></span>EVT-227: AI 回答(未解決)を受信

> **画面 SCR-030 のイベント「AI 回答(未解決)を受信」を定義します。** 対応画面・対応業務UC・対象項目・呼出API・遷移先と、処理内容を記述します。

*版数 v1.0 ・ 更新 2026-06-21 ・ 再構成 P3*

## 項目

| 項目 | 内容 |
|---|---|
| 画面イベントID | `EVT-227` |
| イベント名 | AI 回答(未解決)を受信 |
| 対応画面ID | [SCR-030](../01_screens/SCR-030.md#SCR-030) |
| 対応業務UC | [UC-227](../../01_requirements/02_business_usecases/UC-227.md#UC-227) |
| 対象項目ID | [IT-08](../01_screens/SCR-030.md#IT-08) |
| 呼出API | [ウィジェット質問送信](../03_apis/API-038.md#API-038) ・ [ウィジェット未解決質問登録](../03_apis/API-039.md#API-039) |
| 遷移先 | — |

## 処理

<ul>
<li><a href="../03_apis/API-038.md#API-038">ウィジェット質問送信</a> API のレスポンスが <code>type=unanswered</code> の場合に発生する(EV-05 の結果として続いて処理される)</li>
<li>回答できなかった旨をシステム返信として会話欄に追加表示する</li>
<li><a href="../03_apis/API-039.md#API-039">ウィジェット未解決質問登録</a> API を呼び出し、質問ログと未解決質問を登録する</li>
<li>確認済みプロジェクト連絡先メールが設定済みの場合は連絡先メールを案内表示する(IT-08)</li>
<li>管理用の問い合わせ ID はウィジェットに表示しない</li>
<li>別の質問の入力・送信は引き続き可能(EV-04 / EV-05 へ戻れる)</li>
</ul>

## 備考

再構成 P3 で旧画面イベント `SCR-WIDGET/EV-06`(現 SCR-030 §6 の `EV-06`)から導出。処理内容の正本は SCR-030 §6 と本ページ。
