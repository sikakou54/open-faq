# <span id="EVT-199"></span>EVT-199: 初期表示

> **画面 SCR-026 のイベント「初期表示」を定義します。** 対応画面・対応業務UC・対象項目・呼出API・遷移先と、処理内容を記述します。

*版数 v1.0 ・ 更新 2026-06-21 ・ 再構成 P3*

## 項目

| 項目 | 内容 |
|---|---|
| 画面イベントID | `EVT-199` |
| イベント名 | 初期表示 |
| 対応画面ID | [SCR-026](../01_screens/SCR-026.md#SCR-026) |
| 対応業務UC | [UC-199](../../01_requirements/02_business_usecases/UC-199.md#UC-199) |
| 対象項目ID | — |
| 呼出API | [利用量サマリ(プロジェクト)](../03_apis/API-billing.md#API-BIL-001) ・ [プロジェクト上限・アラート取得](../03_apis/API-billing.md#API-BIL-006) |
| 遷移先 | — |

## 処理

<ul>
<li><a href="../03_apis/API-billing.md#API-BIL-001">利用量サマリ(プロジェクト)</a> で当月利用を取得し IT-03・IT-04 に表示する</li>
<li><a href="../03_apis/API-billing.md#API-BIL-006">プロジェクト上限・アラート取得</a> で月次上限設定を取得し IT-03 に反映する</li>
<li>成功 / 上限 OFF 時: 上限 OFF に対応した表示を行う(IT-03・IT-04 の ON/OFF 表示切替)</li>
<li>集計前 / 取得失敗時: IT-06 空状態を表示する</li>
</ul>

## 備考

再構成 P3 で旧画面イベント `SCR-021/EV-01`(現 SCR-026 §6 の `EV-01`)から導出。処理内容の正本は SCR-026 §6 と本ページ。
