# <span id="EVT-213"></span>EVT-213: 「支払い方法を登録」を押下(バナー CTA)

> **画面 SCR-028 のイベント「「支払い方法を登録」を押下(バナー CTA)」を定義します。** 対応画面・対応業務UC・対象項目・呼出API・遷移先と、処理内容を記述します。

*版数 v1.0 ・ 更新 2026-06-21 ・ 再構成 P3*

## 項目

| 項目 | 内容 |
|---|---|
| 画面イベントID | `EVT-213` |
| イベント名 | 「支払い方法を登録」を押下(バナー CTA) |
| 対応画面ID | [SCR-028](../01_screens/SCR-028.md#SCR-028) |
| 対応業務UC | [UC-213](../../01_requirements/02_business_usecases/UC-213.md#UC-213) |
| 対象項目ID | [IT-10](../01_screens/SCR-028.md#IT-10) |
| 呼出API | [支払方法取得・更新](../03_apis/API-billing.md#API-BIL-005) |
| 遷移先 | — |

## 処理

<ul>
<li>課金情報変更につき再認証(現パスワード再入力)を求める</li>
<li>再認証成功時: <a href="../03_apis/API-billing.md#API-BIL-005">支払方法取得・更新</a> API で支払方法を登録し、IT-05 を最新の情報へ更新し、IT-09 バナーを非表示にする</li>
<li>再認証失敗時: エラーを表示し操作を中断する</li>
</ul>

## 備考

再構成 P3 で旧画面イベント `SCR-022/EV-06`(現 SCR-028 §6 の `EV-06`)から導出。処理内容の正本は SCR-028 §6 と本ページ。
