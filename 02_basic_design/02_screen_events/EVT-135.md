# <span id="EVT-135"></span>EVT-135: 「同意して続ける」を押下

> **画面 SCR-015 のイベント「「同意して続ける」を押下」を定義します。** 対応画面・対応業務UC・対象項目・呼出API・遷移先と、処理内容を記述します。

*版数 v1.0 ・ 更新 2026-06-21 ・ 再構成 P3*

## 項目

| 項目 | 内容 |
|---|---|
| 画面イベントID | `EVT-135` |
| イベント名 | 「同意して続ける」を押下 |
| 対応画面ID | [SCR-015](../01_screens/SCR-015.md#SCR-015) |
| 対応業務UC | [UC-135](../../01_requirements/02_business_usecases/UC-135.md#UC-135) |
| 対象項目ID | [IT-03](../01_screens/SCR-015.md#IT-03) |
| 呼出API | [利用規約 同意](../03_apis/API-terms.md#API-TRM-003) |
| 遷移先 | — |

## 処理

<ul>
<li><a href="../03_apis/API-terms.md#API-TRM-003">利用規約 同意</a> を呼び出し、最新版バージョンへの同意を記録する(冪等)</li>
<li>成功時: 同意状態バッジ(IT-01)を緑バッジ(同意済み)に更新し、遷移元画面(またはトップ)へ戻る</li>
<li>失敗時(未ログイン / 認証エラー): エラーメッセージを表示し、ログイン画面へ誘導する</li>
</ul>

## 備考

再構成 P3 で旧画面イベント `SCR-010/EV-03`(現 SCR-015 §6 の `EV-03`)から導出。処理内容の正本は SCR-015 §6 と本ページ。
