# <span id="EVT-179"></span>EVT-179: 「変更を破棄」を押下

> **画面 SCR-022 のイベント「「変更を破棄」を押下」を定義します。** 対応画面・対応業務UC・対象項目・呼出API・遷移先と、処理内容を記述します。

*版数 v1.0 ・ 更新 2026-06-21 ・ 再構成 P3*

## 項目

| 項目 | 内容 |
|---|---|
| 画面イベントID | `EVT-179` |
| イベント名 | 「変更を破棄」を押下 |
| 対応画面ID | [SCR-022](../01_screens/SCR-022.md#SCR-022) |
| 対応業務UC | [UC-179](../../01_requirements/02_business_usecases/UC-179.md#UC-179) |

| 呼出API | — |
