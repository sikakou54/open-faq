# <span id="EVT-164"></span>EVT-164: 初期表示

> **画面 SCR-020 のイベント「初期表示」を定義します。** 対応画面・対応業務UC・対象項目・呼出API・遷移先と、処理内容を記述します。

*版数 v1.0 ・ 更新 2026-06-21 ・ 再構成 P3*

## 項目

| 項目 | 内容 |
|---|---|
| 画面イベントID | `EVT-164` |
| イベント名 | 初期表示 |
| 対応画面ID | [SCR-020](../01_screens/SCR-020.md#SCR-020) |
| 対応業務UC | [UC-164](../../01_requirements/02_business_usecases/UC-164.md#UC-164) |
| 対象項目ID | — |
| 呼出API | [利用規約 最新版取得](../03_apis/API-terms.md#API-TRM-001) ・ [プライバシーポリシー 最新版取得](../03_apis/API-terms.md#API-TRM-002) |
| 遷移先 | — |

## 処理

<ul>
<li><a href="../03_apis/API-terms.md#API-TRM-001">利用規約 最新版取得</a> API と <a href="../03_apis/API-terms.md#API-TRM-002">プライバシーポリシー 最新版取得</a> API で改定文書の差分サマリを取得</li>
<li>改定対象の文書(利用規約 / プライバシーポリシー、または両方)の主な変更点を全画面モーダルで表示</li>
<li>改定対象外の文書のチェックボックスは非表示とする</li>
</ul>

## 備考

再構成 P3 で旧画面イベント `SCR-015/EV-01`(現 SCR-020 §6 の `EV-01`)から導出。処理内容の正本は SCR-020 §6 と本ページ。
