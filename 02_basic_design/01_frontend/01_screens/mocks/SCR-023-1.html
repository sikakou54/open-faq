備考
  • 招待は登録済みユーザー限定。本画面はログイン中の本人が利用する(アカウントは独立サインアップで事前作成済み。氏名・パスワード・規約同意の入力は行わない)
  • 招待メールのリンク(受諾用トークン / 有効期限 7 日)から到達。未ログインの場合はログインへ誘導し、ログイン後に本画面へ戻す
  • 「参加を受諾する」押下時は招待受諾 API を発行し、同一トランザクションで「招待先メールとログイン本人の一致確認 / 対象プロジェクトへの自分の割当の有効化 / トークン消費 / 監査ログ記録」を実行する
  • 招待先メールと異なるアカウントでログイン中の場合は受諾できず、本人としてログインし直すよう案内する
