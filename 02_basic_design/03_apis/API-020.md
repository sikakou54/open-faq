# <span id="API-020"></span>API-020: メンバー一覧

> **このページは メンバー一覧 API の契約を定義します。** 指定プロジェクトのメンバー割当一覧を返す。

*版数 v2.0 ・ 更新 2026-06-21 ・ 再構成 P4*

## 項目

| 項目 | 内容 |
|---|---|
| API ID | `API-020` |
| API名 | メンバー一覧 |
| 対応業務UC | [UC-115](../../01_requirements/02_business_usecases/UC-115.md#UC-115) ・ [UC-116](../../01_requirements/02_business_usecases/UC-116.md#UC-116) ・ [UC-117](../../01_requirements/02_business_usecases/UC-117.md#UC-117) ・ [UC-121](../../01_requirements/02_business_usecases/UC-121.md#UC-121) ・ [UC-124](../../01_requirements/02_business_usecases/UC-124.md#UC-124) ・ [UC-236](../../01_requirements/02_business_usecases/UC-236.md#UC-236) |
| 対応画面ID | [SCR-013](../01_screens/SCR-013.md#SCR-013) ・ [SCR-014](../01_screens/SCR-014.md#SCR-014) |
| 対応画面イベントID | [EVT-115](../02_screen_events/EVT-115.md#EVT-115) ・ [EVT-116](../02_screen_events/EVT-116.md#EVT-116) ・ [EVT-117](../02_screen_events/EVT-117.md#EVT-117) ・ [EVT-124](../02_screen_events/EVT-124.md#EVT-124) |
| エンドポイント | `/projects/{id}/members` |
| HTTPメソッド | GET |
| 認証 | Cookie |
| 認可 | オーナー / 当該プロジェクトのメンバー |

## 処理概要

指定プロジェクトのメンバー割当一覧を返す。

| 処理 ID | 処理内容                                                |
|---------|---------------------------------------------------------|
| P-01    | 認証・認可(オーナー / 当該プロジェクトのメンバー)を検証する |
| P-02    | 指定プロジェクトの有効な割当を取得する                  |
| P-03    | ページング(カーソル)で整形して返す                      |

## リクエスト

#### Query Parameters

```json
{
  "cursor": "(共通仕様)",
  "limit": "(共通仕様)"
}
```

| 項目     | 型      | 説明                                             |
|----------|---------|--------------------------------------------------|
| `cursor` | string  | 次ページ取得用カーソル(任意・共通ページング仕様) |
| `limit`  | integer | 1 ページの取得件数(任意・共通ページング仕様)     |

#### Path Parameters

```json
{ "id": "プロジェクト ID" }
```

| 項目 | 型     | 説明                                      |
|------|--------|-------------------------------------------|
| `id` | string | 対象プロジェクトの ID(必須・パス内で指定) |

#### Request Body

```json
{}
```

## レスポンス

**レスポンス(200)**

```json
{
  "items": [
    {
      "id": "01HZ...",
      "email": "member@...",
      "status": "active|pending_activation",
      "isOwner": false,
      "invitationExpiresAt": "2026-05-20T..."
    }
  ],
  "nextCursor": "01HZ..." | null
}
```

| 項目 | 型 | 説明 |
|----|----|----|
| `items` | array\<object\> | 指定プロジェクトのメンバー割当一覧 |
| `　└ id` | string | メンバー(利用者)の ID |
| `　└ email` | string | メンバーのメールアドレス |
| `　└ status` | enum | アカウント状態(`active` / `pending_activation`) |
| `　└ isOwner` | boolean | オーナーかどうか |
| `　└ invitationExpiresAt` | string (ISO 8601) | 未有効化時の招待トークン有効期限 |
| `nextCursor` | string | 次ページ取得用カーソル(末尾は `null`) |

> [!NOTE]
> **補足** `items[]` は URL で指定したプロジェクトの割当だけを返す。最終ログイン日時と他プロジェクトの割当は返さない。
