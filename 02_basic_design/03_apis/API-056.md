# <span id="API-056"></span>API-056: 退会申請

> **このページは 退会申請 API の契約を定義します。** オーナーの退会を申請する。

*版数 v2.0 ・ 更新 2026-06-21 ・ 再構成 P4*

## 項目

| 項目 | 内容 |
|---|---|
| API ID | `API-056` |
| API名 | 退会申請 |
| 対応業務UC | [UC-159](../../01_requirements/02_business_usecases/UC-159.md#UC-159) |
| 対応画面ID | [SCR-019](../01_screens/SCR-019.md#SCR-019) |
| 対応画面イベントID | [EVT-159](../02_screen_events/EVT-159.md#EVT-159) |
| エンドポイント | `/withdrawal-requests` |
| HTTPメソッド | POST |
| 認証 | オーナー専有、再認証必須 |
| 認可 | オーナー専有 |

## 処理概要

オーナーの退会を申請する。

| 処理 ID | 処理内容                                       |
|---------|------------------------------------------------|
| P-01    | 認証・認可(オーナー専有、再認証必須)を検証する |
| P-02    | 退会申請を登録し受付状態とする                 |
| P-03    | 受付結果を返却する                             |

## リクエスト

#### Query Parameters

```json
{}
```

#### Path Parameters

```json
{}
```

#### Request Body

```json
{ "reason": "退会申請内容(理由等)" }
```

| 項目     | 型     | 説明                     |
|----------|--------|--------------------------|
| `reason` | string | 退会理由・申請内容(任意) |

## レスポンス

**レスポンス(2xx)**

退会申請の受付結果。

## バリデーション

入力値の検証規則は `## リクエスト` の各パラメータ説明を参照する。

## エラー

| HTTP ステータス | エラーコード        | 内容               |
|-----------------|---------------------|--------------------|
| 403             | `PERMISSION_DENIED` | メンバーは申請不可 |
