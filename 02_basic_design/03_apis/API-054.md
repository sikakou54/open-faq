# <span id="API-054"></span>API-054: 利用規約 同意

> **このページは 利用規約 同意 API の契約を定義します。** 利用規約への同意を記録する(冪等)。

*版数 v2.0 ・ 更新 2026-06-21 ・ 再構成 P4*

## 項目

| 項目 | 内容 |
|---|---|
| API ID | `API-054` |
| API名 | 利用規約 同意 |
| 対応業務UC | [UC-135](../../01_requirements/04_business_usecases/UC-135.md#UC-135) ・ [UC-169](../../01_requirements/04_business_usecases/UC-169.md#UC-169) |
| 対応画面ID | [SCR-015](../01_screens/SCR-015.md#SCR-015) ・ [SCR-020](../01_screens/SCR-020.md#SCR-020) |
| 対応画面イベントID | [EVT-135](../02_screen_events/EVT-135.md#EVT-135) ・ [EVT-169](../02_screen_events/EVT-169.md#EVT-169) |
| エンドポイント | `/terms/agree` |
| HTTPメソッド | POST |
| 認証 | Cookie + CSRF |
| 認可 | オーナー / メンバー |

## 処理概要

利用規約への同意を記録する(冪等)。

| 処理 ID | 処理内容                                                       |
|---------|----------------------------------------------------------------|
| P-01    | 認証・認可(オーナー / メンバー、CSRF)を検証する                |
| P-02    | 指定バージョンへの利用規約同意を記録する(既存同意があれば冪等) |
| P-03    | 同意記録結果を返却する                                         |

## リクエスト

#### Query Parameters

```json
{}
```

#### Path Parameters

```json
{}
```

#### Request Body

```json
{ "version": "..." }
```

| 項目      | 型     | 説明                               |
|-----------|--------|------------------------------------|
| `version` | string | 同意対象の利用規約バージョン(必須) |

`T_TERMS_AGREE` に `(user_id, doc_type='terms_of_service', terms_version)` で 1 行記録(冪等: 既存同意があれば 200)。

## レスポンス

**レスポンス(200)**

同意記録結果。

## バリデーション

入力値の検証規則は `## リクエスト` の各パラメータ説明を参照する。

## エラー

本 API 固有のエラーコードはない([API 共通仕様](index.md#conv) の標準エラー体系に従う)。

## 利用テーブル

本 API はテーブルを直接読み書きしない(外部サービス連携 IF)。

## 備考
