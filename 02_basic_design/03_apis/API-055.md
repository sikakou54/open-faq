# <span id="API-055"></span>API-055: プライバシーポリシー 同意

> **このページは プライバシーポリシー 同意 API の契約を定義します。** プライバシーポリシーへの同意を記録する(冪等)。

*版数 v2.0 ・ 更新 2026-06-21 ・ 再構成 P4*

## 項目

| 項目 | 内容 |
|---|---|
| API ID | `API-055` |
| API名 | プライバシーポリシー 同意 |
| 対応業務UC | [UC-013](../../01_requirements/04_business_usecases/UC-013.md#UC-013) |
| 対応画面ID | [SCR-020](../01_screens/SCR-020.md#SCR-020) |
| 対応画面イベントID | [EVT-169](../02_screen_events/EVT-169.md#EVT-169) |
| エンドポイント | `/privacy/agree` |
| HTTPメソッド | POST |
| 認証 | Cookie + CSRF |
| 認可 | オーナー / メンバー |

## 処理概要

プライバシーポリシーへの同意を記録する(冪等)。

| 処理 ID | 処理内容 |
|----|----|
| P-01 | 認証・認可(オーナー / メンバー、CSRF)を検証する |
| P-02 | 指定バージョンへのプライバシーポリシー同意を記録する(既存同意があれば冪等) |
| P-03 | 同意記録結果を返却する |

## リクエスト

#### Query Parameters

```json
{}
```

#### Path Parameters

```json
{}
```

#### Request Body

```json
{ "version": "..." }
```

| 項目      | 型     | 説明                                           |
|-----------|--------|------------------------------------------------|
| `version` | string | 同意対象のプライバシーポリシーバージョン(必須) |

`T_TERMS_AGREE` に `(user_id, doc_type='privacy_policy', terms_version)` で 1 行記録(冪等: 既存同意があれば 200)。SCR-020 / SCR-023 では利用規約・プライバシーポリシー双方の同意が必須条件であり、改定再同意では改定された文書の agree のみを呼び出す。

## レスポンス

**レスポンス(200)**

同意記録結果。

## バリデーション

入力値の検証規則は `## リクエスト` の各パラメータ説明を参照する。

## 利用テーブル

本 API はテーブルを直接読み書きしない(外部サービス連携 IF)。
