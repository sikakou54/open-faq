# <span id="API-023"></span>API-023: プロジェクト割当解除

> **このページは プロジェクト割当解除 API の契約を定義します。** 対象プロジェクトの `M_PRJ_USERS.valid=0` を更新する。

*版数 v2.0 ・ 更新 2026-06-21 ・ 再構成 P4*

## 項目

| 項目 | 内容 |
|---|---|
| API ID | `API-023` |
| API名 | プロジェクト割当解除 |
| 対応業務UC | [UC-130](../../01_requirements/02_business_usecases/UC-130.md#UC-130) |
| 対応画面ID | [SCR-014](../01_screens/SCR-014.md#SCR-014) |
| 対応画面イベントID | [EVT-130](../02_screen_events/EVT-130.md#EVT-130) |
| エンドポイント | `/projects/{id}/members/{userId}` |
| HTTPメソッド | DELETE |
| 認証 | Cookie + CSRF + 再認証必須 |
| 認可 | オーナー / 当該プロジェクトのメンバー |

## 処理概要

対象プロジェクトの `M_PRJ_USERS.valid=0` を更新する。

| 処理 ID | 処理内容 |
|----|----|
| P-01 | 認証・認可(オーナー / 当該プロジェクトのメンバー)と再認証を検証する |
| P-02 | 対象が自分自身またはオーナーの場合は拒否する |
| P-03 | 当該プロジェクトの割当を論理削除する |
| P-04 | 他プロジェクトの有効割当が 0 件になる場合はアカウントを論理削除し、全セッション失効と未使用招待トークン失効を行う |
| P-05 | 解除結果(アカウント無効化の有無)を返す |

## リクエスト

#### Query Parameters

```json
{}
```

#### Path Parameters

```json
{
  "id": "プロジェクト ID",
  "userId": "対象ユーザー ID"
}
```

| 項目     | 型     | 説明                                                |
|----------|--------|-----------------------------------------------------|
| `id`     | string | 割当を解除するプロジェクトの ID(必須・パス内で指定) |
| `userId` | string | 割当を解除する対象利用者の ID(必須・パス内で指定)   |

#### Request Body

```json
{}
```

## レスポンス

**レスポンス(200)**

```json
{ "userId": "...", "projectId": "...", "accountDeactivated": true|false }
```

| 項目 | 型 | 説明 |
|----|----|----|
| `userId` | string | 割当を解除した対象利用者の ID |
| `projectId` | string | 割当を解除したプロジェクトの ID |
| `accountDeactivated` | boolean | 他プロジェクトの有効割当が 0 件となりアカウントを論理削除したかどうか |

## バリデーション

入力値の検証規則は `## リクエスト` の各パラメータ説明を参照する。

## エラー

| HTTP ステータス | エラーコード              | 内容                         |
|-----------------|---------------------------|------------------------------|
| 403             | `E-AUTHZ-OWNER-PROTECTED` | オーナーは解除不可           |
| 403             | `E-AUTHZ-SELF-MUTATION`   | 自分自身は解除不可           |
| 403             | `PROJECT_ACCESS_DENIED`   | 当該プロジェクトへの権限なし |
| 404             | `NOT_FOUND`               | 対象が存在しない             |
