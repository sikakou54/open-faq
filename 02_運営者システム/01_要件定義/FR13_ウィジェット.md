## 1. 機能概要

管理者の Web サイトに埋め込む JavaScript ウィジェットの取得・許可ドメイン制限・表示テーマ設定・モバイル対応・アクセシビリティ・ブラウザサポート・CDN 配信を扱う。**本書側で正本管理する FR は無い**(本書 §8.1.1 のとおり、FR-193〜195: API キーローテーション・公開キー再発行・不正利用検知 はメインシステム側で正本管理)。本書は参照側として要件と整合を維持する。

## 2. 収録 FR-NNN 一覧

| ID | 要件 | 優先度 |
|---|---|:---:|
| FR-150 | 管理者は埋め込みコードを取得し、自社サイトに設置できること | P0 |
| FR-151 | ウィジェットは指定された許可ドメイン上でのみ動作すること | P0 |
| FR-152 | ウィジェットの表示位置・カラーテーマ等の基本的な見た目をプロジェクトごとに設定できること | P1 |
| FR-153 | ウィジェットはモバイル端末でも利用できること | P0 |
| FR-154 | ウィジェットがアクセシビリティ要件(キーボード操作、スクリーンリーダー、コントラスト等)に配慮していること | P1 |
| FR-155 | ウィジェットはサポート対象ブラウザで動作すること | P0 |
| FR-156 | ウィジェット配信は高速化(CDN/キャッシュ)できること | P1 |

## 3. 補足説明・統合参照

- 許可ドメイン検証(FR-151)は本書 [FR15_セキュリティ.md](FR15_セキュリティ.md) FR-173(埋め込み元ドメイン検証)と整合。
- アクセシビリティは NFR-1001〜1003 / 本書 [FR22_アクセシビリティ強化.md](FR22_アクセシビリティ強化.md) を参照。
- 運営者は緊急対応で「ウィジェット強制停止」(FR-224(a))を実施でき、これは §6.2.3 4-eyes 承認対象(本書 [FR23_4-eyes承認.md](FR23_4-eyes承認.md))となる。
- API キー(ウィジェット公開キー)のローテーション猶予期間は 30 日(NFR-322)。具体運用はメインシステム FR-193 / FR-194 を正本とする。
